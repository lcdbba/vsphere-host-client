Was ist eine Kategorie?

Eine Kategorie umfasst verwandte Tags in der vCenter Server-Bestandsliste.

Wenn Sie eine Kategorie erstellen, geben Sie an, ob einem angegebenen Objekt gleichzeitig mehrere Tags in dieser Kategorie zugewiesen werden können oder ob einem Objekt jeweils nur ein Tag zugewiesen werden kann. Beispiel: Eine Kategorie namens Priorität enthält möglicherweise die Tags Hoch, Mittel und Niedrig und ist so konfiguriert, dass einem Objekt jeweils nur ein Tag in der Kategorie zugewiesen werden kann.

Sie können außerdem angeben, ob Tags in einer Kategorie allen Objekten oder nur bestimmten Objekttypen, wie z. B. Hosts oder Datenspeichern, zugewiesen werden können.
