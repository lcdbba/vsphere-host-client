Was ist ein Cluster?

Ein Cluster ist eine Gruppe von Hosts mit gemeinsamen Ressourcen und einer gemeinsamen Verwaltungsschnittstelle. Wenn Sie einem Cluster einen Host hinzufügen, werden die Ressourcen des Hosts zu den Ressourcen des Clusters hinzugefügt. Der Cluster verwaltet die Ressourcen aller zugehörigen Hosts.

Cluster aktivieren die Lösungen vSphere Distributed Resource Scheduler (DRS) und vSphere High Availability (HA). vSphere DRS verteilt fortlaufend die Arbeitslast der virtuellen Maschinen auf Ihre ESX/ESXi-Hosts. vSphere HA sorgt dafür, dass die virtuellen Maschinen, die auf ESX/ESXi-Hosts ausgeführt werden, nach Hostausfällen automatisch wiederhergestellt werden.
