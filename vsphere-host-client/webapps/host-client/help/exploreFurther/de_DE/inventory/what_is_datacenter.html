Was ist ein Datencenter?

Ein Datencenter ist der primäre Container für Bestandslistenobjekte wie z. B. Hosts und virtuelle Maschinen. Im Datencenter können Sie Bestandslistenobjekte hinzufügen und organisieren. Üblicherweise werden einem Datencenter Hosts, Ordner und Cluster hinzugefügt.

Der vCenter Server kann mehrere Datencenter umfassen. Große Unternehmen setzen möglicherweise mehrere Datencenter ein, um die Organisationseinheiten ihres Unternehmens zu repräsentieren.

Bestandslistenobjekte können innerhalb eines Datencenters interagieren, die Interaktion über mehrere Datencenter hinweg ist jedoch begrenzt. Sie können eine virtuelle Maschine beispielsweise mithilfe der vMotion-Technologie zwischen verschiedenen Hosts innerhalb eines Datencenters verschieben, nicht jedoch auf einen Host in einem anderen Datencenter.
