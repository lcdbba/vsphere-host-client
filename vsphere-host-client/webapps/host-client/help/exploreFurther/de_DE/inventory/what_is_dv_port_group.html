Was ist eine verteilte Portgruppe?

Eine verteilte Portgruppe ist eine Portgruppe, die mit einem vSphere Distributed Switch verbunden ist. Sie enthält die Portkonfigurationsoptionen für jeden zur Gruppe gehörenden Port. Verteilte Portgruppen definieren, wie anhand des vSphere Distributed Switch eine Verbindung zum Netzwerk vorgenommen wird.
