Was ist ein Ordner?

Ein Ordner ist der Container, der zur genaueren Gruppierung von Objekten in der Bestandsliste verwendet wird. Ordner bieten eine Struktur, auf die Berechtigungen angewendet werden können.

In Ordnern können Sie beispielsweise virtuelle Maschinen und Vorlagen nach Funktion organisieren. Auf gleiche Weise können Sie Ordner verwenden, um Datencenter nach geografischem Standort zu organisieren.

Sie können folgende Bestandslistenobjekte in Ordnern gruppieren:

  • Datencenter
  • Virtuelle Maschinen (einschließlich Vorlagen)
  • Computerressourcen (einschließlich Hosts und Clustern)

Die Objekte innerhalb eines Ordners müssen den gleichen Typ aufweisen.
