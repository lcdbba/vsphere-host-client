Was ist ein Host?

Ein Host ist ein Computer, der mithilfe von Virtualisierungssoftware virtuelle Maschinen ausführt. Üblicherweise handelt es sich bei einem Host um einen Computer, auf dem die ESX- oder ESXi-Software ausgeführt wird.

Hosts stellen die CPU- und Arbeitsspeicher-Ressourcen zur Verfügung, die von den virtuellen Maschinen verwendet werden, und bieten virtuellen Maschinen den Zugriff auf Massenspeicher und Netzwerk. Auf einem Host können mehrere virtuelle Maschinen gleichzeitig installiert sein.

Indem Sie einen Host zum vCenter Server hinzufügen, können Sie den Host und die darauf ausgeführten virtuellen Maschinen verwalten.
