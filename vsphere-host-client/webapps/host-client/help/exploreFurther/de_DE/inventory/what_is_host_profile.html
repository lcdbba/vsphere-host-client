Was ist ein Hostprofil?

Ein Hostprofil erfasst die Konfiguration eines bestimmten Hosts. Es kann zum Konfigurieren anderer Hosts oder zum Validieren verwendet werden, wenn die Konfiguration eines Hosts den vom Administrator festgelegten Erfordernissen entspricht.
