Was ist ein Datenspeicher-Cluster?

Ein Datenspeicher-Cluster ist eine Sammlung von Datenspeichern mit gemeinsam genutzten Ressourcen und einer Verwaltungsschnittstelle. Wenn Sie einen Datenspeicher zu einem Datenspeicher-Cluster hinzufügen, werden die Clusterressourcen Teil der Ressourcen des Datenspeicher-Clusters. Datenspeicher-Cluster werden dazu verwendet, Speicherressourcen zusammenzufassen, was Ihnen ermöglicht, Ressourcenzuteilungsrichtlinien auf Datenspeicher-Cluster-Ebene zu unterstützen.

Wenn Sie einen Datenspeicher-Cluster erstellen, können Sie Speicher-DRS zum Verwalten von Speicherressourcen verwenden. Die mit Speicher-DRS verfügbare E/A-Lastausgleichsfunktion steht nur dann zur Verfügung, wenn alle Hosts, die mit den Datenspeichern im Datenspeicher-Cluster verbunden sind, vSphere Version 5 oder höher aufweisen.
