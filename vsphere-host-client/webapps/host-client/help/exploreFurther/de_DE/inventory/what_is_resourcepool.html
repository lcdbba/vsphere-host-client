Was ist ein Ressourcenpool?

Ein Ressourcenpool bietet die Möglichkeit, die Ressourcen eines eigenständigen Hosts oder eines Clusters in kleinere Einheiten aufzuteilen. Ein Ressourcenpool wird mit einer Gruppe von CPU- und Arbeitsspeicherressourcen konfiguriert, die von den virtuellen Maschinen im Ressourcenpool gemeinsam genutzt werden. Ressourcenpools sind unabhängig und von anderen Ressourcenpools isoliert.

Sie können mehrere physische Server in einem einzigen Ressourcenpool kombinieren, in dem CPU-, Arbeitsspeicherkapazitäten zusammengefasst sind.

Virtuelle Maschinen werden Ressourcenpools zugewiesen, aus denen sie ihre Ressourcen beziehen. Auf diese Weise können die Arbeitslasten virtueller Maschinen kontinuierlich im Ressourcenpool ausgeglichen werden. Wenn die Arbeitslast steigt, weist der vCenter Server automatisch zusätzliche Ressourcen zu und migriert virtuelle Maschinen transparent zwischen Hosts im Ressourcenpool.
