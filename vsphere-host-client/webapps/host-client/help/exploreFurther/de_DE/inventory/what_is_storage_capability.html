Was ist eine Speicherfunktion?

Eine Speicherfunktion gibt die Speichermerkmale eines Datenspeichers an. Zu den Merkmalen gehören Kapazität, Leistung, Verfügbarkeit usw.

  • Systemdefinierte Speicherfunktionen

    Speichersysteme mit aktivierter Storage Awareness weisen den Datenspeichern, die auf diesem System erstellt werden, systemdefinierte Speicherfunktionen zu.

  • Benutzerdefinierte Speicherfunktionen

    Sie erstellen benutzerdefinierte Speicherfunktionen manuell und weisen diese den Datenspeichern manuell zu, um Datenspeicher zu identifizieren, die auf demselben Speichersystem ausgeführt werden oder die über ähnliche bzw. identische Merkmale bezüglich Leistung, Kapazität, Verfügbarkeit usw. verfügen.
