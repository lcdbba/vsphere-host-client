Was ist ein Tag?

Ein Tag ist eine Bezeichnung, die Sie auf Objekte in der vSphere-Bestandsliste anwenden können. Mithilfe von Tags können Sie Objekten Metadaten zuweisen, damit sie besser sortiert und durchsucht werden können.

Tags sind in Kategorien gruppiert, die angeben, wie die Tags auf Bestandslistenobjekte angewendet werden können.
