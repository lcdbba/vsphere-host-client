Was ist eine Vorlage?

Eine Vorlage ist ein Master-Image einer virtuellen Maschine, das zur Erstellung und Bereitstellung neuer virtueller Maschinen verwendet werden kann. Dieses Image umfasst üblicherweise ein Betriebssystem, Anwendungen und Konfigurationseinstellungen für die virtuelle Maschine, die einen vollständigen Satz virtueller Hardware definieren.

Sie können eine Vorlage erstellen, indem Sie eine vorhandene virtuelle Maschine in eine Vorlage konvertieren, eine virtuelle Maschine in eine Vorlage klonen oder eine vorhandene Vorlage klonen. Anschließend können Sie durch Bereitstellung der Vorlage neue virtuelle Maschinen erstellen.

Die Verwendung von Vorlagen spart Zeit, da Sie die neue virtuelle Maschine weder konfigurieren noch ein Gastbetriebssystem installieren müssen.

Sie können Vorlagen anzeigen, indem Sie im Objektnavigator die Kategorie VM-Vorlagen auswählen oder die Bestandslistenstrukturansicht „VMs und Vorlagen“ verwenden.
