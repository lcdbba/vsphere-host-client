Was ist eine vApp?

Eine vApp ist eine Gruppe virtueller Maschinen, die als einzelnes Objekt verwaltet werden können. vApps vereinfachen die Verwaltung komplexer Multi-Tier-Anwendungen, die auf mehreren gegenseitig abhängigen virtuellen Maschinen ausgeführt werden. vApps bieten dieselben Basisfunktionen wie virtuelle Maschinen und Ressourcenpools. Mit vApps können Sie die Reihenfolge festlegen, in der die virtuellen Maschinen in der vApp eingeschaltet werden, virtuellen Maschinen über vApp automatisch IP-Adressen zuweisen und eine Anpassung auf Anwendungsebene bereitstellen.
