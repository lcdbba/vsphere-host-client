Was ist ein vSphere Distributed Switch?

Ein vSphere Distributed Switch agiert als einzelner virtueller Switch für alle verbundenen Hosts. Dies ermöglicht eine konsistente Netzwerkkonfiguration der virtuellen Maschinen, wenn diese von Host zu Host migrieren.

Wie ein virtueller Switch ist ein verteilter virtueller Switch ein Netzwerk-Hub für virtuelle Maschinen. Ein vSphere Distributed Switch kann Datenverkehr intern zwischen virtuellen Maschinen leiten oder eine Verbindung zu einem externen Netzwerk herstellen.

Einem vSphere Distributed Switch können mehrere verteilte Portgruppen zugewiesen werden.
