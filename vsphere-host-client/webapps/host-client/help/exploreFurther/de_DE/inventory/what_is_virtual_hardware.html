Was ist virtuelle Hardware?

Ebenso wie ein physischer Computer über Hardwaregeräte wie CPU und Arbeitsspeicher verfügt, enthält eine virtuelle Maschine virtuelle Hardwaregeräte. Wenn Sie die Konfiguration einer virtuellen Maschine betrachten, finden sich dort eine virtuelle CPU, Arbeitsspeicher, Festplatten, CD-/DVD-Laufwerke, Diskettenlaufwerke, Ethernet-Adapter, Soundkarten usw. Jedes dieser virtuellen Geräte führt die gleichen Funktionen für die virtuelle Maschine aus wie die Hardware eines physischen Computers.

Die virtuelle Hardware lässt die virtuelle Maschine die physische Hardware entsprechend den Anforderungen aufzuteilen. Sie können beispielsweise zahlreiche virtuelle Maschinen auf einer einzigen physischen CPU ausführen, wobei jede VM mit einer eigenen virtuellen CPU angezeigt wird. Sie können eine virtuelle Maschine von einem Host auf einen anderen verschieben und die Zuordnungen zwischen den virtuellen und physischen Geräten automatisch ändern, während die virtuellen Geräte unverändert bleiben. Dank virtueller Hardware sind virtuelle Maschinen von den Details der physischen Hardware isoliert.

Eine vollständige Liste der virtuellen Geräte finden Sie in der Online-Hilfe von vSphere Web Client.
