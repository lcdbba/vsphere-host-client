Was ist eine virtuelle Maschine?

Eine virtuelle Maschine ist ein Softwarecomputer, auf dem – wie auf einem physischen Computer – ein Betriebssystem und Anwendungen ausgeführt werden. Das auf einer virtuellen Maschine installierte Betriebssystem wird Gastbetriebssystem genannt.

Jede virtuelle Maschine verfügt über virtuelle Geräte, die die gleichen Funktionen bereitstellen wie physische Hardware. Virtuelle Maschinen erhalten CPU- und Arbeitsspeicherressourcen, Grafikkarten, Speicherzugriff und Netzwerkkonnektivität über die Hosts, auf denen sie ausgeführt werden.

In vSphere werden virtuelle Maschinen auf Hosts oder Clustern ausgeführt. Auf einem Host oder Cluster können mehrere virtuelle Maschinen gleichzeitig ausgeführt werden.
