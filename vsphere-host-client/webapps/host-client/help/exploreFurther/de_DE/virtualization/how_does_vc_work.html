Wie funktioniert vSphere?

VMware vSphere ist ein System zur Verwaltung einer virtuellen Infrastruktur. VMware vCenter Server ist ein Tool, das verschiedene Hostserver verwaltet, auf denen mehrere virtuelle Maschinen ausgeführt werden.

Mit vCenter Server können Sie neue Server-VMs schnell bereitstellen und eine Bibliothek mit standardisierten Vorlagen für virtuelle Maschinen erstellen, damit neu bereitgestellte Systeme immer den Anforderungen Ihres Datencenters entsprechen.

Der vCenter Server lässt Sie die ausgeführten virtuellen Maschinen zwischen Hostservern zu migrieren, sodass Sie Wartungsaufgaben für Hardwarekomponenten mit minimalen Ausfallzeiten durchführen können. Mit weiteren Funktionen von vCenter Server können Sie die Arbeitslasten virtueller Maschinen auf verschiedene Hosts verteilen und virtuelle Maschinen für Hochverfügbarkeit und Notfallwiederherstellung verwalten.
