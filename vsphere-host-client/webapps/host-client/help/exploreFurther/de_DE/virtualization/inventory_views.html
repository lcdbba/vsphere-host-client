Was sind Bestandslistenstruktur-Ansichten?

Bestandslistenstruktur-Ansichten bieten hierarchische Ansichten von allen durch vCenter Server verwalteten Objekte, wie z. B. Datencenter, Ressourcenpools, Cluster, Netzwerke, Datenspeicher, Datenspeicher-Cluster, Vorlagen, Hosts und virtuelle Maschinen.

Es gibt vier Bestandslistenansichten:

  • Ansicht "Hosts & Cluster"
    Zeigt die Bestandslistenhierarchie aller Bestandslistenobjekte mit Ausnahme von Vorlagen, Netzwerken und Datenspeichern an.
  • Virtuelle Maschinen und Vorlagen
    Zeigt die Liste der virtuellen Maschinen und Vorlagen an.
  • Netzwerke
    Zeigt die Liste der Netzwerke an.
  • Datenspeicher und Datenspeicher-Cluster
    Zeigt die Liste der Datenspeicher und Datenspeicher-Cluster an.

Sie greifen auf die Bestandslistenstruktur-Ansichten zu, indem Sie auf der Startseite von vSphere Web Client auf Virtuelle Infrastruktur klicken.
