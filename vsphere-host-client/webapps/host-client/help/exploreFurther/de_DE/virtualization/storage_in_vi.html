Speicher in vSphere

Jede virtuelle Maschine verwendet eine virtuelle Festplatte zum Speichern ihres Betriebssystems, der Programmdateien und anderer Daten im Zusammenhang mit ihren Aktivitäten. Eine virtuelle Festplatte besteht aus einer großen physischen Datei oder mehreren Dateien, die wie jede andere Datei kopiert, verschoben, archiviert und gesichert werden kann. Virtuelle Festplattendateien werden auf einem dedizierten Speicherplatz auf verschiedenen physischen Speichersystemen abgelegt, einschließlich interner und externer Geräte eines Hosts, oder auf dem Netzwerkspeicher, der für die spezifische Aufgabe von Speichern und Schützen der Daten dediziert ist.

Ein Host kann Speichergeräte erkennen, auf die er Zugriff hat, und diese als Datenspeicher formatieren. Der Datenspeicher ist ein spezieller logischer Container, entsprechend einem Dateisystem auf einem logischen Volume, auf dem Hosts virtuelle Festplattendateien und andere Dateien ablegen, die die wesentlichen Komponenten einer virtuellen Maschine enthalten. Datenspeicher können die Besonderheiten verschiedener Speicherprodukte verbergen und ein uniformes Modell zum Speichern von Dateien virtueller Maschinen bereitstellen.
