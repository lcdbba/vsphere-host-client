Was ist eine vSphere Distributed Switch-Architektur?

Ein Netzwerk mit vSphere Distributed Switches wird von der Datencenterebene aus gesteuert. Ein vSphere Distributed Switch ist ein Objekt im Datencenter, das als virtueller Switch für alle verbundenen Hosts agiert. Dies ermöglicht eine konsistente Netzwerkkonfiguration der virtuellen Maschinen, wenn diese von Host zu Host migrieren.

Uplinks verbinden den vSphere Distributed Switch mit physischen Netzwerkadaptern auf den zugehörigen Hosts. Jeder Host kann für jeden Uplink-Port auf dem vSphere Distributed Switch einen physischen Netzwerkadapter mit einem vSphere Distributed Switch verbinden.

NICs virtueller Maschinen können über verteilte Portgruppen eine Verbindung zu vSphere Distributed Switches herstellen. Diese Gruppen fassen mehrere Ports unter einer gemeinsamen Konfiguration zusammen und bieten einen stabilen Ankerpunkt für virtuelle Maschinen, die eine Verbindung zu einem benannten Netzwerk herstellen.
