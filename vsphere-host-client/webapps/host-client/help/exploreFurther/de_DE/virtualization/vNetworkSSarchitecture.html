Was ist eine vSphere-Standard-Switch-Architektur?

Ein Netzwerk mit vSphere-Standard-Switches wird von der Hostebene aus gesteuert. Ein vSphere-Standard-Switch funktioniert in etwa so wie ein physischer Switch. Er erkennt, welche virtuellen Maschinen mit seinen virtuellen Ports verbunden sind, und leitet anhand dieser Informationen den Datenverkehr an die entsprechenden virtuellen Maschinen weiter. Eine oder mehrere Portgruppen stellen eine Verbindung zu einem Standard-Switch her und geben für jeden Port der Gruppe die Port-Konfigurationsoptionen an, wie z. B. die Bandbreite und die VLAN-Tagging-Richtlinien. Jede NIC einer virtuellen Maschine kann einer Portgruppe zugewiesen werden.
