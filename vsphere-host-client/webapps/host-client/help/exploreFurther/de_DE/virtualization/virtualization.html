Was ist Virtualisierung?

Mit Virtualisierung wird eine Abstraktionsschicht bezeichnet, die die enge Verbindung zwischen physischer Hardware und dem Betriebssystem aufhebt. Eine virtuelle Infrastruktur ist eine unternehmensweite Lösung, die hohe Computing-Leistung für maximale Ressourcennutzung und niedrigere Kosten bietet.

Virtuelle Maschinen sind die wichtigsten Elemente einer virtuellen Infrastruktur. Mithilfe der Virtualisierung können Sie mehrere virtuelle Maschinen mit heterogenen Betriebssystemen und Anwendungen voneinander isoliert auf dem gleichen physischen Computer ausführen.

Die Virtualisierung ermöglicht das dynamische Verschieben von Ressourcen und das Verlagern von Verarbeitungsleistung abhängig von den aktuellen Anforderungen der verschiedenen Komponenten. Dies ist möglich, da VMware Betriebssystem und Anwendungen von der Hardware trennt, auf der sie ausgeführt werden.
