Was ist vCenter Server?

VMware vCenter Server stellt eine einzige, zentrale Steuerungslösung für das Datencenter dar. Er wird auf einem Windows-Server ausgeführt, ermöglicht die zentrale Verwaltung der VMware ESX/ESXi-Hosts und bietet grundlegende Datencenterdienste wie Zugriffssteuerung, Leistungsüberwachung und Konfiguration.

vCenter Server fasst die Ressourcen von einzelnen Hosts zusammen, damit diese von den virtuellen Maschinen im gesamten Datencenter gemeinsam genutzt werden können. Er erreicht dies, indem er die Zuweisung virtueller Maschinen zu Hosts und die Zuweisung von Ressourcen zu virtuellen Maschinen innerhalb eines angegebenen Hosts auf Basis der vom Systemadministrator festgelegten Richtlinien verwaltet.

vCenter Server ermöglicht den Einsatz erweiterter vSphere-Funktionen wie vSphere DRS, vSphere HA und vMotion.
