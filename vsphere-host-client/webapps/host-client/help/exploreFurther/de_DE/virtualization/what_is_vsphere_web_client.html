Was ist vSphere Web Client?

vSphere Web Client ist die primäre Schnittstelle zur Erstellung, Verwaltung und Überwachung virtueller Maschinen sowie deren Ressourcen und Hosts. Die Komponente bietet darüber hinaus Konsolenzugriff auf virtuelle Maschinen.

vSphere Web Client ist eine Webanwendung, die auf einer Maschine mit Netzwerkzugriff auf Ihre vCenter Server-Installation installiert wird. Sie können vSphere Web Client auf einer Windows-Maschine installieren oder ihn als Teil der Linux-basierten vCenter Server Appliance bereitstellen. Sie greifen über einen Webbrowser auf vSphere Web Client zu.

Auch wenn alle vSphere-Aktivitäten von vCenter Server durchgeführt werden, verwenden Sie vSphere Web Client zur Überwachung, Verwaltung und Steuerung von vCenter Server.
