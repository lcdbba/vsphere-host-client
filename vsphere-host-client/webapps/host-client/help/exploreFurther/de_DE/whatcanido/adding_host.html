Hinzufügen eines Hosts

Ein Host ist ein Computer, der mithilfe von Virtualisierungssoftware virtuelle Maschinen ausführt.

Um dem Datencenter einen Host hinzuzufügen, müssen Sie über einen Computer verfügen, auf dem die ESX- oder ESXi-Software ausgeführt wird. Wenn Sie einen Host hinzufügen, wird dieser durch den vCenter Server verwaltet. Nachdem der Host hinzugefügt wurde, können Sie den Host und die darauf ausgeführten virtuellen Maschinen mit dem vSphere Web Client verwalten.

Um einen Host hinzuzufügen, wählen Sie das gewünschte Datencenter aus und wählen Sie dann Aktionen &gt Host hinzufügen.
