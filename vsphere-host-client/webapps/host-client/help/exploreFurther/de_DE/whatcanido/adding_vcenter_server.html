Hinzufügen eines vCenter Server-Systems

Wenn Sie sich beim vSphere Web Client anmelden, haben Sie Zugriff auf alle vCenter Server 5.1-Systeme, die während der Installation mit dem Lookup Service des Clients registriert wurden und für die Sie über die entsprechenden Berechtigungen verfügen. Es ist nicht nötig, vCenter Server 5.1-Systeme mit dem vSphere Web Client manuell zu registrieren.

Wenn Sie über keinen Zugriff auf ein vCenter Server 5.1-System verfügen, führen Sie Folgendes durch:

  • Stellen Sie sicher, dass das vCenter Server-System mit dem Lookup Service von vSphere Web Client registriert wurde.
  • Stellen Sie sicher, dass Ihnen der SSO-Administrator die Berechtigung erteilt hat, auf das vCenter Server-System zuzugreifen.

Um ein vCenter Server 5.0-System vom vSphere Web Client aus verwalten zu können, müssen Sie vCenter Server mit dem vSphere Web Client registrieren. Verwenden Sie das vSphere-Verwaltungstool, um ein vCenter Server 5.0-System mit einer unter Windows installierten vSphere Web Client-Instanz zu registrieren. Um ein vCenter Server 5.0-System mit einer auf der vCenter Server Appliance installierten vSphere Web Client-Instanz zu registrieren, verwenden Sie das Befehlszeilenskript admin-app.
