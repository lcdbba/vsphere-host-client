Klonen einer virtuellen Maschine

Bei einem Klon handelt es sich um die exakte Kopie einer virtuellen Maschine. Wenn Sie einen Klon erstellen, können Sie über vCenter Server das Gastbetriebssystem dieser virtuellen Maschine anpassen. Sie können den neuen Klon auf jedem Host im gleichen Rechenzentrum wie die ursprüngliche virtuelle Maschine ausführen.

Um eine virtuelle Maschine zu klonen, wählen Sie die virtuelle Maschine aus und wählen Sie dann Aktionen > Alle vCenter-Aktionen > Klonen. Der Assistent zum Klonen virtueller Maschinen leitet Sie Schritt für Schritt durch das Klonen einer virtuellen Maschine. Sie können eine virtuelle Maschine in jedem Betriebszustand klonen.

Während des Vorgangs müssen Sie folgende Aufgaben ausführen:

  • Festlegen eines VM-Namens, der für alle weiteren virtuellen Maschinen und Vorlagen im Datencenter eindeutig ist
  • Auswählen des Hosts oder Clusters, auf dem die neue virtuelle Maschine ausgeführt werden soll
  • Zuweisen eines Datenspeichers für die VM-Dateien

Während Sie den Assistenten zum Klonen virtueller Maschinen ausführen, haben Sie die Möglichkeit, das Gastbetriebssystem der neuen virtuellen Maschine anzupassen. Bei Auswahl dieser Option wird der Assistent zum Anpassen des Gastbetriebssystems gestartet.
