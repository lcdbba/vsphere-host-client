Erstellen von Clustern

Bei der Erstellung eines Clusters müssen Sie entscheiden, ob Sie die Funktionen vSphere HA oder vSphere DRS im Cluster nutzen möchten. Sie können eine (oder beide) dieser Optionen während der Erstellung des Clusters mithilfe des Assistenten aktivieren.

Zum Erstellen eines Clusters wählen Sie zunächst das Datencenter, dem Sie den Cluster hinzufügen möchten. Wählen Sie anschließend Aktionen > Neuer Cluster.

Fügen Sie Hosts, die sich bereits in der Bestandsliste befinden, zum Cluster hinzu, indem Sie den Cluster auswählen und Aktionen > Host in Cluster verschieben wählen. Wählen Sie anschließend die hinzuzufügenden Hosts aus. Sie können neue Hosts direkt zum Cluster hinzufügen, indem Sie den Cluster auswählen und Aktionen > Host hinzufügen wählen.
