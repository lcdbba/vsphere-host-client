Erstellen einer neuen virtuellen Maschine

Durch das Erstellen einer neuen virtuellen Maschine können Sie Optionen wie beispielsweise die Anzahl der Prozessoren, Netzwerkverbindungen sowie die Größe von Arbeits- und Festplattenspeicher anpassen. Sie können neue virtuelle Maschine auf Hosts oder in Clustern erstellen.

Bevor Sie eine virtuelle Maschine erstellen, sollten Sie entscheiden, auf welchem Host bzw. in welchem Cluster sich die neue virtuelle Maschine befinden soll und welches Gastbetriebssystem Sie auf der neuen virtuellen Maschine installieren möchten. Außerdem müssen Sie über die CD oder die Image-Dateien für die Installation verfügen. Ferner müssen Sie über geeignete vCenter Server-Berechtigungen zum Erstellen von virtuellen Maschinen verfügen.

Um eine neue virtuelle Maschine zu erstellen, wählen Sie in der Bestandsliste den Host oder Cluster aus, auf dem die neue virtuelle Maschine ausgeführt werden soll, und wählen Sie anschließend Aktionen > Neue virtuelle Maschine.

Eine neue virtuelle Maschine ähnelt einem physischen Computer mit einer leeren Festplatte. Nachdem Sie die virtuelle Maschine erstellt haben, müssen Sie ein Gastbetriebssystem darauf installieren. Sie können die Einstellungen der virtuellen Maschine jederzeit ändern.
