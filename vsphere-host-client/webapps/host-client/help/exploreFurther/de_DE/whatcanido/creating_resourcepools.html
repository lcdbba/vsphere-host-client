Erstellen von Ressourcenpools

Wenn Sie einen Ressourcenpool erstellen, können Sie für eine Gruppe von virtuellen Maschinen Ressourcen wie beispielsweise CPU- und Arbeitsspeicheranteile festlegen. Sie können Reservierungen, Grenzwerte und Anteile spezifizieren, die bestimmen, wie die Ressourcen zugewiesen werden.

Um einen Ressoucenpool zu erstellen, wählen Sie einen Host aus und wählen Sie anschließend Aktionen > Alle vCenter-Aktionen > Neuer Ressourcenpool oder wählen Sie einen Cluster aus und wählen Sie anschließend Aktionen > Neuer Ressourcenpool. Der Assistent zur Erstellung eines neuen Ressourcenpools leitet Sie durch die erforderlichen Schritte.

Nachdem Sie einen Ressourcenpool erstellt haben, können Sie diesem per Drag & Drop virtuelle Maschinen hinzufügen. Virtuelle Maschinen im Ressourcenpool nutzen gemeinsam die CPU- und Arbeitsspeicherressourcen gemäß der Konfiguration des Ressourcenpools.
