Erstellen von Vorlagen

Sie können eine Vorlage auf die folgenden Arten erstellen:

  • Konvertieren Sie eine virtuelle Maschine in eine Vorlage.
  • Klonen Sie eine virtuelle Maschine in eine Vorlage.
  • Klonen Sie eine vorhandene Vorlage.

Wenn Sie eine virtuelle Maschine in eine Vorlage klonen, wird eine neue Vorlage mit derselben Konfiguration wie die virtuelle Maschine erstellt, und die ursprüngliche virtuelle Maschine ist weiterhin in der Bestandsliste verfügbar. Ebenso gilt: Wenn Sie eine Vorlage klonen, wird eine Kopie der Vorlage erstellt, und die ursprüngliche Vorlage bleibt bestehen.

Um eine Vorlage zu erstellen, wählen Sie eine virtuelle Maschine aus der Bestandsliste aus und wählen Sie anschließend Aktionen > Alle vCenter-Aktionen > Vorlage > In Vorlage konvertieren oder Aktionen > Alle vCenter-Aktionen > Vorlage > In Vorlage klonen. Um eine vorhandene Vorlage zu klonen, wählen Sie die Vorlage aus und wählen Sie anschließend Aktionen > Klonen.

Nachdem Sie eine Vorlage erstellt haben, fügt sie vCenter Server zur Bestandslistenstruktur „Virtuelle Maschinen & Vorlagen“ oder zur Bestandsliste „VM-Vorlagen“ hinzu.
