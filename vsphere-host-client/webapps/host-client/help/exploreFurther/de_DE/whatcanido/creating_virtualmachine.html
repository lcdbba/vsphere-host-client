Erstellen einer virtuellen Maschine

Es gibt verschiedene Möglichkeiten zum Erstellen von virtuellen Maschinen, die Ihrer Bestandsliste hinzugefügt werden sollen. Sie können eine neue, leere virtuelle Maschine erstellen, einen physischen Computer in eine virtuelle Maschine konvertieren, eine virtuelle Maschine anhand einer Vorlage bereitstellen oder eine vorhandene virtuelle Maschine klonen.

Der vSphere Web Client bietet eine einfache und flexible Benutzeroberfläche, über die Sie virtuelle Maschinen erstellen können. Ein Assistent leitet Sie durch die erforderlichen Schritte, um eine vollständige und funktionierende virtuelle Maschine zu erstellen.

Eine virtuelle Maschine wird auf dem ausgewählten Host erstellt. Die virtuelle Maschine wird auf diesem Host ausgeführt und nutzt die Ressourcen des Hosts. Virtuelle Maschinen können innerhalb eines Datencenters von einem Host auf einen anderen verschoben werden.
