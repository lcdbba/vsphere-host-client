Anpassen des Gastbetriebssystems

Wenn Sie eine neue virtuelle Maschine aus einer Vorlage erstellen oder eine vorhandene virtuelle Maschine klonen, können Sie das neue Gastbetriebssystem anpassen. Sie können die Identitäts- und Netzwerkeinstellungen des Gastbetriebssystems der virtuellen Maschine anpassen, sodass diese in der Zielumgebung sofort einsatzfähig ist. Anschließend können Sie die Einstellungen der virtuellen Maschine in einer Spezifikation speichern, die Sie zu einen späteren Zeitpunkt wiederverwenden können.

Bevor Sie den Assistenten „Von Vorlage bereitstellen“ oder den Assistenten zum Klonen von virtuellen Maschinen abschließen, haben Sie die Option, das Gastbetriebssystem anzupassen. Wenn Sie diese Option wählen, werden Sie von einem Assistenten für die Anpassung durch die Konfigurationsoptionen geleitet.

Um auf die gespeicherten Anpassungsspezifikationen zuzugreifen, wählen Sie auf der Startseite von vSphere Web Client Regeln und Profile aus und wählen Sie anschließend VM-Anpassungsspezifikationen. Hier können Sie vorhandene Spezifikationen bearbeiten oder neue Spezifikationen erstellen, die für eine spätere Verwendung gespeichert werden.
