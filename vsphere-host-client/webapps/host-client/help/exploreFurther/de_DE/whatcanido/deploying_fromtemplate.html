Bereitstellen anhand einer Vorlage

Wenn Sie eine virtuelle Maschine anhand einer Vorlage bereitstellen, bietet vCenter Server die Möglichkeit, das Gastbetriebssystem dieser virtuellen Maschine anzupassen. Sie können die neue virtuelle Maschine auf jedem Host im gleichen Datencenter wie die Vorlage ausführen.

Der Assistent zum Bereitstellen von Vorlagen leitet Sie durch die erforderlichen Schritte. Wählen Sie zum Starten des Assistenten eine Vorlage aus und wählen Sie anschließend Aktionen > Virtuelle Maschine über diese Vorlage bereitstellen.

Während des Vorgangs müssen Sie folgende Aufgaben ausführen:

  • Festlegen eines VM-Namens, der für alle weiteren virtuellen Maschinen und Vorlagen im Datencenter eindeutig ist
  • Auswählen des Hosts oder Clusters, auf dem die neue virtuelle Maschine ausgeführt werden soll
  • Zuweisen eines Datenspeichers für die VM-Dateien

Bei der Ausführung des Assistenten zum Bereitstellen einer Vorlage können Sie das Gastbetriebssystem der neuen virtuellen Maschine anpassen. Bei Auswahl dieser Option wird der Assistent zum Anpassen des Gastbetriebssystems gestartet.
