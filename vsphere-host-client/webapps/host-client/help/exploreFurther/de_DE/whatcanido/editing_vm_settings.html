Bearbeiten von VM-Einstellungen

Das Dialogfeld „Virtuelle Maschine bearbeiten“ ist ein Editor, mit dem Sie die Hardwarekonfiguration und die Optionen von virtuellen Maschinen sowie die SDRS-Regeln und die vApp-Optionen ändern können.

In vielen Fällen muss die virtuelle Maschine vor Änderung der Einstellungen ausgeschaltet werden.

Um auf die Einstellungen einer virtuellen Maschine zuzugreifen, wählen Sie die zu bearbeitende virtuelle Maschine aus und wählen Sie anschließend Aktionen > Einstellungen bearbeiten.

Das Dialogfeld mit den VM-Eigenschaften verfügt über vier Registerkarten: Virtuelle Hardware, VM-Optionen, SDRS-Regeln und vApp-Optionen. Auf der Registerkarte Virtuelle Hardware können Sie virtuelle Hardwaregeräte hinzufügen, entfernen und neu konfigurieren. Auf der Registerkarte VM-Optionen können Sie die Konfigurationseinstellungen der virtuellen Maschine ändern. Auf der Registerkarte SDRS-Regeln können Sie die SDRS-Regeln der virtuellen Maschine ändern. Auf der Registerkarte vApp-Optionen können Sie OVF-Eigenschaften konfigurieren und die IP-Zuteilung und die Produktinformationen für diese virtuelle Maschine angeben.
