Importieren einer virtuellen Appliance

Eine virtuelle Appliance ist eine virtuelle Maschine, auf der bereits ein Betriebssystem und weitere Softwareanwendungen installiert sind. Nach dem Importieren stellt die virtuelle Appliance eine voll funktionsfähige virtuelle Maschine dar.

Virtuelle Appliances können aus verschiedenen Quellen stammen. Im VMware Virtual Appliance-Marktplatz finden Sie eine Vielzahl von virtuellen Appliances zum Download. VMware oder andere Hersteller bieten außerdem virtuelle Maschinen auf CD oder DVD an.

Falls Sie bereits eine virtuelle Appliance erworben haben, können Sie virtuelle Appliances von einer lokalen Festplatte installieren oder Sie können eine URL einer Website eingeben, wo Sie auf eine virtuelle Appliance zugreifen können.

Wählen Sie zum Starten des Assistenten einen Host oder Cluster aus und wählen Sie anschließend Aktionen > OVF-Vorlage bereitstellen.
