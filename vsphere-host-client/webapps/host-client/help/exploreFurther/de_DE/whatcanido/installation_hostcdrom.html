Installationsschritte: Host-CD-ROM

  1. Legen Sie den Datenträger mit dem Betriebssystem in das CD-ROM-Laufwerk des Hosts ein.
  2. Wählen Sie die virtuelle Maschine in der Bestandsliste aus und wählen Sie anschließend Aktionen > Einschalten.
  3. Klicken Sie auf der Registerkarte „Übersicht“ der virtuellen Maschine auf das Symbol „CD-/DVD-ROM verbinden“ und wählen Sie Mit CD-Laufwerk des Hosts verbinden.
  4. Klicken Sie auf den Link Konsole starten.
  5. Nachdem die virtuelle Maschine eingeschaltet wurde, installieren Sie das Gastbetriebssystem auf die gleiche Weise wie auf anderen Computern.
    Für die Installation von Windows XP werden zusätzliche Treiber benötigt. Anweisungen zur Installation spezifischer Betriebssysteme finden Sie im Handbuch für die Installation von Gastbetriebssystemen.
  6. Nachdem das Gastbetriebssystem installiert wurde und ausgeführt wird, installieren Sie VMware Tools. Dadurch verbessert sich die Leistung der virtuellen Maschine.
