Schritt-für-Schritt-Installationsanleitung

So installieren Sie ein Gastbetriebssystem:

  1. Stellen Sie sicher, dass das VMware Client-Integrations-Plug-In im Browser installiert ist.
  2. Wählen Sie in der Bestandsliste die gewünschte virtuelle Maschine aus.
  3. Wählen Sie Aktionen > Einstellungen bearbeiten aus, um das Dialogfeld „Virtuelle Maschine bearbeiten“ zu öffnen.
  4. Klicken Sie auf der Registerkarte Virtuelle Hardware auf das CD-/DVD-Laufwerk, in das die Betriebssystem-CD eingelegt werden soll.
  5. Wählen Sie im rechten Fensterbereich unter „Gerätetyp“ den Speicherort der CD/DVD-ROM oder der ISO-Image-Datei mit dem Betriebssystem.
  6. Wenn Sie Datenspeicher-ISO-Datei ausgewählt haben, wählen Sie die ISO-Datei aus. Aktivieren Sie für die beiden Optionen „Datenspeicher-ISO-Datei“ und „Hostgerät“ das Kontrollkästchen Verbindungsherstellung beim Start.
  7. Klicken Sie auf OK.
  8. Klicken Sie auf einen der folgenden Links, um spezifische Installationsanweisungen zu erhalten:
     
    • Client-CD-ROM-Laufwerk
    • Host-CD-ROM-Laufwerk
    • ISO-Image-Datei
