Installieren eines Gastbetriebssystems

Eine neue virtuelle Maschine ähnelt einem physischen Computer mit einer leeren Festplatte. Bevor Sie die Maschine verwenden können, müssen Sie ein Betriebssystem installieren. Das Betriebssystem auf einer virtuellen Maschine wird Gastbetriebssystem genannt.

Ein Gastbetriebssystem wird auf einer virtuellen Maschine auf die gleiche Weise installiert wie ein Betriebssystem auf einem physichen Computer. Sie müssen eine CD-ROM oder ein ISO-Image mit den Installationsdateien vom Anbieter des Betriebssystems beziehen. Ebenso wie bei einem physischen Computer ist für jede Installation eines Betriebssystems eine separate Lizenz erforderlich.

Nach der Installation des Gastbetriebssystems können Sie die virtuelle Maschine über den vCenter Server genauso nutzen wie einen physischen Computer. Sie können beispielsweise Anwendungen installieren oder den Betriebsstatus verwalten.
