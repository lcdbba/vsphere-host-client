Verwenden von Clustern

Mithilfe von Clustern können Sie die Hostnutzung optimieren und eine hohe Verfügbarkeit für virtuelle Maschinen sicherstellen. Cluster sorgen für eine ausgewogene Verteilung der Ressourcen zwischen den Hosts und verwalten den Lastenausgleich. Mit Clustern können Sie folgende VMware-Lösungen nutzen:

  • vSphere Distributed Resource Scheduler (DRS)
    vSphere DRS verteilt die Arbeitslasten virtueller Maschinen kontinuierlich auf alle ESX/ESXi-Hosts. Wenn eine virtuelle Maschine mehr CPU- oder Arbeitsspeicherressourcen erfordert, als der zugehörige Host bereitstellen kann, löst das vCenter Server-System vMotion-Migrationen zur Verteilung der Arbeitslast aus.
  • vSphere High Availability (HA)
    Mit vSphere HA können virtuelle Maschinen auf ESX/ESXi-Hosts nach einem Ausfall des Hosts automatisch wiederhergestellt werden. Wenn ein Host ausfällt, werden alle verknüpften virtuellen Maschinen sofort auf anderen Hosts im Cluster neu gestartet.
  • VMware Enhanced vMotion Compatibility (EVC)
    EVC hilft, die vMotion-Kompatibilität zwischen Hosts in einem Cluster sicherzustellen, indem sichergestellt wird, dass den virtuellen Maschinen von allen Hosts im Cluster der gleiche CPU-Funktionssatz präsentiert wird.

Sie können die Clusternutzung weiter optimieren, indem Sie Ressourcenpools erstellen, in denen die verfügbaren CPU- und Arbeitsspeicherressourcen eines Clusters hierarchisch partitioniert werden.
