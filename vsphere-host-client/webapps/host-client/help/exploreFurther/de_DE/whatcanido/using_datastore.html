Erstellen eines Datenspeichers

Sie können einen neuen Datenspeicher durch Formatieren von LUNs oder Mounten von NFS-Volumes auf einen vorhandenen Host erstellen. Darüber hinaus können Sie der Bestandsliste einen Host mit vorhandenen Datenspeichern hinzufügen. Sie können Ordner außerdem zum Erstellen logischer Gruppen von Datenspeichern für organisatorische Zwecke und zum Einstellen von Berechtigungen und Alarmen für die gesamte Datenspeichergruppe erstellen.

Um einen Datenspeicher zu erstellen, wählen Sie einen Host aus und wählen Sie anschließend Aktionen > Neuer Datenspeicher. Nachdem Sie die Datenspeicher erstellt haben, können Sie die VM-Dateien darin speichern.
