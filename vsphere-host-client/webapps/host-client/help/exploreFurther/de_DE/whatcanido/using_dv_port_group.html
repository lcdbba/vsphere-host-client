Verwenden verteilter Portgruppen

Sie fügen die verteilten Portgruppen den vSphere Distributed Switches hinzu. Sie können Richtlinien für Portgruppen bearbeiten, wie z. B. die Anzahl der Ports, die jeweiligen Zuweisungseigenschaften und das Verschieben des Live-Ports.
