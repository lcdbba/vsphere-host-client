Verwenden von Ordnern

Sie können die folgenden Bestandslistenobjekte in Ordnern gruppieren, um eine weitere Unterteilung der Objekte innerhalb Ihrer Bestandsliste vorzunehmen: Datencenter, Virtuelle Maschinen und Vorlagen, Hosts und Cluster, Datenspeicher und Datenspeicher-Cluster sowie Netzwerkobjekte.

Die Objekte innerhalb eines Ordners müssen den gleichen Typ aufweisen.

So erstellen Sie einen Ordner:

  • Wählen Sie ein vCenter Server-System aus und wählen Sie dann Aktionen > Neuer Ordner.
  • Wählen Sie ein Datencenter aus und wählen Sie anschließend Aktionen > Alle vCenter-Aktionen > Neuer Host- und Clusterordner oder Neuer Netzwerkordner oder Neuer Speicherordner oder Neuer VM- und Vorlagenordner.

Um ein Objekt zu einem Ordner hinzuzufügen, wählen Sie das Objekt aus und wählen Sie anschließend Aktionen > Verschieben nach. Wählen Sie den Zielordner für das Objekt aus.

Um einen Ordner zu löschen, wählen Sie den Ordner in der Bestandsliste aus und wählen Sie anschließend Aktionen > Alle vCenter-Aktionen > Aus der Bestandsliste entfernen. Wenn Sie einen Ordner löschen, werden alle Bestandslistenobjekte in diesem Ordner ebenfalls gelöscht. Handelt es sich bei einem solchen Objekt um einen Host, wird die Verbindung dieses Hosts zum vCenter Server-System getrennt.
