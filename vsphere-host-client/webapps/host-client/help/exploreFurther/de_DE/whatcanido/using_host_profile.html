Verwenden von Hostprofilen

In der Ansicht „Hostprofile“ von vSphere Web Client können Administratoren Hostprofile erstellen, bearbeiten oder löschen. Klicken Sie zum Zugreifen auf diese Ansicht von der Startseite von vSphere Web Client aus auf Regeln und Profile und klicken Sie anschließend auf Hostprofile.

Klicken Sie zum Durchführen weiterer Hostprofilaktionen, z. B. zum Anhängen eines Profils, mit der rechten Maustaste auf einen Host oder Cluster in der Bestandsliste und wählen Sie unter Alle vCenter-Aktionen > Hostprofile die entsprechende Aktion aus. Um Profilvorgänge auf Clusterebene durchzuführen, z. B. Prüfen auf Übereinstimmung, wählen Sie den Cluster aus, klicken Sie auf die Registerkarte Überwachen und klicken Sie anschließend auf Profil-Übereinstimmung.

Um ein Hostprofil zu verwenden, hängen Sie es an einen Host in der Bestandsliste an und wenden Sie es auf diesen an. Sie können gleichzeitig mehrere Hosts oder einen Cluster für das Anhängen eines Hostprofils auswählen.
