Verwenden des Navigators

Der linke Bereich von vSphere Web Client ist der Navigator, Ã¼ber den auf alle vCenter Server-Bestandslisten und andere vSphere-Funktionen zugegriffen werden kann.

So verwenden Sie den Objektnavigator:

  • Klicken Sie auf ein Element, um dessen Informationen im mittleren Bereich von vSphere Web Client anzuzeigen.
  • Klicken Sie auf ein Element mit >, um es im Navigatorbereich nach oben zu verschieben und darunter Kategorien verwandter Elemente anzuzeigen.
  • Klicken Sie auf eine Kategorie mit einer Anzahl, um deren Objekte im Navigatorbereich anzuzeigen.
