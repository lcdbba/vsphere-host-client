Verwenden von Ressourcenpools

Nachdem Sie einen Ressourcenpool erstellt haben, verwaltet vCenter Server die gemeinsam genutzten Ressourcen und weist sie virtuellen Maschinen in diesem Ressourcenpool zu. Mit Ressourcenpools können Sie folgende Aufgaben durchführen:

  • Weisen Sie virtuellen Maschinen auf dem gleichen Host oder im selben Cluster Prozessor- und Arbeitsspeicherressourcen zu.
  • Legen Sie minimale, maximale und proportionale Ressourcenanteile für CPU und Arbeitsspeicher fest.
  • Ändern Sie Zuteilungen, während die virtuellen Maschinen ausgeführt werden.
  • Ermöglichen Sie Anwendungen, dynamisch weitere Ressourcen anzufordern, um Spitzenlasten zu bewältigen.

Verwenden Sie vSphere Web Client, um Ressourcenpools virtuelle Maschinen hinzuzufügen, die Ressourcennutzung anzuzeigen und Ressourcenreservierungen, -grenzen und -anteile festzulegen.
