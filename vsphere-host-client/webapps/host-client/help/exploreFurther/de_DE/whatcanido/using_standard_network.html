Einrichten einer vSphere-Standard-Switch-Architektur

Das Einrichten einer Netzwerkumgebung unter Verwendung von vSphere-Standard-Switches kann auf der Registerkarte Netzwerk der Registerkarte Verwalten des Hosts durchgeführt werden. Portgruppen für das Netzwerk virtueller Maschinen oder für Netzwerkdienste, wie z. B. Servicekonsole-Netzwerke, vMotion und iSCSI-Netzwerke, werden mithilfe des Assistenten zum Hinzufügen von Netzwerken konfiguriert. Neue Standard-Switches können beim Erstellen der Portgruppe erstellt werden. Sie können Ihre neue Portgruppe aber auch einem bereits vorhandenen Standard-Switch zuordnen. Eine virtuelle Maschine wird mit einem virtuellen Netzwerk verbunden, indem die Netzwerkkarte der virtuellen Maschine der Portgruppe dieses Netzwerks zugewiesen wird.
