Verwenden von Speicherfunktionen

Sie können von der Ansicht „Speicherdienstklassen“ im vSphere Web Client aus auf die Speicherfunktionen zugreifen.

Klicken Sie auf das Symbol Speicherfunktion erstellen, bearbeiten oder entfernen, um vorhandene
systemdefinierte Speicherfunktionen anzuzeigen und eine benutzerdefinierte Speicherfunktion
hinzuzufügen, zu bearbeiten oder zu löschen.

Klicken Sie auf das Symbol Neue Speicherdienstklasse erstellen, um Speicherfunktionen zu einer neuen Speicherdienstklasse hinzuzufügen.

Klicken Sie mit der rechten Maustaste auf eine vorhandene Speicherdienstklasse und wählen Sie Speicherdienstklasse bearbeiten, um die Liste der Speicherfunktionen zu ändern, die eine Speicherdienstklasse zur Übereinstimmung benötigt.

Klicken Sie mit der rechten Maustaste auf einen Datenspeicher in der Bestandsliste und wählen Sie
Speicherfunktion zuweisen, um anzugeben, dass ein Datenspeicher über eine bestimmte Speicherfunktion verfügt.
