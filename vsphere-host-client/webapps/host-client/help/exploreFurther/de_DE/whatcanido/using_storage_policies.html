Verwenden von Speicherdienstklassen

Um einer virtuellen Maschine eine Speicherdienstklasse zuzuweisen, wählen Sie die virtuelle Maschine aus der Bestandsliste aus, klicken Sie auf die Registerkarte Verwalten und anschließend auf Dienstklassen.

vSphere Web Client zeigt Datenspeicher an, die kompatibel bzw. inkompatibel mit einer Speicherdienstklasse sind, wenn Sie die Speicherdienstklasse auf der Seite „Speicher“ der folgenden Assistenten auswählen:

  • Neue virtuelle Maschine
  • Virtuelle Maschine klonen
  • Virtuelle Maschine migrieren

Wenn eine virtuelle Maschine oder eine virtuelle Festplatte auf einem Datenspeicher ausgeführt wird, der mit der Speicherdienstklasse nicht kompatibel ist, können Sie diese auf einen kompatiblen Datenspeicher verschieben.

Klicken Sie zum Erstellen, Bearbeiten oder Löschen von Speicherdienstklassen auf der Startseite von vSphere Web Client auf Regeln und Profile und anschließend auf Speicherdienstklassen.
