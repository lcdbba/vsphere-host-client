Verwenden von Vorlagen

Nach der Erstellung einer Vorlage können Sie diese zum Bereitstellen neuer virtueller Maschinen auf einem beliebigen Host im Datencenter einsetzen. Mithilfe von Vorlagen können Sie folgende Aufgaben durchführen:

  • Legen Sie Vorlagen in gemeinsam genutztem Speicher ab, um die Zuverlässigkeit zu erhöhen.
  • Personalisieren Sie Vorlagen mit dem Namen und der Netzwerkadresse einer virtuellen Maschine.
  • Konfigurieren Sie Vorlagen mit bewährten Standards, und stellen Sie sie anschließend als virtuelle Maschine bereit.
  • Minimieren Sie Fehler und Ausfallzeiten bei der Erstellung von neuen virtuellen Maschinen.
  • Nutzen Sie Vorlagen, um Patches und Updates für virtuelle Maschinen bereitzustellen.
