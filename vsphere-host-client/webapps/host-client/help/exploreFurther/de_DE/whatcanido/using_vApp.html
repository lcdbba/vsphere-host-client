Verwenden von vApps

Eine vApp hat dieselben Grundfunktionen wie eine virtuelle Maschine, kann jedoch mehrere virtuelle Maschinen oder virtuelle vApps enthalten.

Wie individuelle virtuelle Maschinen sind vApps Bestandslistenobjekte, die Sie mit vSphere verwalten können. In vSphere Web Client finden Sie vApps in der vApps-Bestandsliste und in den Bestandslistenstrukturen „Hosts und Cluster“ und „VMs und Vorlagen“. Mithilfe dieser Ansichten können Sie vApps erstellen, konfigurieren, ein-und ausschalten sowie klonen - genauso wie individuelle virtuelle Maschinen

Wenn Sie die Startreihenfolge der virtuellen Maschinen in einer vApp angeben möchten, bearbeiten Sie die vApp-Eigenschaften.

vApps können importiert und exportiert werden. Das Distributionsformat einer vApp ist OVF.
