Einrichten einer vSphere Distributed Switch-Architektur

Beim Einrichten einer Netzwerkumgebung mithilfe von vSphere Distributed Switches besteht der erste Schritt darin, in der Netzwerk-Bestandslistenansicht des vSphere-Clients einen vSphere Distributed Switch zu erstellen. vSphere Distributed Switches werden auf Datencenterebene in der Netzwerkansicht des vSphere-Clients erstellt. Nachdem ein vSphere Distributed Switch erstellt wurde, können ihm Hosts und verteilte Portgruppen hinzugefügt werden.

Host-Netzwerkoptionen werden über die entsprechende Host-Netzwerkkonfigurationsseite oder unter Verwendung von Hostprofilen konfiguriert.

Schließlich werden NICs virtueller Maschinen mit den verteilten Portgruppen verbunden. Verbinden Sie die NICs virtueller Maschinen anhand der individuellen Konfiguration der NICs virtueller Maschinen oder durch das Migrieren des Netzwerks virtueller Maschinen vom vSphere Distributed Switch aus.
