Verwenden von virtuellen Maschinen

Virtuelle Maschinen sind die wichtigste Komponente in einer virtuellen Infrastruktur und können zur Verbesserung der Systemleistung in Ihrem Unternehmen eingesetzt werden. Mithilfe von virtuellen Maschinen können mehrere, heterogene Betriebssysteme und Anwendungen in einem Unternehmen gleichzeitig ausgeführt werden. So lässt sich die Hardware konsolidieren und die Zuverlässigkeit steigern.

Nachdem Sie ein Gastbetriebssystem und Anwendungen auf einer virtuellen Maschine installiert haben, können Sie den Betrieb der virtuellen Maschine über den vCenter Server verwalten. Sie können den Betriebsstatus der virtuellen Maschine ändern, die Konfiguration der virtuellen Maschine bearbeiten, die virtuelle Maschine auf einen anderen Host migrieren und der virtuellen Maschine virtuelle Hardwaregeräte hinzufügen.
