VMware Tools

VMware Tools besteht aus einer Reihe von Dienstprogrammen, die die Leistung des Gastbetriebssystems der virtuellen Maschine und die Verwaltung der virtuellen Maschine verbessern. Installieren Sie VMware Tools, nachdem Sie ein Gastbetriebssystem auf einer virtuellen Maschine installiert haben.

Die Installation der VMware Tools im Gastbetriebssystem ist von entscheidender Bedeutung. Das Gastbetriebssystem funktioniert zwar auch ohne VMware Tools, eine Vielzahl von wichtigen und praktischen Funktionen steht jedoch nicht zur Verfügung. Bei der Installation der VMware Tools werden der VMware Tools-Dienst sowie verschiedene VMware-Gerätetreiber installiert, die die Anzeige, Netzwerkfunktionen, SCSI, Speichersteuerung, E/A-Vorgänge und Mausunterstützung verbessern.

Installieren der VMware Tools

Bevor Sie VMware Tools installieren, stellen Sie sicher, dass das VMware Client-Integrations-Plug-In im Browser installiert ist.

  1. Klicken Sie mit der rechten Maustaste auf die virtuelle Maschine in der Bestandsliste und wählen Sie Alle vCenter-Aktionen > Gastbetriebssystem > VMware Tools installieren/aktualisieren.
  2. Klicken Sie auf Konsole starten auf der Registerkarte Übersicht der virtuellen Maschine.
  3. Befolgen Sie in der VM-Konsole die Anweisungen des Installationsassistenten.

Falls bei einer Windows-Installation der Assistent nicht automatisch angezeigt wird, öffnen Sie im Gastbetriebssystem das CD-ROM-Laufwerk mit der Bezeichnung "VMware Tools", und führen Sie die Installationsanwendung aus.

Bei Linux-Installationen folgen Sie den Anweisungen im Handbuch für die Installation von Gastbetriebssystemen.
