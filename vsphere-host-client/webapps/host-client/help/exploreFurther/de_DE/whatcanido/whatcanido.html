Verwenden von vSphere

vSphere ist eine Softwarelösung zur Bereitstellung und Verwaltung von virtuellen Maschinen in einem Datencenter. Mit vSphere können in einem Datencenter sofort Server bereitgestellt, Ressourcen global verwaltet und geplante Ausfallzeiten für die Hardwarewartung eliminiert werden.

Lesen Sie mehr zu folgenden Themen
  • Erstellen eines Datencenters
  • Hinzufügen eines Hosts
  • Erstellen einer virtuellen Maschine
  • Erstellen von Vorlagen
  • Verwenden von virtuellen Maschinen
  • Verwenden von Ordnern
  • Erstellen von Clustern
  • Erstellen von Ressourcenpools
  • Verwenden des Objektnavigators
