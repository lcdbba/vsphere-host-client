What is a host profile?

A host profile captures the configuration of a specific host. A host profile can be used to configure other hosts or to validate if a host's configuration meets the requirements set by the administrator. Host profiles greatly reduce the manual steps involved in configuring a host and can help maintain consistency and correctness in host configuration across the datacenter.
