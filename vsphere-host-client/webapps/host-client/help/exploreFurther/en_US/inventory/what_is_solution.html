What is a solution?

A vCenter solution is an extension of the vCenter Server that adds new functionality to a vCenter Server instance. For example, vSphere ESX Agent Manager is a standard vCenter solution provided by VMware that allows you to manage ESX agents adding new capabilities to ESX hosts.

You can install a solution to add functionality from third-party technologies to the standard functions of vCenter Server. Solutions are often delivered as OVF packages, and can be installed and deployed directly from the vSphere Web Client.
