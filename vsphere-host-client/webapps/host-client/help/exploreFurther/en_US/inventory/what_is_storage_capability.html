What is a storage capability?

A storage capability indicates the storage characteristics that a datastore provides. The characteristics can be capacity, performance, availability, and so on.

  • System-defined storage capabilities

    Storage systems that have Storage Awareness enabled assign system-defined storage capabilities to the datastores that are created on that system.

  • User-defined storage capabilities

    You manually create and assign user-defined storage capabilities to datastores to identify datastores that run on the same storage system or that have similar or identical characteristics for performance, capacity, availability, and so on.
