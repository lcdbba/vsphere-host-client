What is a VM storage policy?

A VM storage policy defines the storage capabilities required by the virtual machine and its virtual disks. You can assign different VM storage policies to the virtual machine files and the virtual disks depending on their storage needs.

After it is assigned to a virtual machine or a virtual disk, the VM storage policy performs a compliance check to verify that the virtual machine or the virtual disk uses the storage capabilities defined in the VM storage policy.
