What is an Asset?

An asset is any licensable object in vSphere.  Examples of assets are ESXi hosts, vCenter Server systems, and solutions such as, vCenter Operations Manager, vCloud Networking and Security, and others.

When initially installed, an asset has a 60 day evaluation period that allows you to explore the full set of features that are available for the asset’s product. For example, ESXi hosts have 60 day evaluation periods that allow you to use the features equal to the vSphere Enterprise Plus edition.  To use the features that you have configured for an asset during its evaluation period, you must assign the asset a license that unlocks these features.
