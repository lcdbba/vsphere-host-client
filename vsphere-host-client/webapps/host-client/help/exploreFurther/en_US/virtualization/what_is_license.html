What is a License?

A license is an object that stores a license key of a single VMware product or a product suite that contains multiple components. A license key encodes information about the features of its product, certain amount of capacity in cost units, and an expiration date. To use a license key, you create a license object and add the license key to that object. The license object inherits the characteristics of its license key. You can assign the license object to one or multiple assets.

If the license belongs to a single product, you can assign it to multiple assets associated with that product. If the license belongs to a suite, you can assign it to the assets that are associated with the suite.
