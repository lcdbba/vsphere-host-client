What is a Solution?

A solution is a VMware product that integrates its licensing with vCenter Server. Examples of solutions are vCenter Site Recovery Manager, vCloud Networking and Security, vCenter Operations Manager, and others. Solutions have their own licensing models that are independent from vSphere.

When installed, solutions register with a particular vCenter Server system. In the license management interface, solutions appear in the assets list for the vCenter Server system to which they are registered. From a vSphere licensing perspective, all assets except for vCenter Server systems and ESXi hosts are solutions.

To license a solution asset, you create a license object and add the license key of solution to that object. Then, you assign the license to one or more solution assets.
