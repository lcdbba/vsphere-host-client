What is a Suite?

A suite is a product that combines multiple components under a single license. The suite components provide the capabilities of different products that you can purchase standalone as well. For example, vCloud Suite combines vSphere, vCloud Director, vCloud Networking and Security, and others. You can obtain all components of the vCloud Suite through a single license key.

To unlock the components of a suite, you create a license object and add the suite license key to that object. Next, you assign the license to all the assets that are associated with the components of the suite. For example, if you create a license that stores a vCloud Suite Advanced license key, you assign the license to ESXi hosts, vCenter Site Recovery Manager, vCloud Networking and Security, and so on.
