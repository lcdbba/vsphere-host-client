Adding virtual hardware

You might need to add virtual hardware, such as DVD/CD-ROM drives, Ethernet adapters, or an additional hard drive, to your virtual machine after it is created.

In most cases, the virtual machine must be powered off for you to modify the virtual hardware.

To add virtual hardware devices, select the virtual machine in the inventory and select Actions > Edit Settings. In the Edit VM dialog box, click the Virtual Hardware tab, select a device, and click Add.

The new virtual hardware appears in the hardware list displayed in the Edit VM dialog box.
