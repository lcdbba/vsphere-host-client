Adding a vCenter Server system

When you log in to the vSphere Web Client, you have access to all vCenter Server 5.1 systems that were registered with the client's Lookup Service during installation and for which you have the correct permissions. There is no need to manually register vCenter Server 5.1 systems with the vSphere Web Client.

If you do not have access to a vCenter Server 5.1 system, do the following:

  • Verify that the vCenter Server system was registered with the vSphere Web Client's Lookup Service.
  • Verify that the SSO Administrator has granted you permission to access the vCenter Server system.

To manage a vCenter Server 5.0 system from the vSphere Web Client, you must register the vCenter Server with the vSphere Web Client. To register a vCenter Server 5.0 system with a vSphere Web Client instance installed on Windows, use the vSphere Administration Tool. To register a vCenter Server 5.0 system with a vSphere Web Client instance installed on the vCenter Server Appliance, use the admin-app command-line script.
