Creating a datastore cluster

When you create a datastore cluster, decide whether to enable vSphere Storage DRS so that virtual disk files can be automatically migrated to optimize resource use.

To create a datastore cluster, select a datacenter and select Actions > New Datastore Cluster. The New Datastore Cluster wizard guides you through the creation process.
