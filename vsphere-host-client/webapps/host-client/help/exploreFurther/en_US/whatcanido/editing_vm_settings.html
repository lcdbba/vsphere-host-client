Editing virtual machine settings

The Edit VM dialog box is an editor that you can use to change the hardware configuration of a virtual machine, virtual machine options, SDRS rules, and vApp options.

In many cases, the virtual machine must be powered off for you to modify its settings.

To access a virtual machine’s settings, select the virtual machine you want to edit and select Actions > Edit Settings.

The Virtual Machine Properties dialog box has four tabs: Virtual Hardware, VM Options, SDRS Rules, and vApp Options. The Virtual Hardware tab enables you to add, remove, and reconfigure virtual hardware devices. The VM Options tab enables you to modify configuration settings for the virtual machine. The SDRS Rules tab enables you to modify the SDRS rules for the virtual machine. The vApp Options tab enables you to configure OVF properties and specify IP allocation and product information for this virtual machine.
