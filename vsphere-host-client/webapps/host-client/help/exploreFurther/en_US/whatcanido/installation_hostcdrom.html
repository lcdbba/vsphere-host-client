Installation steps: Host CD-ROM

  1. Insert the operating system disk into the host CD-ROM drive.
  2. Select the virtual machine in the inventory and select Actions > Power On.
  3. On the virtual machine Summary tab, click the Connect CD/DVD-ROM icon and select Connect to host CD device.
  4. Click the Launch Console link.
  5. After the virtual machine is powered on, install the guest operating system as you would on any other computer.
    Windows XP installation requires additional drivers. For specific operating system installation instructions, see the Guest Operating System Installation Guide.
  6. After the guest operating system is installed and running, install VMware Tools. This improves the performance of your virtual machine.
