Step-by-step installation guide

To install a guest operating system:

  1. Ensure that the VMware Client Integration Plug-in is installed in your browser.
  2. Select the virtual machine in the inventory.
  3. Select Actions > Edit Settings to open the Edit VM dialog box.
  4. On the Virtual Hardware tab, click the CD/DVD drive that the operating system CD will be inserted into.
  5. Select the location of the operating system CD/DVD-ROM or ISO image file under Device Type in the pane on the right.
  6. If you selected Datastore ISO File, browse to the ISO file and select it. For both the Datastore ISO File and Host Device options, check the Connect at Power On checkbox.
  7. Click OK.
  8. Click one of the following links below for specific installation instructions:
     
    • Client CD-ROM Drive
    • Host CD-ROM Drive
    • ISO Image File
