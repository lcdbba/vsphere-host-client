Creating a datastore

You can create a new datastore by formatting LUNs or by mounting NFS volumes to an existing host. In addition, you can add a host with existing datastores to the inventory. You can also use folders to create logical groups of datastores for organizational purposes, and for setting permissions and alarms across the datastore group.

To create a datastore, select a host and select Actions > New Datastore. After you create the datastores, you can use them to store virtual machine files.
