Using host profiles

The Host Profile view is the management area of the vSphere Web Client for host profiles where administrators can create, edit, or delete host profiles. To access this view from the vSphere Web Client Home, click Policies and Profiles and then click Host Profiles.

To perform other host profile actions, such as attaching a profile, right-click a host or cluster in the inventory and select the appropriate action from All vCenter Actions > Host Profiles. To perform cluster-level profile operations, such as checking compliance, select the cluster, click the Monitor tab, and then click Profile Compliance.

To use a host profile, attach and apply it to a host in the inventory. You can select multiple hosts at one time or a cluster for attaching a host profile.
