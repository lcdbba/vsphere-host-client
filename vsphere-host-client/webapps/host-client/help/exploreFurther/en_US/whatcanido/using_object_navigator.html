Using the navigator

The left pane in the vSphere Web Client is the navigator, which provides access to all vCenter Server inventories and other vSphere functions.

To use the object navigator:

  • Click an item to see its information in the center pane of the vSphere Web Client.
  • Click an item with a > to bring it to the top of the navigator pane and display categories of related items beneath it.
  • Click a category with a count to display its objects in the navigator pane.
