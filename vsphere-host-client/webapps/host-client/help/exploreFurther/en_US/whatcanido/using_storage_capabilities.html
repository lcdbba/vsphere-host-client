Using storage capabilities

You can access storage capabilities from the VM Storage Policies view in the vSphere Web Client.

Click the Create, Edit or Remove Storage Capabilities icon to view existing
system-defined storage capabilities and add, edit, or delete a
user-defined storage capability.

Click the Create a new VM Storage Policy icon to add storage capabilities to a new VM storage policy.

Right-click an existing VM storage policy and select Edit VM Storage Policy to modify the list of storage capabilities that a VM storage policy requires for compliance.

Right-click a datastore in the inventory and select
Assign Storage Capability, to identify that a datastore has a certain storage capability.
