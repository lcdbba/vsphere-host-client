Using VM storage policies

To assign a VM storage policy to a virtual machine, select the virtual machine in the inventory, click the Manage tab, and then click VM Storage Policies.

The vSphere Web Client shows datastores that are compatible or incompatible with a VM storage policy as you select the VM storage policy in the Storage page of the following wizards:

  • New Virtual Machine
  • Clone Virtual Machine
  • Migrate Virtual Machine

If a virtual machine or a virtual disk run on a datastore that is incompatible with the VM storage policy, you can move them to a compatible datastore.

To create, edit, or delete VM storage policies, from the vSphere Web Client Home, click Policies and Profiles, and then click VM Storage Policies.
