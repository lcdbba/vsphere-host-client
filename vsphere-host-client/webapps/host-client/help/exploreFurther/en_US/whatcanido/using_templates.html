Using templates

After you create a template, you can use it to deploy new virutal machines to any host within the datacenter. Using templates, you can:

  • Store templates on shared storage for greater reliability.
  • Personalize templates with a machine name and network address.
  • Configure templates with established standards and then deploy them as virtual machines.
  • Minimize errors and downtime when creating new virtual machines.
  • Use templates for patching and updating virtual machines.
