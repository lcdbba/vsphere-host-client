Using vApps

A vApp has the same basic operations as a virtual machine but can contain multiple virtual machines or multiple vApps.

Like individual virtual machines, vApps are inventory objects you can manage using vSphere. In the vSphere Web Client, you can find vApps in the vApps inventory list and in the Hosts and Clusters and VMs and Templates inventory trees. From these views, you can create vApps, configure them, power them on and off, and clone them just as you can for individual virtual machines.

To specify the startup order of the virtual machines contained in a vApp, edit the vApp properties.

vApps can be imported and exported. The distribution format for a vApp is OVF.
