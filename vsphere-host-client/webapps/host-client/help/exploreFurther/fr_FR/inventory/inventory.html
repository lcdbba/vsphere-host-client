Explorer l'inventaire

L'inventaire est une hiérarchie d'objets qui permettent de gérer votre entreprise virtuelle. Ces objets sont les conteneurs d'autres objets, tels que des dossiers et des centres de données, ou des objets que vous gérez, comme des hôtes et des machines virtuelles. L'inventaire affiche tous les objets et leurs relations et permet d'organiser ces objets et d'accéder aux opérations vCenter qui les gèrent.

Il existe deux méthodes dans vSphere Web Client pour naviguer dans l'inventaire. La première méthode consiste à utiliser le navigateur. Dans le navigateur, les objets sont regroupés en catégories dans le panneau de gauche de vSphere Web Client. En sélectionnant une catégorie, vous pouvez afficher tous les objets de cette catégorie. En sélectionnant un objet individuel et en cliquant sur l'onglet Objets associés, vous pouvez afficher et parcourir tous les autres objets associés à cet objet.

Les arborescences d'inventaires constituent la deuxième méthode de navigation dans l'inventaire. Dans ces arborescences, les objets sont organisés hiérarchiquement dans l'une de ces quatre vues : Hôtes et clusters, VM et modèles, Stockage et Mise en réseau.
