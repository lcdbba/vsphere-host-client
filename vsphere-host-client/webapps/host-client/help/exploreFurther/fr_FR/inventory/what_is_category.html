Qu'est-ce qu'une catégorie ?

Une catégorie regroupe des balises associées dans l'inventaire de vCenter Server.

Lorsque vous créez une catégorie, vous précisez si plusieurs balises de cette catégorie peuvent être attribuées à un même objet ou si seulement une balise peut être attribuée à chaque objet. Par exemple, une catégorie intitulée Priorité peut contenir les balises Haute, Moyenne et Basse et peut être configurée afin que seulement une balise de cette catégorie puisse être appliquée à chaque objet.

Vous précisez également si les balises d'une catégorie peuvent être appliquées à tous les objets ou seulement à certains types d'objets, tels que les hôtes ou les centres de données.
