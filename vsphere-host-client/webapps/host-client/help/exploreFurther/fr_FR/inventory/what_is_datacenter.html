Qu'est-ce qu'un centre de données ?

un centre de données est le conteneur principal d'objets d'inventaire tels que des hôtes et des machines virtuelles. À partir du centre de données, vous pouvez ajouter et organiser des objets d'inventaire. Habituellement on ajoute des hôtes, des dossiers et des clusters à un centre de données.

vCenter Server peut contenir plusieurs centres de données. Les grandes entreprises emploient plusieurs centres de données pour représenter leurs unités d'organisation.

Les objets d'inventaire peuvent interagir dans les centres de données, mais l'interaction entre les centres de données est limitée. Par exemple, vous pouvez déplacer une machine virtuelle avec la technologie vMotion entre les hôtes d'un centre de données, mais vous ne pouvez pas la déplacer sur l'hôte d'un autre centre de données.
