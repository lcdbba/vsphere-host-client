Qu'est-ce qu'une banque de données ?

Une banque de données est un conteneur logique qui contient des fichiers de machine virtuelle et d'autres fichiers nécessaires pour les opérations de machine virtuelle. Les banques de données peuvent exister sous différents types de stockage physique, notamment mémoire locale, iSCSI, réseau SAN Fibre Channel ou NFS. Une banque de données peut être VMFS ou NFS.
