Qu'est-ce qu'un groupe de ports distribué ?

Un groupe de ports distribué est un groupe de ports lié à un commutateur distribué vSphere et spécifie les options de configuration de chaque port membre. Les groupes de ports distribués définissent le mode de connexion au réseau via le commutateur distribué vSphere.
