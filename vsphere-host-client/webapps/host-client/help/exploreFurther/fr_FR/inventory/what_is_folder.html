Qu'est-ce qu'un dossier ?

Un dossier est un conteneur utilisé pour affiner encore le groupement des objets dans votre inventaire. Les dossiers fournissent une structure naturelle sur laquelle appliquer des autorisations.

Par exemple, en utilisant les dossiers, vous pouvez organiser des machines virtuelles et des modèles selon leur fonction. De même, vous pouvez employer des dossiers pour grouper les centres de données selon leur situation géographique.

Vous pouvez grouper les objets d'inventaire suivants dans les dossiers :

  • Centres de données
  • Machines virtuelles (qui incluent les modèles)
  • Ressources informatiques (qui incluent les hôtes et les clusters)

Les objets groupés dans un dossier doivent être du même type.
