Qu'est-ce qu'un profil d'hôte ?

Un profil d'hôte capture la configuration d'un hôte spécifique. Un profil d'hôte peut être employé pour configurer d'autres hôtes ou pour valider si la configuration d'un hôte répond aux exigences définies par l'administrateur. Les profils d'hôte réduisent considérablement les étapes manuelles impliquées dans la configuration d'un hôte et peuvent aider à maintenir l'uniformité et l'exactitude de configuration de l'hôte à travers le centre de données.
