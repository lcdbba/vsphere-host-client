Qu'est-ce qu'un cluster de banque de données ?

Un cluster de banque de données est un ensemble de banques de données qui partage des ressources et une interface de gestion. Lorsque vous ajoutez une banque de données à un cluster de banque de données, les ressources du cluster deviennent partie intégrante des ressources du cluster de banque de données. Vous utilisez des clusters de banque de données pour agréger les ressources de stockage, ce qui vous permettra de prendre en charge les règles d'allocation de ressource au niveau du cluster de banque de données.

Quand vous créez un cluster de banque de données, vous pouvez utiliser Storage DRS pour gérer les ressources de stockage. La fonction d'équilibrage de charge E/S disponible avec Storage DRS n'est valable que quand tous les hôtes connectés aux banques de données dans le cluster de banque de données sont en cours d'exécution sur vSphere version 5 ou ultérieure.
