Qu'est-ce qu'une solution ?

Une solution vCenter est une extension de vCenter Server qui ajoute une nouvelle fonctionnalité à une instance vCenter Server. Par exemple, le gestionnaire d'agent vSphere ESX est une solution vCenter standard fournie par VMware. Elle permet de gérer les agents ESX qui ajoutent de nouvelles fonctions aux hôtes ESX.

Vous pouvez installer une solution pour ajouter une fonctionnalité de technologies tierces aux fonctions standard de vCenter Server. Les solutions sont souvent livrées sous forme de modules OVF qui peuvent être installés et déployés directement depuis vSphere Web Client.
