Qu'est ce qu'une capacité de stockage ?

Une capacité de stockage indique les caractéristiques de stockage qu'une banque de donnée fournit. Les caractéristiques peuvent être la capacité, les performances, la disponibilité etc.

  • Capacités de stockage définies par le système

    Les systèmes de stockage dont la reconnaissance de stockage est activée attribuent les capacités de stockage définies par le système aux banques de données qui sont créées sur ce système.

  • Capacités de stockage définies par l'utilisateur

    Vous créez manuellement et attribuez les capacités de stockage définies par l'utilisateur afin d'identifier les banques de données qui sont en cours d'exécution sur le même système de stockage ou qui ont des caractéristiques identiques ou similaires concernant les performances, capacité, disponibilité, etc.
