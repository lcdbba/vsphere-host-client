Qu'est-ce qu'un profil de stockage VM ?

Un profil de stockage VM définit les capacités de stockage requise par la machine virtuelle et ses disques virtuels. Vous pouvez assigner différents profils de stockage VM aux fichiers des machines virtuelles et aux disques virtuels selon leurs besoins.

Après avoir été assigné à une machine virtuelle ou à un disque virtuel, le profil de stockage VM effectue une vérification de conformité pour confirmer que la machine virtuelle ou le disque virtuel utilise bien les capacités de stockage définies par le profil de stockage VM.
