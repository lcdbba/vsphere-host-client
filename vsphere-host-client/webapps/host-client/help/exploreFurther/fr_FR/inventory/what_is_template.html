Qu'est-ce qu'un modèle ?

Un modèle est une image principale d'une machine virtuelle, utilisable pour créer et fournir de nouvelles machines virtuelles. Cette image inclut en général un système d'exploitation, des applications et une configuration de machine virtuelle qui définit un ensemble complet de matériels virtuels.

Vous pouvez créer un modèle en convertissant une machine virtuelle existante en modèle, en clonant une machine virtuelle en modèle ou en clonant un modèle existant. Vous pouvez ensuite créer de nouvelles machines virtuelles en déployant le modèle.

L'utilisation des modèles peut économiser le temps de configuration d'une nouvelle machine virtuelle et d'installation d'un système d'exploitation client.

Vous pouvez afficher les modèles en sélectionnant la catégorie Modèles de VM dans le navigateur d'objets ou en utilisant la vue d'arborescence d'inventaire VM et modèles.
