Qu'est-ce qu'un matériel virtuel ?

Au même titre qu'un ordinateur physique possède des périphériques matériels tels que le CPU et mémoire, une machine virtuelle possède des périphériques matériels virtuels. Si vous regardez la configuration d'une machine virtuelle, vous trouvez CPU, mémoire, disques durs, lecteurs de CD/DVD, lecteurs de disquette, cartes Ethernet, cartes son et ainsi de suite sous forme virtuelle. Chaque périphérique virtuel remplit la même fonction pour la machine virtuelle qu'un matériel sur un ordinateur physique.

Le matériel virtuel laisse la machine virtuelle diviser le matériel physique comme nécessaire. Par exemple, vous pouvez exécuter de nombreuses machines virtuelles sur un unique CPU physique, chacun semblant avoir son propre CPU virtuel. Vous pouvez déplacer une machine virtuelle d'un hôte à un autre et les mappages entre périphériques virtuels et physiques changent automatiquement, alors que les périphériques virtuels demeurent sans changement. À travers les matériels virtuels, les machines virtuelles sont isolées des détails de matériel physique.

Pour obtenir une liste complète des périphériques virtuels, consultez l'aide en ligne de vSphere Web Client.
