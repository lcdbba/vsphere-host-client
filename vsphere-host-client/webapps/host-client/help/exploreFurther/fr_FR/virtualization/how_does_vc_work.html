Comment vSphere fonctionne-t-il ?

VMware vSphere est un système de gestion de votre infrastructure virtuelle. vCenter Server de VMware est un outil qui gère plusieurs serveurs hôtes fonctionnant sur plusieurs machines virtuelles.

Avec vCenter Server, vous pouvez rapidement fournir de nouvelles machines virtuelles serveur et créer une bibliothèque de modèles de machines virtuelles normalisées ; ainsi, vos systèmes nouvellement fournis répondent toujours à vos exigences de centre de données.

vCenter Server permet de faire migrer les machines virtuelles en cours d'exécution entre des serveurs hôtes, de sorte que vous pouvez effectuer la maintenance matérielle avec une période d'interruption minimale. D'autres fonctions de vCenter Server permettent d'équilibrer des charges de travail de machine virtuelle sur les hôtes et de gérer les machines virtuelles pour la haute disponibilité et la récupération sur incident.
