Que sont les vues d'arborescence d'inventaire ?

Les vues d'arborescence d'inventaire offrent des vues hiérarchiques de tous les objets gérés par vCenter Server tels que les centres de données, les pools de ressources, les clusters, les réseaux, les banques de données, les clusters de banques de données, les modèles, les hôtes et les machines virtuelles.

Les quatre vues d'inventaire sont les suivantes :

  • Hôtes et clusters
    Affiche la hiérarchie d'inventaire de tout les objets d'inventaire à l'exception des modèles, réseaux et banques de données.
  • Machines virtuelles et modèles
    Affiche la liste des machines virtuelles et de modèles.
  • Réseaux
    Affiche la liste des réseaux.
  • Banques de données et clusters de banques de données
    Affiche la liste des banques de données et des clusters de banques de données.

Pour accéder aux vues d'arborescence d''inventaire, cliquez sur Infrastructure virtuelle depuis la page d'accueil de vSphere Web Client.
