Qu'est-ce que vCenter Server ?

vCenter Server de VMware fournit un unique point de contrôle pratique au centre de données. Il fonctionne sur un serveur Windows pour gérer centralement vos hôtes VMware ESX/ESXi et pour fournir des services essentiels de centre de données tels que le contrôle d'accès, le suivi des performances et la configuration.

vCenter Server unifie les ressources de différents hôtes pour les partager parmi les machines virtuelles dans tout le centre de données. Pour ce faire, il gère l'attribution des machines virtuelles aux hôtes et l'attribution des ressources aux machines virtuelles dans un hôte donné en fonction des stratégies que l'administrateur système définit.

vCenter Server permet d'utiliser les fonctions vSphere avancées, telles que vSphere DRS, vSphere HA et vMotion.
