Qu'est-ce que vSphere Web Client ?

vSphere Web Client est l'interface principale pour créer, gérer et contrôler des machines virtuelles, leurs ressources et leurs hôtes. Il permet également un accès par console aux machines virtuelles.

vSphere Web Client est une application Web installée sur une machine offrant un accès en réseau à votre installation vCenter Server. Vous pouvez installer vSphere Web Client sur une machine Windows ou le déployer dans le cadre du dispositif vCenter Server basé sur Linux. Vous pouvez accéder à vSphere Web Client à l'aide d'un navigateur Web.

Bien que vCenter Server exerce toutes les activités vSphere, vous pouvez utiliser vSphere Web Client pour surveiller, gérer et contrôler vCenter Server.
