Ajout de matériels virtuels

Il se peut que vous deviez ajouter du matériel virtuel, tel que des lecteurs de DVD/CD-ROM, des cartes Ethernet ou un disque dur supplémentaire, à votre machine virtuelle après l'avoir créée.

Dans la plupart des cas, la machine virtuelle doit être désactivée pour pouvoir modifier le matériel virtuel.

Pour ajouter des périphériques matériels virtuels, sélectionnez la machine virtuelle dans l'inventaire et choisissez Actions > Modifier les paramètres. Dans la boîte de dialogue Modifier VM, cliquez sur l'onglet Matériels virtuels, sélectionnez un périphérique et cliquez sur Ajouter.

Le nouveau matériel virtuel apparaît dans la liste de matériels qui s'affiche dans la boîte de dialogue Modifier VM.
