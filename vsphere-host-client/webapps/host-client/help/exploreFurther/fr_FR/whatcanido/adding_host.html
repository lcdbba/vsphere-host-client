Ajout d'un hôte

Un hôte est un ordinateur qui emploie le logiciel de virtualisation pour exécuter des machines virtuelles.

Pour ajouter un hôte au centre de données, vous devez avoir un ordinateur exécutant le logiciel ESX ou ESXi. Quand vous ajoutez un hôte, il est géré par vCenter Server. Une fois l'hôte ajouté, vous pouvez utiliser vSphere Web Client pour gérer l'hôte et les machines virtuelles qui s'exécutent sur celui-ci.

Pour ajouter un hôte, sélectionnez le centre de données auquel vous souhaitez ajouter l'hôte et choisissez Actions > Ajouter hôte.
