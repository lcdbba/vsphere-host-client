Ajout d'un système vCenter Server

Lorsque vous vous connectez à vSphere Web Client, vous avez accès à tous les systèmes vCenter Server 5.1 enregistrés avec Lookup Service du client pendant l'installation et pour lesquels vous disposez des autorisations adéquates. Il n'est pas nécessaire d'enregistrer manuellement les systèmes vCenter Server 5.1 dans vSphere Web Client.

Si vous n'avez pas accès à un système vCenter Server 5.1, procédez comme suit :

  • Assurez-vous que le système vCenter Server a été enregistré dans Lookup Service de vSphere Web Client.
  • Assurez-vous que l'administrateur SSO vous a accordé l'autorisation d'accéder au système vCenter Server.

Pour gérer un système vCenter Server 5.0 depuis vSphere Web Client, vous devez enregistrer vCenter Server dans vSphere Web Client. Pour enregistrer un système vCenter Server 5.0 dans une instance de vSphere Web Client installée sur Windows, utilisez l'outil d'administration vSphere. Pour enregistrer un système vCenter Server 5.0 dans une instance de vSphere Web Client installée sur le dispositif vCenter Server, utilisez le script de ligne de commande admin-app.
