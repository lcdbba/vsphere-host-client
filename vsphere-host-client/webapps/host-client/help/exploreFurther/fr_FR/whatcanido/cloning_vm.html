Clonage d'une machine virtuelle

Un clone est une copie exacte d'une machine virtuelle. Lorsque vous créez un clone, vCenter Server vous permet de personnaliser le système d'exploitation invité de cette machine virtuelle. Vous pouvez exécuter le nouveau clone sur n'importe quel hôte du même centre de données que la machine virtuelle d'origine.

Pour cloner une machine virtuelle, sélectionnez la machine virtuelle et choisissez Actions > Toutes les actions vCenter > Cloner. L'assistant Cloner une machine virtuelle vous guide au cours des étapes nécessaires pour cloner la machine virtuelle. Vous pouvez cloner une machine virtuelle dans n'importe quel état d'activation.

Pendant ce processus vous devez :

  • donner un nom à la machine virtuelle qui n'est pas donné aux autres machines virtuelles et modèles du centre de données ;
  • choisir l'hôte ou le cluster sur lequel la nouvelle machine virtuelle sera exécutée ;
  • affecter une banque de données pour les fichiers de machine virtuelle.

Pendant l'assistant Cloner une machine virtuelle, vous pouvez personnaliser le système d'exploitation client de la nouvelle machine virtuelle. Cette option lance l'assistant Personnalisation du système d'exploitation client.
