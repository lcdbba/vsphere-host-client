Création des clusters

Quand vous créez un cluster, déterminez si vous devez activer les solutions vSphere HA ou vSphere DRS sur le cluster. Vous pouvez activer l'une ou l'autre de ces options en utilisant l'assistant pendant la création du cluster.

Pour créer un cluster, choisissez le centre de données auquel ajouter le cluster. Sélectionnez ensuite Actions > Nouveau cluster.

Ajoutez au cluster des hôtes déjà présents dans l'inventaire ; pour cela, sélectionnez le cluster et choisissez Actions > Transférer les hôtes vers le cluster. Sélectionnez ensuite les hôtes à ajouter. Vous pouvez ajouter de nouveaux hôtes directement dans le cluster en sélectionnant Actions > Ajouter hôte.
