Création d'un centre de données

Créez un centre de données comme conteneur pour les hôtes, les machines virtuelles, les pools de ressources et les clusters. Les centres de données sont toujours créés sous le dossier racine dans l'inventaire Hôtes et clusters d'un seul vCenter Server, dans un dossier sous le dossier racine ou sous un objet vCenter dans un inventaire Mode lié.

Pour créer un centre de données, sélectionnez le système vCenter Server sur lequel vous souhaitez ajouter le centre de données et choisissez Actions > Nouveau centre de données.

La création d'un centre de données permet d'y ajouter des hôtes.
