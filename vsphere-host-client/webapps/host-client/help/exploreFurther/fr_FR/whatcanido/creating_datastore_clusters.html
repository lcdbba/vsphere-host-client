Créer un cluster de banque de données

Quand vous créez un cluster de banque de données, vous pouvez décider d'activer vSphere Storage DRS, ainsi les fichiers de disques virtuels peuvent être automatiquement migrés afin d'optimiser l'utilisation de ressources.

Pour créer un cluster de banque de données, sélectionnez un centre de données et choisissez Actions > Nouveau cluster de banque de données. L'assistant "Nouveau cluster de banque de données" vous guidera à travers le processus de création.
