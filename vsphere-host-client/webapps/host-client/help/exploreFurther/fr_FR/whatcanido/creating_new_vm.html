Création d'une nouvelle machine virtuelle

La création d'une nouvelle machine virtuelle permet de personnaliser des options telles que le nombre de CPU, de mémoire, de connexions réseau et de stockage. Vous pouvez créer de nouvelles machines virtuelles sur des hôtes ou des clusters.

Avant de créer une machine virtuelle, décidez sur quel hôte ou cluster la nouvelle machine virtuelle va résider, le type de système d'exploitation client que vous allez installer sur la nouvelle machine virtuelle, et l'emplacement du CD ou des fichiers d'image pour l'installation. Vous avez besoin également des autorisations appropriées vCenter Server pour créer des machines virtuelles.

Pour créer une nouvelle machine virtuelle, sélectionnez dans l'inventaire l'hôte ou le cluster sur lequel vous souhaitez exécuter la nouvelle machine virtuelle et choisissez Actions > Nouvelle machine virtuelle.

Une nouvelle machine virtuelle est comme un ordinateur physique avec un disque dur vierge. Après avoir créé la nouvelle machine virtuelle, vous devez installer un système d'exploitation client dessus. Vous pouvez également modifier les paramètres de la machine virtuelle à tout moment.
