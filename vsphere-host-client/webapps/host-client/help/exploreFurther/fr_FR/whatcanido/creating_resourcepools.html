Création des pools de ressources

Quand vous créez un pool de ressources, vous pouvez spécifier les ressources, telles que des partages de CPU et de mémoire, pour un ensemble de machines virtuelles. Vous pouvez spécifier les réservations, les limites et les partages qui déterminent comment les ressources sont allouées.

Pour créer un pool de ressources, sélectionnez un hôte et choisissez Actions > Toutes les actions vCenter > Nouveau pool de ressources ou sélectionnez un cluster et choisissez Actions > Nouveau pool de ressources. L'assistant Nouveau pool de ressources vous guide au cours du processus de création.

Après avoir créé un pool de ressources, vous pouvez y ajouter des machines virtuelles en les glissant sur l'objet de pool de ressources. Les machines virtuelles ajoutées à un pool de ressources partagent les ressources en CPU et mémoire selon la configuration du pool de ressources.
