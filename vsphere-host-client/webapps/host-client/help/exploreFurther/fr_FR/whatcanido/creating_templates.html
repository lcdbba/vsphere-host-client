Création des modèles

Vous pouvez créer un modèle avec les méthodes suivantes :

  • Convertir une machine virtuelle en modèle
  • Cloner une machine virtuelle dans un modèle
  • Cloner un modèle existant

Quand vous clonez une machine virtuelle dans un modèle, un nouveau modèle est créé avec la même configuration que la machine virtuelle, et la machine virtuelle d'origine reste disponible dans l'inventaire. De même, quand vous clonez un modèle, une copie du modèle est créée et le modèle d'origine demeure.

Pour créer un modèle, sélectionnez une machine virtuelle dans l'inventaire puis choisissez Actions > Toutes les actions vCenter > Modèle > Convertir en modèle ou Actions > Toutes les actions vCenter > Modèle > Cloner en modèle. Pour cloner un modèle existant, sélectionnez le modèle et choisissez Actions > Cloner.

Une fois que vous avez créé un modèle, vCenter Server l'ajoute à l'arborescence d'inventaire Machines virtuelles et modèles ou à la liste d'inventaire Modèles de VM.
