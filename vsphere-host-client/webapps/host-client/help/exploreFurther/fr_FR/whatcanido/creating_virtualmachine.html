Création d'une machine virtuelle

Vous pouvez employer plusieurs méthodes pour créer des machines virtuelles à ajouter à votre inventaire. Vous pouvez créer une nouvelle machine virtuelle vide, convertir un ordinateur physique en machine virtuelle, déployer à partir d'un modèle ou cloner une machine virtuelle existant.

vSphere Web Client propose une interface utilisateur simple et flexible depuis laquelle vous pouvez créer des machines virtuelles. Un assistant vous guide au cours des étapes pour produire une machine virtuelle complète et opérationnelle.

Quand vous créez une machine virtuelle, elle est créée sur l'hôte choisi. La machine virtuelle fonctionne sur cet hôte et utilise les ressources de cet hôte. Vous pouvez déplacer des machines virtuelles d'un hôte à un autre dans le même centre de données.
