Personnalisation du système d'exploitation client

Quand vous déployez une nouvelle machine virtuelle à partir d'un modèle ou clonez une machine virtuelle existante, vous pouvez personnaliser le nouveau système d'exploitation client. Vous pouvez personnaliser l'identité et les paramètres réseau du système d'exploitation client de votre machine virtuelle pour commencer à travailler immédiatement dans votre environnement cible. Vous pouvez ensuite enregistrer les paramètres de votre machine virtuelle dans une spécification que vous pouvez récupérer et réutiliser plus tard.

Avant de terminer l'assistant Déployer à partir d'un modèle ou Cloner une machine virtuelle, vous avez la possibilité de personnaliser le système d'exploitation invité. Si vous choisissez cette option, un assistant de personnalisation vous guide tout au long des options de configuration.

Pour accéder aux spécifications de personnalisation enregistrées, sélectionnez Règles et profils depuis la page d'accueil de vSphere Web Client puis choisissez Spécifications de personnalisation VM. De là, vous pouvez modifier des spécifications existantes ou en créer de nouvelles qui sont stockées pour un usage ultérieur.
