Déploiement à partir d'un modèle

Lorsque vous déployez une machine virtuelle à partir d'un modèle, vCenter Server vous permet de personnaliser le système d'exploitation invité de cette machine virtuelle. Vous pouvez exécuter la nouvelle machine virtuelle sur n'importe quel hôte du même centre de données que le modèle.

L'assistant Déployer un modèle vous guide au cours des étapes de déploiement d'un modèle. Pour démarrer l'assistant, sélectionnez un modèle puis choisissez Actions > Déployer la VM à partir de ce modèle.

Pendant ce processus vous devez :

  • donner un nom à la machine virtuelle qui n'est pas donné aux autres machines virtuelles et modèles du centre de données ;
  • choisir l'hôte ou le cluster sur lequel la nouvelle machine virtuelle sera exécutée ;
  • affecter une banque de données pour les fichiers de machine virtuelle.

Pendant l'assistant Déployer un modèle, vous pouvez personnaliser le système d'exploitation client de la nouvelle machine virtuelle. Cette option lance l'assistant Personnalisation du système d'exploitation client.
