Modification des paramètres d'une machine virtuelle

La boîte de dialogue Modifier VM est un éditeur que vous pouvez utiliser pour changer la configuration matérielle et les options d'une machine virtuelle, les règles SDRS et les options d'un vApp.

Dans beaucoup de cas, la machine virtuelle doit être désactivée pour pouvoir modifier ses paramètres.

Pour accéder aux paramètres d'une machine virtuelle, sélectionnez la machine que vous souhaitez modifier puis choisissez Actions > Modifier les paramètres.

La boîte de dialogue Propriétés de la machine virtuelle présente quatre onglets : Matériels virtuels, Options VM, Règles SDRS et Options vApp. L'onglet Matériels virtuels permet d'ajouter, supprimer et reconfigurer des périphériques matériels virtuels. L'onglet Options VM permet de modifier les paramètres de configuration de la machine virtuelle. L'onglet Règles SDRS permet de modifier les règles SDRS de la machine virtuelle. L'onglet Options vApp permet de configurer les propriétés OVF et de spécifier l'allocation IP ainsi que les informations de produit de cette machine virtuelle.
