Étapes d'installation : CD-ROM client

  1. Insérez le disque du système d'exploitation dans le lecteur de CD-ROM client.
  2. Sélectionnez la machine virtuelle dans l'inventaire puis choisissez Actions > Mettre sous tension.
  3. Dans l'onglet Résumé de la machine virtuelle, cliquez sur l'icône Connecter CD/DVD-ROM et sélectionnez Connecter à une image CD/DVD sur un disque local.
  4. Cliquez sur le lien Console de lancement.
  5. Une fois la machine virtuelle activée, installez le système d'exploitation client comme sur n'importe quel autre ordinateur.
    L'installation de Windows XP exige les pilotes supplémentaires. Pour des instructions d'installation propres au système d'exploitation, consultez le Guide d'installation du système d'exploitation client.
  6. Une fois le système d'exploitation client installé et opérationnel, installez VMware Tools. Ceci améliore les performances de votre machine virtuelle.
