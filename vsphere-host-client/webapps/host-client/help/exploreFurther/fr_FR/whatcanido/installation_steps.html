Guide d'installation pas à pas

Pour installer un système d'exploitation client :

  1. Assurez-vous que le plug-in d'intégration du client VMware est installé dans votre navigateur.
  2. Sélectionnez la machine virtuelle dans l'inventaire.
  3. Pour ouvrir la boîte de dialogue Modifier VM, sélectionnez Actions > Modifier les paramètres.
  4. Dans l'onglet Matériels virtuels, cliquez sur le lecteur CD/DVD dans lequel le CD du système d'exploitation va être inséré.
  5. Choisissez l'emplacement du CD/DVD-ROM ou fichier d'image ISO du système d'exploitation sous Type de périphérique dans le volet de droite.
  6. Si vous avez sélectionné Fichier ISO banque de données, accédez au fichier ISO et sélectionnez-le. Pour les options Fichier ISO de banque de données et Périphérique hôte, cochez la case Connecter lors de la mise sous tension.
  7. Cliquez sur OK.
  8. Cliquez sur un des liens suivants ci-dessous pour des instructions d'installation spécifiques :
     
    • Lecteur de CD-ROM du client
    • Lecteur de CD-ROM de l'hôte
    • Fichier d'image ISO
