Installation d'un système d'exploitation client

Une nouvelle machine virtuelle est comme un ordinateur physique avec un disque dur vierge. Avant de pouvoir l'utiliser, vous devez y installer un système d'exploitation. Un système d'exploitation sur une machine virtuelle s'appelle un système d'exploitation client.

L'installation d'un système d'exploitation client sur une machine virtuelle est globalement identique à une installation sur un ordinateur physique. Vous devez avoir un CD-ROM ou une image ISO contenant les fichiers d'installation d'un éditeur de systèmes d'exploitation. Comme avec les ordinateurs physiques, une licence de système d'exploitation distincte est exigée pour chaque installation.

Après avoir installé un système d'exploitation client, vCenter Server permet d'utiliser la machine virtuelle comme un ordinateur physique, pour des tâches telles que l'installation d'applications ou la gestion des opérations d'alimentation électrique.

Vous pouvez installer le système d'exploitation client à partir de plusieurs emplacements :
Lecteur de CD-ROM du client
Lecteur de CD-ROM de l'hôte
Fichier d'image ISO
