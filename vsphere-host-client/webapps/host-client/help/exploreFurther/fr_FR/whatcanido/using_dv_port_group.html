Utilisation des groupes de ports distribués

Vous ajoutez des groupes de ports distribués aux commutateurs distribués vSphere. Vous pouvez modifier les stratégies des groupes de ports tels que le nombre de ports, les propriétés d'affectation pour chacun et le déplacement de sports en temps réel.
