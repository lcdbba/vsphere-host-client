Utilisation des dossiers

Vous pouvez grouper les objets suivants de l'inventaire dans des dossiers pour raffiner encore le groupement des objets dans votre inventaire : les centres de données, les machines virtuelles et modèles, les hôtes et clusters, les banques de données et clusters de banques de données ainsi que les objets réseau.

Les objets groupés dans un dossier doivent être du même type.

Pour créer un dossier :

  • Sélectionnez un système vCenter Server, puis choisissez Actions > Nouveau dossier.
  • Sélectionnez un centre de données et choisissez Actions > Toutes les actions vCenter > Nouveau dossier d'hôte et de cluster, Nouveau dossier réseau, Nouveau dossier de stockage ou Nouveau dossier de VM et de modèle.

Pour ajouter un objet ou un dossier, sélectionnez l'objet et choisissez Actions > Transférer vers. Sélectionnez le dossier de destination de l'objet.

Pour supprimer un dossier, sélectionnez-le dans l'inventaire et choisissez Actions > Toutes les actions vCenter > Supprimer de l'inventaire. Si vous supprimez un dossier, vous supprimez tous les éléments d'inventaire qu'il contient. Si les éléments du dossier sont des hôtes, ils sont déconnectés du système vCenter Server.
