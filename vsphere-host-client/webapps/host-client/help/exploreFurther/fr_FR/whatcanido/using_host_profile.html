Utilisation des profils d'hôte

La vue Profil d'hôte est la zone de gestion de vSphere Web Client pour les profils d'hôte, où les administrateurs peuvent créer, modifier ou supprimer des profils d'hôte. Pour accéder à cette vue depuis la page d'accueil de vSphere Web Client, cliquez sur Règles et profils puis sur Profils d'hôte.

Pour effectuer d'autres opérations sur les profils d'hôte, telles qu'attacher un profil, cliquez avec le bouton droit de la souris sur un hôte ou un cluster dans l'inventaire et sélectionnez l'action appropriée dans Toutes les actions vCenter > Profils d'hôte. Pour effectuer des opérations sur les profils au niveau du cluster, telles que vérifier la conformité, sélectionnez le cluster, cliquez sur l'onglet Contrôler puis sur Conformité de profil.

Pour utiliser un profil d'hôte, attachez-le et appliquez-le à un hôte dans l'inventaire. Vous pouvez sélectionner plusieurs hôtes à la fois ou un cluster pour attacher un profil d'hôte.
