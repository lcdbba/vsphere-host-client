Utilisation du navigateur

Le panneau de gauche dans vSphere Web Client correspond au navigateur. Il permet d'accéder à tous les inventaires vCenter Server et aux autres fonctions de vSphere.

Pour utiliser le navigateur d'objets :

  • Cliquez sur un élément pour afficher ses informations dans le volet central de vSphere Web Client.
  • Cliquez sur un élément avec un > pour le placer en haut du volet de navigation et afficher les catégories d'éléments associés dessous.
  • Cliquez sur une catégorie avec un nombre pour afficher ses objets dans le volet de navigation.
