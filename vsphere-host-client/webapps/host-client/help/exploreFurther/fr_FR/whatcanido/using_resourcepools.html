Utilisation des pools de ressources

Une fois que vous avez créé un pool de ressources, vCenter Server gère les ressources partagées et les alloue aux machines virtuelles du pool de ressources. En utilisant les pools de ressources, vous pouvez :

  • allouer les ressources en CPU et mémoire aux machines virtuelles fonctionnant sur le même hôte ou cluster ;
  • établir les partages de ressources minimum, maximum et proportionnels pour le CPU et la mémoire ;
  • modifier les allocations pendant l'exécution des machines virtuelles ;
  • permettre aux applications d'acquérir dynamiquement plus de ressources pour répondre aux pics de performances ;

Utilisez vSphere Web Client pour ajouter des machines virtuelles à des pools de ressources, visualiser l'utilisation des ressources et définir les réservations, les limites et les partages des ressources.
