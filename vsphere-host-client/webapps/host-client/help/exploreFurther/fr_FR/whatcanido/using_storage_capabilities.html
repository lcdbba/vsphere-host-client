Utilisation des capacités de stockage

Vous pouvez accéder aux capacités de stockage à partir des profils de stockage VM dans vSphere Web Client.

Pour afficher les
capacités de stockage définies par le système et ajouter, modifier ou supprimer une
capacité de stockage définie par l'utilisateur, cliquez sur l'icône Créer, modifier ou supprimer des capacités de stockage.

Pour ajouter des capacités de stockage à un nouveau profil de stockage VM, cliquez sur l'icône Créer un nouveau profil de stockage VM.

Pour modifier la liste des capacités de stockage nécessaires pour qu'un profil de stockage VM soit conforme, cliquez avec le bouton droit de la souris sur un profil de stockage VM existant et sélectionnez Modifier le profil de stockage VM.

Pour déterminer si une banque de données a une certaine capacité de stockage, cliquez avec le bouton droit de la souris sur une banque de données dans l'inventaire et sélectionnez
Attribuer une capacité de stockage.
