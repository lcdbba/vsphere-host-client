Utilisation des profils de stockage VM

Pour attribuer un profil de stockage VM à une machine virtuelle, sélectionnez la machine virtuelle dans l'inventaire et cliquez sur l'onglet Gérer puis sur Profil.

Lorsque vous sélectionnez un profil de stockage VM dans la page Stockage des assistants suivants, vSphere Web Client présente les banques de données qui sont compatibles ou incompatibles avec un profil de stockage VM :

  • Nouvelle machine virtuelle
  • Cloner machine virtuelle
  • Migrer la machine virtuelle

Si une machine virtuelle ou un disque virtuel est en cours d'exécution sur une banque de donnée qui est incompatible avec le profil de stockage VM, vous pouvez alors les migrer vers une banque de données compatible.

Pour créer, modifier ou supprimer des profils de stockage VM, cliquez sur Règles et profils depuis la page d'accueil de vSphere Web Client puis sur Profils de stockage VM.
