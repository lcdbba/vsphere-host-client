Utilisation des modèles

Après avoir créé un modèle, vous pouvez l'utiliser pour déployer de nouvelles machines virtuelles à un hôte du centre de données. En utilisant les modèles, vous pouvez :

  • stocker les modèles sur un stockage partagé pour une plus grande fiabilité ;
  • personnaliser les modèles avec un nom d'ordinateur et une adresse réseau ;
  • configurer les modèles avec des normes établies, puis les déployer comme machines virtuelles ;
  • réduire au minimum les erreurs et les périodes d'interruption en créant de nouvelles machines virtuelles ;
  • utiliser les modèles pour corriger et mettre à jour des machines virtuelles.
