Utilisation des vApps

Un vApp possède les mêmes opérations de base qu'une machine virtuelle, mais elle peut contenir plusieurs machines virtuelles ou vApps.

Comme les machines virtuelles individuelles, les vApps sont des objets d'inventaire que vous pouvez gérer à l'aide de vSphere. Dans vSphere Web Client, les vApps se trouvent dans la liste d'inventaire vApps ainsi que dans les arborescences d'inventaire Hôtes et clusters et VM et modèles. À partir de ces views, vous pouvez créer des vApps, les configurer, les activer et les désactiver et les cloner comme pour les machines virtuelles individuelles.

Pour spécifier l'ordre de démarrage des machines virtuelles contenues dans un vApp, modifiez les propriétés de le vApp.

Les vApps peuvent être importées et exportées. Le format de distribution d'un vApp est Open Virtualization Format (OVF).
