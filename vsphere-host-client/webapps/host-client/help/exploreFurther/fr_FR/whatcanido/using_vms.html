Utilisation des machines virtuelles

Les machines virtuelles sont l'élément clé d'une infrastructure virtuelle et peuvent être utilisées pour améliorer la performance au sein de votre entreprise. Les machines virtuelles permettent le fonctionnement simultané de plusieurs systèmes d'exploitation et applications hétérogènes au sein d'une entreprise avec un matériel physique groupé et une fiabilité accrue.

Après avoir installé un système d'exploitation client et des applications sur une machine virtuelle, utilisez vCenter Server pour gérer son fonctionnement. Vous pouvez changer l'état d'activation d'une machine virtuelle, en modifier la configuration, la faire migrer vers un autre hôte et y ajouter des périphériques matériels virtuels.
