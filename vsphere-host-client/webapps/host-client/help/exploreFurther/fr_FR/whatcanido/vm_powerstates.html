États d'activation d'une machine virtuelle

L'état d'activation d'une machine virtuelle indique si cette dernière est active et fonctionnelle. Une machine virtuelle a plusieurs états d'activation, mais les trois états de base sont les suivants : actif, inactif et interrompu. Des fonctions supplémentaires existent en plus des opérations d'activation de base des machines virtuelles, mais VMware Tools doit d'abord être installé dans la machine virtuelle pour les utiliser.

Vous pouvez utiliser plusieurs points d'accès pour changer ces états d'activation :

  • Pour modifier manuellement un état d'activation, cliquez avec le bouton droit de la souris sur la machine virtuelle et sélectionnez l'option d'activation appropriée dans le menu.
  • Pour programmer un changement automatique d'état d'activation, appuyez sur la touche CTRL, cliquez avec le bouton droit de la souris sur la machine virtuelle puis sélectionnez l'option d'activation appropriée dans le menu.

Si une option d'état d'activation n'est pas actuellement disponible, elle est désactivée et elle ne peut pas être choisie.
