VMware Tools

VMware Tools est une suite d'utilitaires qui accroît les performances du système d'exploitation client de la machine virtuelle et améliore la gestion de cette dernière. Installez VMware Tools après avoir installé un système d'exploitation client sur une machine virtuelle.

L'installation de VMware Tools dans le système d'exploitation client est essentiel. Bien que le système d'exploitation client puisse fonctionner sans VMware Tools, vous perdez beaucoup en fonctionnalité et en commodité. Quand vous installez VMware Tools, vous installez le service VMware Tools et un ensemble de pilotes de périphériques VMware qui améliorent la prise en charge des affichages, de la gestion de réseau, des périphériques SCSI, de la gestion de mémoire, des E/S et des souris.

Installation de VMware Tools

Avant d'installer VMware Tools, assurez-vous que le plug-in d'intégration du client VMware est installé pour votre navigateur.

  1. Cliquez avec le bouton droit de la souris sur la machine virtuelle dans l'inventaire et sélectionnez Toutes les actions vCenter > SE invité > Installer/Mettre à niveau VMware Tools.
  2. Cliquez sur Console de lancement dans l'onglet Résumé de la machine virtuelle.
  3. Dans la console de machine virtuelle, suivez les instructions de l'assistant d'installation.

Pour les installations Windows, si l'assistant n'apparaît pas automatiquement, ouvrez le lecteur de CD-ROM dans le système d'exploitation client intitulé VMware Tools et exécutez l'application Setup.

Pour les installations Linux, suivez les instructions du Guide d'installation du système d'exploitation client.
