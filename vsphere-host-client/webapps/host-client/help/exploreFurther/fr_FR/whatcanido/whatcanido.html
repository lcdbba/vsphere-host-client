Utilisation de vSphere

vSphere est une solution logicielle de déploiement et de gestion des machines virtuelles dans un centre de données. Avec vSphere, les centres de données peuvent immédiatement dimensionner des serveurs, gérer globalement les ressources et éliminer les périodes d'interruption programmées pour maintenance matérielle.
