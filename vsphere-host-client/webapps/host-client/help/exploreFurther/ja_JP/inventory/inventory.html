インベントリについて

インベントリは、オブジェクトの階層で、このオブジェクトによって仮想エンタープライズを管理できます。このオブジェクトは、ほかのオブジェクトのコンテナ（フォルダ、データセンターなど）、または管理対象のオブジェクト（ホスト、仮想マシンなど）のいずれかです。インベントリを使用することにより、これらのオブジェクトの整理、およびオブジェクト管理を行う vCenter の操作ができます。

vSphere Web Client でのインベントリへの移動方法には 2 つの方法があります。1 つ目はナビゲータです。ナビゲータでは、オブジェクトは vSphere Web Client の左ペイン内のカテゴリにグループ分けされます。カテゴリを選択すると、カテゴリ内のすべてのオブジェクトが表示されます。個々のオブジェクトを選択して [関連オブジェクト] タブをクリックすると、そのオブジェクトに関連する他のオブジェクトをすべて表示し、参照することができます。

インベントリへの移動の 2 つ目の方法はインベントリ ツリーです。これらのツリーでは、オブジェクトは [ホストおよびクラスタ]、[仮想マシンおよびテンプレート]、[ストレージ]、[ネットワーク] の 4 つのビューのいずれかで階層的に表示されます。
