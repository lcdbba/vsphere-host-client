カテゴリについて

カテゴリは、vCenter Server インベントリで、関連するタグをグループ化します。

カテゴリを作成する場合、そのカテゴリ内の複数のタグを特定のオブジェクトに一度に割り当てることができるかどうか、またはオブジェクトに一度に割り当てるタグを 1 つに限定するかどうかを指定します。たとえば、優先順位 と呼ばれるカテゴリで高、中、低のタグを使用したり、オブジェクトにカテゴリ内で一度に割り当てられるタグを 1 つに限定することもできます。

また、カテゴリのタグをすべてのオブジェクトに割り当てられるかどうかを指定したり、ホストやデータストアといった特定のオブジェクト タイプに割り当てを限定することもできます。
