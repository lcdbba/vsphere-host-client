データセンターについて

データセンターは、ホストや仮想マシンなどのインベントリ オブジェクトの主要コンテナです。データセンターから、インベントリ オブジェクトの追加および整理ができます。通常、データセンターにホスト、フォルダ、およびクラスタを追加します。

vCenter Server には、複数のデータセンターを格納できます。大企業では、複数のデータセンターを使用して、企業の組織単位に分けることがあります。

インベントリ オブジェクトはデータセンター内で互いにやりとりできますが、データセンター間のやりとりは制限されています。たとえば、vMotion テクノロジーを使用して、仮想マシンをデータセンター内のホスト間で移動できますが、別のデータセンターのホストに移動することはできません。
