フォルダについて

フォルダとは、インベントリ内でオブジェクトをさらに細かくグループ化するために使用するコンテナです。フォルダは、自然な構造を提供し、権限を適用できます。

たとえば、フォルダを使用して、仮想マシンおよびテンプレートを機能別に整理できます。同様に、地理的な場所ごとにデータセンターをグループ化するためにフォルダを使用できます。

フォルダで次のインベントリ オブジェクトをグループ化できます。

  • データセンター
  • 仮想マシン（テンプレートを含む）
  • コンピュータ リソース（ホストおよびクラスタを含む）

フォルダ内でグループ化するオブジェクトのタイプは、同じにする必要があります。
