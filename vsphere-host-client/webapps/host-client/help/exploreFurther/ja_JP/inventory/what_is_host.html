ホストについて

ホストは、仮想化ソフトウェアを使用して仮想マシンを実行するコンピュータです。通常、ホストは ESX または ESXi ソフトウェアを実行するコンピュータです。

ホストは、仮想マシンが使用する CPU およびメモリ リソースを提供し、さらに仮想マシンがストレージおよびネットワークにアクセスできるようにします。同一のホスト上で同時に複数の仮想マシンを操作できます。

vCenter Server にホストを追加すると、ホストおよびホスト上で動作する仮想マシンを管理できます。
