リソース プールについて

リソース プールを使用すると、スタンドアロン ホストまたはクラスタのリソースをより小さなプールに分割できます。リソース プールは、そのリソース プール内で動作する仮想マシンが共有する、一連の CPU およびメモリ リソースで構成されています。リソース プールには必要なものが完備されており、ほかのリソース プールから隔離されています。

CPU およびメモリ容量を集約した単一のリソース プールに複数の物理サーバを統合できます。

仮想マシンは、リソース プールで実行され、リソース プールからリソースを取得します。これにより、リソース プール間で継続的に仮想マシンのワークロードのバランスをとることができます。ワークロードが増加すると、vCenter Server は自動的に追加リソースを割り当て、リソース プールのホスト間で仮想マシンを透過的に移行します。
