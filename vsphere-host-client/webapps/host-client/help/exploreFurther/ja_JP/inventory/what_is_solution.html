ソリューションについて

vCenter ソリューションは、vCenter Server インスタンスに新しい機能を追加する、vCenter Server の拡張機能です。たとえば、vSphere ESX Agent Manager は当社が提供する標準の vCenter ソリューションであり、これを使用すると、ESX エージェントを管理でき、ESX ホストに新しい機能が加わります。

サードパーティ製のテクノロジーからソリューションを vCenter Server の標準機能にインストールして、機能を追加することもできます。通常、ソリューションは OVF パッケージとして配信され、vSphere Web Client から直接インストールし、デプロイすることができます。
