テンプレートについて

テンプレートは、新規仮想マシンの作成およびプロビジョニングに使用できる、仮想マシンのマスタ イメージです。通常、このイメージには、オペレーティング システム、アプリケーション、および仮想マシンの仮想ハードウェア一式をすべて定義する構成設定が含まれます。

既存の仮想マシンをテンプレートに変換したり、仮想マシンをテンプレートにクローン作成したり、既存のテンプレートをクローン作成したりすることによりテンプレートを作成できます。そのあとテンプレートをデプロイすることにより新規仮想マシンが作成できます。

テンプレートを使用すると、新規仮想マシンの構成およびゲスト OS のインストールの時間が短縮できます。

テンプレートを表示するには、オブジェクト ナビゲータの [仮想マシン テンプレート] カテゴリを選択するか、[仮想マシンおよびテンプレート] インベントリ ツリー ビューを使用します。
