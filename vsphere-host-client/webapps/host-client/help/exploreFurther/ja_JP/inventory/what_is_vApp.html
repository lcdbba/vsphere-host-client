vApp について

vApp は、1 つのオブジェクトとして管理できる仮想マシンのグループです。vApp は、複数の相互に依存する仮想マシン上で実行される複雑な多重階層アプリケーションの管理を簡単にします。vApp の基本操作は、仮想マシンやリソース プールと同じです。vApp では、vApp 内の仮想マシンのパワーオン順序を設定したり、vApp 内の仮想マシンに IP アドレスを自動的に割り当てたり、アプリケーション レベルのカスタマイズを行うことができます。
