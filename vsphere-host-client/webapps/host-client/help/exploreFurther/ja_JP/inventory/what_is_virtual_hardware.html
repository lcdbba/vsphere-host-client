仮想ハードウェアについて

物理コンピュータに CPU、メモリなどのハードウェア デバイスがあるのと同様に、仮想マシンにも仮想ハードウェア デバイスがあります。たとえば、仮想マシンの構成では、仮想 CPU、メモリ、ハード ディスク、CD/DVD ドライブ、フロッピー ドライブ、イーサネット アダプタ、サウンド カードなどがあります。各仮想デバイスは、物理コンピュータ上のハードウェアと同様に仮想マシンに対して機能します。

仮想ハードウェアにより、仮想マシンが必要に応じて物理ハードウェアを分配できます。たとえば、単一の物理 CPU で、多数の仮想マシンを各仮想マシンに独自の仮想 CPU が搭載されているように実行できます。ホスト間で仮想マシンを移動すると、仮想デバイスと物理デバイスとのマッピングは自動的に変更されますが、仮想デバイスは変更されません。仮想マシンは、仮想ハードウェアを介しているため、物理ハードウェア全体からは隔離されています。

仮想デバイスの全リストについては、vSphere Web Client のオンライン ヘルプを参照してください。
