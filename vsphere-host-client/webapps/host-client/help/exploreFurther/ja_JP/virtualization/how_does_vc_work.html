vSphere の機能

VMware vSphere は、仮想インフラストラクチャを管理するためのシステムです。VMware vCenter Server は、複数の仮想マシンを実行する複数のホスト サーバを管理するツールです。

vCenter Server を使用すると、新規サーバ仮想マシンをすばやくプロビジョニングして標準化した仮想マシン テンプレートのライブラリを作成できるため、新しくプロビジョニングしたシステムは常にデータセンター要件に準拠します。

vCenter Server では動作中の仮想マシンをサーバ間で移行できるため、最小限のダウンタイムでハードウェアのメンテナンスを行うことができます。その他の vCenter Server 機能として、ホスト間で仮想マシンのワークロードのバランスをとったり、高可用性および災害復旧用に仮想マシンを管理したりすることができます。
