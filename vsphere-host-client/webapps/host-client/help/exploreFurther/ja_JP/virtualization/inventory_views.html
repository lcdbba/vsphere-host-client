インベントリ ツリー ビューについて

インベントリ ツリー ビューは、vCenter Server が管理するすべてのオブジェクト (データセンター、リソース プール、クラスタ、ネットワーク、データストア、データストア クラスタ、テンプレート、ホスト、および仮想マシン) の階層ビューを提供します。

次の 4 つのインベントリ ビューがあります。

  • [ホストおよびクラスタ]
    テンプレート、ネットワーク、およびデータストアを除く、すべてのインベントリ オブジェクトのインベントリ階層を表示します。
  • 「仮想マシンおよびテンプレート」
    仮想マシンおよびテンプレートのリストを表示します。
  • 「ネットワーク」
    ネットワークのリストを表示します。
  • 「データストアおよびデータストア クラスタ」
    データストアおよびデータストア クラスタのリストを表示します。

インベントリ ツリー ビューにアクセスするには、vSphere Web Client ホーム ページから [仮想インフラストラクチャ] をクリックします。
