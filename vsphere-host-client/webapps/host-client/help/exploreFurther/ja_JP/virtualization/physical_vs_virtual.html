物理マシンと仮想マシン

仮想マシンとは、物理コンピュータのようにオペレーティング システムとアプリケーションを実行するソフトウェア コンピュータです。仮想マシンには、ゲスト OS およびそのアプリケーションを実行する、独自の仮想ハードウェア一式があります。オペレーティング システムは、実際の物理ハードウェア コンポーネントにかかわらず、一貫性のある一連のハードウェアを検出します。

仮想マシンはエミュレータまたはシミュレータではありません。仮想マシンは、物理コンピュータと同等かそれ以上のことを実行できる、実際のマシンです。仮想マシンに柔軟性があるため、物理コンピュータは、サービス（アプリケーション、データベースなど）を自ら提供するよりも、これらのサービスを提供する仮想マシンの格納用として利用されています。
