vCenter Server について

VMware vCenter Server によって、データセンターが便利な単一の制御ポイントとして機能するようになります。これは Windows サーバ上で動作し、VMware ESX/ESXi ホストを集中管理したり、アクセス制御、パフォーマンス監視、構成などのデータセンターの基幹サービスを提供したりします。

vCenter Server は、個々のホストのリソースを統合し、データセンター全体の仮想マシン間でリソースを共有します。これは、システム管理者が設定したポリシーに基づいて、ホストに対する仮想マシンの割り当て、および特定のホスト内の仮想マシンに対するリソースの割り当てを管理することで実現されます。

vCenter Server では、vSphere DRS、vSphere HA、vMotion などの高度な vSphere 機能を使用できます。
