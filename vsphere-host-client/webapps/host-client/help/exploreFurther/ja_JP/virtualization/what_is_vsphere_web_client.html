vSphere Web Client について

vSphere Web Client とは、仮想マシンと仮想マシンのリソースおよびホストを作成、管理、および監視するための主要インターフェイスです。これは、仮想マシンへのコンソール アクセスも提供します。

vSphere Web Client は、ご使用の vCenter Server 環境にネットワーク経由でアクセスできるコンピュータにインストールされた Web アプリケーションです。vSphere Web Client は Windows マシンにインストールしたり、Linux ベースの vCenter Server アプライアンスの一部としてデプロイすることができます。vSphere Web Client には Web ブラウザを使用してアクセスします。

vCenter Server がすべての vSphere アクティビティを実行しますが、vCenter Server の監視、管理、および制御には vSphere Web Client を使用します。
