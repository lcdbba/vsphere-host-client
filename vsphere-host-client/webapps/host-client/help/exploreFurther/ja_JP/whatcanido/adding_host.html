ホストの追加

ホストは、仮想化ソフトウェアを使用して仮想マシンを実行するコンピュータです。

データセンターにホストを追加するには、ESX または ESXi ソフトウェアを実行するコンピュータが必要です。ホストを追加すると、vCenter Server がそのホストを管理します。ホストを追加したあと、vSphere Web Client を使用してホストおよびそのホストで実行する仮想マシンを管理します。

ホストを追加するには、ホストを追加するデータセンタを選択し、[アクション] > [ホストの追加] を選択します。
