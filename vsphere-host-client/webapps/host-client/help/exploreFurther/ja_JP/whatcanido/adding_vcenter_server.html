vCenter Server システムの追加

vSphere Web Client にログインすると、インストール中にクライアントの Lookup Service に登録したすべての vCenter Server 5.1 システムのうち、正しい権限を持っているシステムにアクセスできるようになります。vSphere Web Client に vCenter Server 5.1 システムを手動で登録する必要はありません。

vCenter Server 5.1 システムにアクセスできない場合は、次の手順を実行してください。

  • vCenter Server システムが、vSphere Web Client の Lookup Service に登録されていることを確認します。
  • SSO 管理者から vCenter Server システムへアクセスするための権限が付与されていることを確認します。

vCenter Server 5.0 システムを vSphere Web Client から管理するには、vSphere Web Client に vCenter Server を登録する必要があります。vCenter Server 5.0 システムを Windows にインストールされている vSphere Web Client インスタンスに登録するには、vSphere 管理ツールを使用します。vCenter Server 5.0 システムを vCenter Server アプライアンスにインストールされている vSphere Web Client インスタンスに登録するには、admin-app コマンドライン スクリプトを使用します。
