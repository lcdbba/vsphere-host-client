仮想マシンのクローン作成

クローンとは、仮想マシンの厳密なコピーです。クローンを作成するとき、vCenter Server で仮想マシンのゲスト OS をカスタマイズできます。クローン作成元の仮想マシンと同じデータセンター内にある任意のホスト上で、新しいクローンを作成できます。

仮想マシンのクローンを作成するには、仮想マシンを選択し、[アクション] > [すべての vCenter アクション ] > [クローン作成] を選択します。[仮想マシンのクローン作成] ウィザードに従って、仮想マシンのクローン作成を行います。どのようなパワー状態でも仮想マシンのクローンを作成できます。

このプロセスでは、次の作業を行う必要があります。

  • データセンター内のほかの仮想マシンおよびテンプレートに対して一意の名前を仮想マシンに付ける。
  • 新規仮想マシンを実行するホストまたはクラスタを選択する。
  • 仮想マシン ファイルにデータストアを割り当てる。

「仮想マシンのクローン作成」 ウィザードでの作業を進めていくと、新規仮想マシンのゲスト OS をカスタマイズできます。このオプションを選択すると、「ゲスト OS のカスタマイズ」 ウィザードが起動します。
