データセンターの作成

ホスト、仮想マシン、リソース プール、およびクラスタのコンテナとして、データセンターを作成します。データセンターは、常に 1 つの vCenter Server の「ホストおよびクラスタ」インベントリ内の root フォルダの下、root フォルダの下のフォルダ内、または「リンク モード」インベントリ内の vCenter オブジェクトの下に作成されます。

データセンターを作成するには、データセンターを追加する vCenter Server システムを選択し、[アクション] > [新規データセンター] を選択します。

データセンターを作成すると、データセンターにホストを追加できます。
