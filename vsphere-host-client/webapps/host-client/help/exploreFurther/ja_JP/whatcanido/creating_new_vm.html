新規仮想マシンの作成

新規仮想マシンの作成では、プロセッサの数、メモリ、ネットワーク接続、ストレージなどのオプションをカスタマイズできます。新規仮想マシンは、ホスト上またはクラスタ上に作成できます。

仮想マシンを作成する前に、新規仮想マシンを格納するホスト、新規仮想マシンにインストールするゲスト OS のタイプ、およびインストール用の CD またはイメージ ファイルの場所を決定します。また、仮想マシンの作成には、適切な vCenter Server の権限も必要です。

新規仮想マシンを作成するには、新規仮想マシンを実行するインベントリ内のホストまたはクラスタを選択し、[アクション] > [新規仮想マシン] を選択します。

新しい仮想マシンは、空のハード ディスクを持つ物理コンピュータに似ています。新規仮想マシンを作成したあと、その仮想マシンにゲスト OS をインストールする必要があります。また、いつでも仮想マシンの設定を変更できます。
