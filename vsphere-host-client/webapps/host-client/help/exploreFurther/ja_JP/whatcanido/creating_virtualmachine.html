仮想マシンの作成

いくつかの方法で、インベントリに追加する仮想マシンを作成できます。空の仮想マシンを新規作成する、物理コンピュータを仮想マシンに変換する、テンプレートからデプロイする、または既存の仮想マシンのクローンを作成することができます。

vSphere Web Client には、仮想マシンを作成するためのシンプルで柔軟なユーザー インターフェイスがあります。ウィザードに従って、機能する完全な仮想マシンを作成します。

仮想マシンを作成すると、選択したホスト上に作成されます。仮想マシンはその選択したホスト上で実行され、そのホストのリソースを使用します。仮想マシンは、同じデータセンター内のホスト間で移動できます。
