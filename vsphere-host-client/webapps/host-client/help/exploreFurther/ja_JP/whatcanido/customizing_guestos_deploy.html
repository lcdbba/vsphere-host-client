ゲスト OS のカスタマイズ

テンプレートから新規仮想マシンをデプロイする場合や、既存の仮想マシンのクローンを作成する場合に、新規ゲスト OS をカスタマイズできます。仮想マシンのゲスト OS の ID とネットワーク設定をカスタマイズして、ターゲット環境でただちに稼動するようにできます。その後、仮想マシンの設定は仕様に保存できます。この仕様は、後で再度呼び出して再利用できます。

[テンプレートからのデプロイ] または [仮想マシンのクローン作成] ウィザードを終了する前に、ゲスト OS をカスタマイズするかどうかを選択できるオプションがあります。このオプションを選択した場合、カスタマイズ ウィザードに従って、構成オプションを設定します。

保存済みのカスタマイズ仕様にアクセスするには、vSphere Web Client のホーム ページで、[ルールおよびプロファイル] を選択してから [仮想マシンのカスタマイズ仕様] を選択します。ここで、既存の仕様を編集したり、新しく仕様を作成してあとで使用するために保存したりすることができます。
