テンプレートからのデプロイ

テンプレートから仮想マシンをデプロイするとき、vCenter Server で仮想マシンのゲスト OS をカスタマイズできます。テンプレートと同じデータセンターにある任意のホスト上で、新しい仮想マシンを実行できます。

[テンプレートのデプロイ] ウィザードに従って、テンプレートのデプロイを行います。ウィザードを開始するには、テンプレートを選択して [アクション] > [このテンプレートからの仮想マシンのデプロイ] を選択します。

このプロセスでは、次の作業を行う必要があります。

  • データセンター内のほかの仮想マシンおよびテンプレートに対して一意の名前を仮想マシンに付ける。
  • 新規仮想マシンを実行するホストまたはクラスタを選択する。
  • 仮想マシン ファイルにデータストアを割り当てる。

「テンプレートのデプロイ」 ウィザードで、新規仮想マシンのゲスト OS をカスタマイズできます。このオプションを選択すると、「ゲスト OS のカスタマイズ」 ウィザードが起動します。
