インストールの手順: ISO ファイル

  1. インベントリで仮想マシンを選択し、[アクション] > [パワーオン] を選択します。
  2. 仮想マシンの [サマリ] タブで、[接続 CD/DVD-ROM] アイコンをクリックし、[データストアの CD/DVD イメージに接続] を選択します。
  3. [コンソールの起動] リンクをクリックします。
  4. 仮想マシンがパワーオンされたら、ほかのコンピュータと同様に、ゲスト OS をインストールします。
    Windows XP のインストールには、ドライバを追加する必要があります。特定のオペテーティング システムのインストール手順については、『ゲスト OS インストール ガイド』を参照してください。
  5. ゲスト OS をインストールして実行してから、VMware Tools をインストールします。これにより、仮想マシンのパフォーマンスが向上します。
