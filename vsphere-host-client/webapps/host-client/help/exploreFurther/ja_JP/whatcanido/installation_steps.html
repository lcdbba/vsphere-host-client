インストール手順ガイド

ゲスト OS をインストールする方法

  1. VMware クライアント統合プラグインがブラウザにインストールされていることを確認します。
  2. インベントリで仮想マシンを選択します。
  3. [アクション] > [設定の編集] を選択して [仮想マシンの編集] ダイアログ ボックスを開きます。
  4. [仮想ハードウェア] タブで、オペレーティング システムの CD を挿入する CD/DVD ドライブをクリックします。
  5. 右側のペインにある [デバイス タイプ] の下から、オペレーティング システムの CD/DVD-ROM または ISO イメージ ファイルの場所を選択します。
  6. [データストア ISO ファイル] を選択した場合は、ISO ファイルを参照して選択します。[データストア ISO ファイル] および [ホスト デバイス] の両方のオプションの場合、[パワーオン時に接続] チェック ボックスを選択します。
  7. OK をクリックします。
  8. 特定のインストール手順については、次のいずれかのリンクをクリックしてください。
     
    • クライアント CD-ROM ドライブ
    • ホスト CD-ROM ドライブ
    • ISO イメージ ファイル
