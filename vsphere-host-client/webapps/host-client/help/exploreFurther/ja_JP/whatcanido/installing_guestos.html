ゲスト OS のインストール

新しい仮想マシンは、空のハード ディスクを持つ物理コンピュータに似ています。仮想マシンを使用する前に、オペレーティング システムをインストールする必要があります。仮想マシン上のオペレーティング システムのことを、ゲスト オペレーティング システム（ゲスト OS）といいます。

仮想マシンへのゲスト OS のインストールは、物理コンピュータでのインストールと実質的に同じです。オペレーティング システムのベンダーからインストール ファイルを含む CD-ROM または ISO イメージを入手する必要があります。物理コンピュータの場合と同様に、インストールごとに個別のオペレーティング システム ライセンスが必要です。

ゲスト OS をインストールしたあと、vCenter Server を使用すると、アプリケーションのインストールやパワー操作の管理などのタスクに、物理コンピュータと同様に仮想マシンを使用することができます。
