データストアの作成

LUN をフォーマットするか、NFS ボリュームを既存のホストにマウントすることにより、新規データストアを作成できます。また、既存のデータストアを使用しているホストをインベントリに追加できます。さらに、整理する目的で、およびデータストア グループにまたがる権限やアラームを設定するために、フォルダを使用してデータストアの論理グループを作成することもできます。

データストアを作成するには、ホストを選択して [アクション] > [新しいデータストア] を選択します。データストアの作成後、仮想マシンのファイルの保存用にデータストアを使用できます。
