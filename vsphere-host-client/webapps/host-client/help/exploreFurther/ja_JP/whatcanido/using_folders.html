フォルダの使用

フォルダでは、インベントリ内でオブジェクトをさらに細かくグループ化するために、次のインベントリ オブジェクトをグループ化できます。データセンター、仮想マシンおよびテンプレート、ホストおよびクラスタ、データストアおよびデータストア クラスタ、ネットワーク オブジェクト。

フォルダ内でグループ化するオブジェクトのタイプは、同じにする必要があります。

フォルダの作成方法:

  • vCenter Server システムを選択して [アクション] > [新規フォルダ] を選択します。
  • データセンターを選択して、[アクション] > [すべての vCenter アクション] > [新規ホストおよびクラスタ フォルダ]、[新規ネットワーク フォルダ]、[新規ストレージ フォルダ] または [新規仮想マシンおよびテンプレート フォルダ] を選択します。

フォルダにオブジェクトを追加するには、オブジェクトを選択して [アクション] > [移動先] を選択します。フォルダをオブジェクトの配置先として選択します。

フォルダを削除するには、インベントリでフォルダを選択し、[アクション] > [すべての vCenter アクション] > [インベントリからの除去] を選択します。フォルダを削除した場合は、フォルダ内のすべてのインベントリ項目も削除されます。フォルダ内の項目がホストである場合、そのホストは vCenter Server システムから切断されます。
