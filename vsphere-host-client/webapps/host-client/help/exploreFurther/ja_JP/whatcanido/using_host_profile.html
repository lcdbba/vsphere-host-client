ホスト プロファイルの使用

[ホスト プロファイル] ビューは、ホスト プロファイルを操作するための vSphere Web Client の管理領域であり、ここで管理者はホスト プロファイルを作成、編集、または削除できます。vSphere Web Client のホーム ページからこのビューにアクセスするには、[ルールおよびプロファイル] をクリックしてから [ホスト プロファイル] をクリックします。

プロファイルの添付などの他のホスト プロファイル アクションを実行するには、インベントリのホストまたはクラスタを右クリックし、[すべての vCenter アクション] > [ホスト プロファイル] から適切なアクションを選択します。コンプライアンスの確認などのクラスタ レベルのプロファイル操作を実行するには、クラスタを選択し、[監視] タブをクリックした後に [プロファイルのコンプライアンス] をクリックします。

ホスト プロファイルを使用するには、それをインベントリ内にホストに添付および適用します。複数のホストまたは 1 つのクラスタを選択してホスト プロファイルを添付できます。
