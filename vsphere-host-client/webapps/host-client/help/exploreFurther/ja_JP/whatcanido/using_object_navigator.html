ナビゲータの使用

vSphere Web Client の左ペインはナビゲータです。ここでは、すべての vCenter Server インベントリおよび他の vSphere 機能にアクセスできます。

オブジェクト ナビゲータの使用方法:

  • アイテムをクリックし、vSphere Web Client の中央のペインにその情報を表示します。
  • 「>」 のあるアイテムをクリックして、ナビゲータ ペインの最上部に移動して、その下の関連アイテムのカテゴリを表示します。
  • カウントのあるカテゴリをクリックして、ナビゲータ ペインにそのオブジェクトを表示します。
