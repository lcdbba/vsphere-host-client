リソース プールの使用

リソース プールの作成後、vCenter Server は共有リソースを管理し、リソース プール内部の仮想マシンに共有リソースを割り当てます。リソース プールを使用すると、次のことを実行できます。

  • 同じホスト上またはクラスタ上で実行されている仮想マシンに、プロセッサおよびメモリ リソースを割り当てる。
  • CPUおよびメモリに対して、リソース シェアの最小値、最大値、および比率を設定する。
  • 仮想マシンの実行中にリソースの割り当てを変更する。
  • 最高のパフォーマンスを提供するため、アプリケーションがより多くのリソースを動的に取得できるようにする。

vSphere Web Client を使用して、仮想マシンをリソース プールに追加したり、リソースの使用率を表示したり、またはリソースの予約、制限、およびシェアを設定したりします。
