vSphere 標準スイッチ アーキテクチャのセットアップ

vSphere 標準スイッチを使ったネットワーク環境の設定は、ホストの [管理] タブの [ネットワーク] タブから行えます。サービス コンソール ネットワーク、vMotion、iSCSI ネットワークなどの仮想マシンのネットワークまたはネットワーク サービスのポート グループは、ネットワークの追加ウィザードを使って構成します。ポート グループ作成処理中に新規の標準スイッチを作成できます。または、新規のポート グループを既存の標準スイッチに接続できます。仮想マシンは、仮想マシンの NIC をネットワークのポート グループに割り当てることにより、仮想ネットワークに接続されます。
