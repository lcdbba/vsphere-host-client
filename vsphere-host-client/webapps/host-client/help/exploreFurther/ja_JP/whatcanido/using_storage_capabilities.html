ストレージ機能の使用

vSphere Web Client の「ストレージ サービス クラス」ビューから、ストレージ機能にアクセスできます。

「ストレージ機能の作成、編集または削除」アイコンをクリックして、
既存のシステム定義のストレージ機能を閲覧したり、ユーザー定義のストレージ機能を
追加、編集、削除したりできます。

「ストレージ サービス クラスの新規作成」アイコンをクリックして、ストレージ機能を新規のストレージ サービス クラスに追加します。

既存のストレージ サービス クラスを右クリックし、「ストレージ サービス クラスの編集」をクリックして、ストレージ サービス クラスのコンプライアンス チェックに必要なストレージ機能のリストを修正します。

インベントリのデータストアを右クリックして、
「ストレージ機能の割り当て」を選択し、データストアが、特定のストレージ機能を持っていることを確認します。
