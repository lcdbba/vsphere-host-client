ストレージ サービス クラスの使用

ストレージ サービス クラスを仮想マシンに割り当てるには、インベントリで仮想マシンを選択して、「管理」タブをクリックしてから「サービス クラス」をクリックします。

次のウィザードの「ストレージ」ページにあるストレージ サービス クラスを選択するとき、vSphere Web Client には、ストレージ サービス クラスと互換性がある、または互換性のないデータストアが表示されます。

  • 新規仮想マシン
  • 仮想マシンのクローン作成
  • 仮想マシンの移行

ストレージ サービス クラスと互換性のないデータストア上で仮想マシンまたは仮想ディスクが実行されている場合、互換性のあるデータストアにそれらを移動できます。

ストレージ サービス クラスを作成、編集、削除するには、vSphere Web Client のホーム ページから「ルールおよびプロファイル」をクリックしてから「ストレージ サービス クラス」をクリックします。
