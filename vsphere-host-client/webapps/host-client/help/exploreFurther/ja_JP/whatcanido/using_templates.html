テンプレートの使用

テンプレートの作成後、そのテンプレートを使用して、データセンター内の任意のホストに新規仮想マシンをデプロイできます。テンプレートを使用すると、次のことを実行できます。

  • 信頼性を高めるために、共有ストレージにテンプレートを保存する。
  • マシン名およびネットワーク アドレスを使用してテンプレートをカスタマイズする。
  • 設定基準に従ってテンプレートを構成し、仮想マシンとしてデプロイする。
  • 新規仮想マシンの作成時に、エラーおよびダウンタイムを最小限にする。
  • 仮想マシンのパッチおよびアップデートにテンプレートを使用する。
