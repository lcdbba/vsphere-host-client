vApp の使用

vApp の基本操作は仮想マシンと同じですが、vAPP の場合は、複数の仮想マシンや複数の vApp が含まれることがあります。

個々の仮想マシンと同様に、vApp は vSphere を使用して管理できるインベントリ オブジェクトです。vSphere Web Client では、vApp インベントリ リスト、[ホストおよびクラスタ] インベントリ ツリー、[仮想マシンおよびテンプレート] インベントリ ツリーで vApp を検索できます。これらのビューでは、個々の仮想マシンを操作するのと同じように、vApp の作成、構成、パワーオンとパワーオフ、およびクローンを実行できます。

vApp に格納されている仮想マシンの起動順序を指定するには、vApp のプロパティを編集します。

vApps は、インポートおよびエクスポートすることができます。vApp の配布フォーマットは OVF です。
