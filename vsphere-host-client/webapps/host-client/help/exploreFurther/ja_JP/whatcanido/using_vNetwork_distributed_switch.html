vSphere Distributed Switch アーキテクチャのセットアップ

vSphere Distributed Switch を使用するネットワーク環境をセットアップするには、はじめに vSphere Client の [ネットワーク] インベントリ ビューで vSphere Distributed Switch を作成します。vSphere Distributed Switch は、vSphere Client の [ネットワーク] ビューでデータセンター レベルで作成されます。vSphere Distributed Switch を作成した後、ホストおよび分散ポート グループをスイッチに追加できます。

ホストのネットワーク オプションは、個々のホスト ネットワーク構成ページから、または、ホスト プロファイルを使用して構成します。

最後に、仮想マシン NIC を分散ポート グループに接続します。個々の仮想マシン NIC 構成を介して、または、vSphere Distributed Switch 自体から仮想マシン ネットワークを移行して、仮想マシン NIC を接続します。
