仮想マシンの使用

仮想マシンは仮想インフラストラクチャの主要なコンポーネントであり、仮想マシンを使用すると企業内でのパフォーマンスを高めることができます。仮想マシンは、統合された物理ハードウェアと高い信頼性で、企業内の複数の異なるオペレーティング システムやアプリケーションを同時に運用できます。

仮想マシンにゲスト OS およびアプリケーションをインストールすると、vCenter Server を使用して仮想マシンの運用を管理します。仮想マシンのパワー状態の変更、仮想マシンの構成の編集、別のホストへの仮想マシンの移行、および仮想マシンへの仮想ハードウェア デバイスの追加を行うことができます。
