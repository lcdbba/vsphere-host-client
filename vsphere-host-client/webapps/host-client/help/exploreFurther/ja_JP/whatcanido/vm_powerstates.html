仮想マシンのパワー状態

仮想マシンの電源状態は、仮想マシンがアクティブで、機能しているかどうかを示します。仮想マシンにはいくつかのパワー状態がありますが、オン、オフ、およびサスペンドの 3 つが基本的な状態です。仮想マシンの基本的なパワー操作に加えて追加機能があり、その機能を使用するには、まず仮想マシンに VMware Tools をインストールする必要があります。

次のように、いくつかのアクセス ポイントを使用して、パワー状態を変更できます。

  • 手動でパワー状態を変更するには、仮想マシンを右クリックして、適切なパワー オプション メニューの項目を選択します。
  • パワー状態の自動変更をスケジュール設定するには、Ctrl キーを押し、仮想マシンを右クリックして、適切なパワー オプション メニューの項目を選択します。

パワー状態のオプションが現在使用できない場合、そのオプションは無効なため選択できません。
