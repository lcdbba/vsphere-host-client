VMware Tools

VMware Tools は、仮想マシンのゲスト OS のパフォーマンスを強化し、仮想マシンの管理機能を向上させるための一連のユーティリティです。仮想マシンにゲスト OS をインストールしたあと、VMware Tools をインストールします。

ゲスト OS に VMware Tools をインストールすることは非常に重要です。ゲスト OS は VMware Tools がなくても実行できますが、VMware Tools を使用しないと重要な機能および利便性が失われます。VMware Tools のインストール時に、VMware Tools サービスと、ディスプレイ、ネットワーク、SCSI、メモリ制御、I/O、およびマウスのサポートを拡張するVMware デバイス ドライバ一式がインストールされます。

VMware Tools のインストール

VMware Tools をインストールする前に、ブラウザに VMware クライアント統合プラグインがインストールされていることを確認します。

  1. インベントリで仮想マシンを右クリックし、[すべての vCenter アクション] > [ゲスト OS] > [VMware Tools のインストール/アップグレード] を選択します。
  2. 仮想マシンの [サマリ] タブの [コンソールの起動] をクリックします。
  3. [仮想マシン] コンソールで、インストール ウィザードの指示に従って処理を進めます。

Windows へのインストレーションで、ウィザードが自動的に表示されない場合、ゲスト OS の VMware Tools と表示された CD-ROM ドライブを開き、セットアップ アプリケーションを実行します。

Linux をインストールした場合、『ゲスト OS インストール ガイド』の指示に従ってください。
