데이터 센터란?

데이터 센터는 호스트 및 가상 시스템과 같은 인벤토리 개체의 기본 컨테이너입니다. 데이터 센터에서 인벤토리 개체를 추가 및 구성할 수 있습니다. 일반적으로 데이터 센터에 호스트, 폴더 및 클러스터를 추가합니다.

vCenter Server는 여러 데이터 센터를 포함할 수 있습니다. 대기업은 엔터프라이즈 내 조직 단위를 나타내기 위해 여러 데이터 센터를 사용할 수 있습니다.

인벤토리 개체는 데이터 센터 내에서 상호 작용할 수 있지만 데이터 센터 간의 상호 작용은 제한됩니다. 예를 들어 데이터 센터 내의 호스트 간에 vMotion 기술을 사용하여 가상 시스템을 이동할 수 있지만 다른 데이터 센터의 호스트로는 이동할 수 없습니다.
