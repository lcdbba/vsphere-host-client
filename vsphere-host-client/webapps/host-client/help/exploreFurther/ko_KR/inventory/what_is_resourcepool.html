리소스 풀이란?

리소스 풀은 독립형 호스트 또는 클러스터를 보다 작은 풀로 분할할 수 있는 방법을 제공합니다. 리소스 풀은 리소스 풀에서 실행되는 가상 시스템이 공유하는 일련의 CPU 및 메모리 리소스로 구성됩니다. 리소스 풀은 자체적으로 포함되며 다른 리소스 풀로부터 분리되어 있습니다.

CPU 및 메모리 용량을 집계하는 단일 리소스 풀로 여러 물리적 서버를 결합할 수 있습니다.

가상 시스템은 리소스 풀에서 실행되고 리소스 풀로부터 리소스를 얻습니다. 이 정렬을 통해 계속적으로 리소스 풀 전체에서 가상 시스템 워크로드 밸런싱이 수행됩니다. 워크로드가 증가하면 vCenter Server가 자동으로 추가 리소스를 할당하고 리소스 풀의 호스트 간에 가상 시스템을 투명하게 마이그레이션합니다.
