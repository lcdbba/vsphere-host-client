vSphere Standard Switch 아키텍처란?

vSphere Standard Switch를 사용한 네트워킹은 호스트 수준에서 이루어집니다. vSphere Standard Switch(vSwitch)는 물리적 스위치처럼 작동합니다. 어느 가상 시스템이 각 해당 가상 포트에 연결되었는지 탐지하고 이 정보를 이용해 트래픽을 적당한 가상 시스템으로 전달합니다. 하나 이상의 포트 그룹이 Standard Switch에 연결되어 각 멤버 포트에 대해 대역폭 및 VLAN 태깅 정책과 같은 포트 구성 옵션을 지정합니다. 각 가상 시스템 NIC는 한 포트 그룹에 할당할 수 있습니다.
