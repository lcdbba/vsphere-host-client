스토리지 기능이란?

스토리지 기능은 데이터스토어가 제공하는 스토리지 특성을 나타냅니다. 예를 들어 용량, 성능, 가용성 등의 특성이 여기에 해당될 수 있습니다.

  • 시스템 정의 스토리지 기능

    스토리지 인식을 지원하는 스토리지 시스템은 해당 시스템에 생성된 데이터스토어에 시스템 정의 스토리지 기능을 할당합니다.

  • 사용자 정의 스토리지 기능

    수동으로 사용자 정의 스토리지 기능을 생성하고 데이터스토어에 할당하여 동일한 스토리지 시스템에서 실행되거나 성능, 용량, 가용성 등의 특성이 유사하거나 동일한 데이터스토어를 식별할 수 있도록 합니다.
