VM 스토리지 프로파일이란?

VM 스토리지 프로파일은 가상 시스템과 해당 가상 디스크에 필요한 스토리지 기능을 정의합니다. 가상 시스템 파일과 가상 디스크에 스토리지 요구 사항에 맞춰 VM 스토리지 프로파일을 할당할 수 있습니다.

VM 스토리지 프로파일을 가상 시스템이나 가상 디스크에 할당하면 규정 준수 검사가 실행되어 가상 시스템이나 가상 디스크가 VM 스토리지 프로파일에 정의된 스토리지 기능을 사용하는지 확인합니다.
