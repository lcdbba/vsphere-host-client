vSphere Distributed Switch란?

vSphere Distributed Switch는 연관된 모든 호스트에서 단일 가상 스위치 역할을 발휘합니다. 이를 통해 가상 시스템은 호스트 간에 마이그레이션할 때 일관된 네트워크 구성을 유지할 수 있습니다.

가상 스위치처럼 분산 가상 스위치는 가상 시스템에 대한 네트워크 허브입니다. vSphere Distributed Switch는 가상 시스템 간에 내부적으로 트래픽을 라우팅하거나 외부 네트워크에 연결할 수 있습니다.

vSphere Distributed Switch에는 vSphere에 할당된 여러 분산 포트 그룹이 있을 수 있습니다.
