vSphere 작동 방식

VMware vSphere는 가상 인프라스트럭처를 관리하기 위한 시스템입니다. VMware vCenter Server는 여러 가상 시스템을 실행하는 여러 호스트 서버를 관리하는 툴입니다.

vCenter Server를 사용하면 신속하게 새 서버 가상 시스템을 프로비저닝하고 표준화된 가상 시스템 템플릿 라이브러리를 생성하여 새로 프로비저닝된 시스템이 항상 데이터 센터 요건을 준수하도록 할 수 있습니다.

vCenter Server를 사용하면 다운타임을 최소한으로 하여 하드웨어 유지 보수를 수행할 수 있도록 실행 중인 가상 시스템을 호스트 서버 간에 마이그레이션할 수 있습니다. 기타 vCenter Server 기능을 사용하여 호스트 전체에서 가상 시스템 워크로드 밸런싱을 수행하고 고가용성 및 재해 복구를 위해 가상 시스템을 관리할 수 있습니다.
