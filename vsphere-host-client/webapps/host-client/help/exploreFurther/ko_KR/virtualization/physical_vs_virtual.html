물리적 대 가상

가상 시스템은 물리적 컴퓨터처럼 운영 체제와 애플리케이션을 실행하는 소프트웨어 컴퓨터입니다. 가상 시스템에는 게스트 운영 체제 및 해당 애플리케이션이 실행되는 자체 가상 하드웨어 집합이 있습니다. 운영 체제는 실제 물리적 하드웨어 구성 요소에 관계없이 일관된 하드웨어 집합을 보게 됩니다.

가상 시스템은 에뮬레이터나 시뮬레이터가 아닙니다. 가상 시스템은 물리적 컴퓨터가 수행할 수 있는 것과 동일한 작업 등을 수행할 수 있는 실제 시스템입니다. 가상 시스템의 유연성 때문에 물리적 컴퓨터는 서비스(애플리케이션, 데이터베이스 등)를 제공하는 용도보다는 이러한 서비스를 제공하는 가상 시스템을 하우징하는 용도로 더 많이 사용됩니다.
