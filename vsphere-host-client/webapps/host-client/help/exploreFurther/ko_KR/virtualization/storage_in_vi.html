vSphere의 스토리지

모든 가상 시스템은 가상 하드 디스크를 사용하여 해당 운영 체제, 프로그램 파일 및 해당 작업과 연관된 기타 데이터를 저장합니다. 가상 디스크는 물리적 대용량 파일이거나 파일 집합이며 다른 파일처럼 쉽게 복사, 이동, 아카이빙 및 백업할 수 있습니다. 가상 디스크 파일은 호스트의 내부 및 외부 디바이스를 비롯한 다양한 물리적 스토리지 시스템의 전용 스토리지 공간이나 데이터 저장 및 보호의 특정 작업 전용인 네트워크 스토리지에 저장됩니다.

호스트는 액세스 권한이 있는 스토리지 디바이스를 탐색하여 데이터스토어로 포맷할 수 있습니다. 데이터스토어는 호스트가 가상 디스크 파일 및 가상 시스템의 중요한 구성 요소를 캡슐화하는 기타 파일을 두는 논리적 볼륨의 파일 시스템과 유사한 특수 논리적 컨테이너입니다. 데이터스토어는 다양한 스토리지 제품의 세부 사항을 숨기고 가상 시스템 파일 저장을 위한 통일된 모델을 제공할 수 있습니다.
