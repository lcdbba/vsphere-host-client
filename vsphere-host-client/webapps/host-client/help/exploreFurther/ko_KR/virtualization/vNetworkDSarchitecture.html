vSphere Distributed Switch 아키텍처란?

vSphere Distributed Switch가 포함된 네트워킹은 데이터 센터 수준에서 이루어집니다. vSphere Distributed Switch는 연관된 모든 호스트에서 가상 스위치 역할을 하는 데이터 센터에 개체로 존재하여 가상 시스템이 호스트 간에 마이그레이션할 때 일관된 네트워크 구성을 유지하도록 합니다.

업링크는 vSphere Distributed Switch를 연관된 호스트의 물리적 네트워크 어댑터에 연결합니다. 각 호스트는 vSphere Distributed Switch의 각 업링크 포트에 대해 vSphere Distributed Switch에 하나의 물리적 네트워크 어댑터를 연결할 수 있습니다.

가상 시스템 NIC는 분산 포트 그룹을 통해 vSphere Distributed Switch에 연결합니다. 이 그룹은 공통 구성 하에 여러 포트를 집계하고 레이블이 지정된 네트워크에 연결되는 가상 시스템에 안정된 기준 위치점을 제공합니다.
