vCenter Server란?

VMware vCenter Server는 데이터 센터에 대한 편리한 단일 제어 지점을 제공합니다. VMware vCenter Server는 Windows 서버 위에서 실행되어 중앙에서 VMware ESX/ESXi 호스트를 관리하고 액세스 제어, 성능 모니터링 및 구성과 같은 중요한 데이터 센터 서비스를 제공합니다.

vCenter Server는 전체 데이터 센터에 있는 가상 시스템 간에 공유할 개별 호스트의 리소스를 통합합니다. 이는 시스템 관리자가 설정한 정책을 기반으로 주어진 호스트 내에서 호스트에 대한 가상 시스템 할당 및 가상 시스템에 대한 리소스 할당을 관리하여 이루어집니다.

vCenter Server에서는 vSphere DRS, vSphere HA및 vMotion과 같은 고급 vSphere 기능을 사용할 수 있습니다.
