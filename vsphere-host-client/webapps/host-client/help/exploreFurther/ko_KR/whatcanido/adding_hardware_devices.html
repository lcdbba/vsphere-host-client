가상 하드웨어 추가

가상 시스템을 생성한 후에 가상 시스템에 DVD/CD-ROM 드라이브, 이더넷 어댑터 또는 추가 하드 드라이브와 같은 가상 하드웨어를 추가해야 할 수 있습니다.

대부분의 경우 가상 하드웨어를 수정하려면 가상 시스템의 전원을 꺼야 합니다.

가상 하드웨어 디바이스를 추가하려면 인벤토리에서 가상 시스템을 선택하고 작업 > 설정 편집을 선택합니다. VM 편집 대화상자에서 가상 하드웨어 탭을 클릭하고 디바이스를 선택한 다음 추가를 클릭합니다.

VM 편집 대화상자에 표시되는 하드웨어 목록에 새 가상 하드웨어가 나타납니다.
