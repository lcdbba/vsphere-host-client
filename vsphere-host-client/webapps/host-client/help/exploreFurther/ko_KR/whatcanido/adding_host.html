호스트 추가

호스트는 가상화 소프트웨어를 사용하여 가상 시스템을 실행하는 컴퓨터입니다.

데이터 센터에 호스트를 추가하려면 컴퓨터가 ESX 또는 ESXi 소프트웨어를 실행하고 있어야 합니다. 호스트를 추가하면 해당 호스트가 vCenter Server에 의해 관리됩니다. 호스트를 추가한 후에는 vSphere Web Client를 사용하여 해당 호스트 및 해당 호스트에서 실행되는 가상 시스템을 관리합니다.

호스트를 추가하려면 호스트를 추가할 데이터 센터를 선택하고 작업 &gt 호스트 추가를 선택합니다.
