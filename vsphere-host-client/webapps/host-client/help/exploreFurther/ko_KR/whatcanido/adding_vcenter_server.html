vCenter Server 시스템 추가

vSphere Web Client에 로그인하면 설치 과정에서 클라이언트의 Lookup Service에 등록되었으며 올바른 사용 권한을 가진 모든 vCenter Server 5.1 시스템에 액세스할 수 있습니다. vSphere Web Client에는 vCenter Server 5.1 시스템을 수동으로 등록할 필요가 없습니다.

vCenter Server 5.1 시스템에 액세스할 수 없는 경우 다음을 수행하십시오.

  • vCenter Server 시스템이 vSphere Web Client의 Lookup Service에 등록되었는지 확인합니다.
  • SSO 관리자가 vCenter Server 시스템에 액세스할 수 있는 권한을 부여했는지 확인합니다.

vSphere Web Client에서 vCenter Server 5.0 시스템을 관리하려면 vCenter Server를 vSphere Web Client에 등록해야 합니다. Windows에 설치된 vSphere Web Client 인스턴스에 vCenter Server 5.0 시스템을 등록하려면 vSphere 관리 도구를 사용해야 합니다. vCenter Server Appliance에 설치된 vSphere Web Client 인스턴스에 vCenter Server 5.0 시스템을 등록하려면 admin-app 명령줄 스크립트를 사용해야 합니다.
