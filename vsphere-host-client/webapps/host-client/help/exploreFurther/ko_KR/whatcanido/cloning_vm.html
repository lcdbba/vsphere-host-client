가상 시스템 복제

복제는 가상 시스템의 정확한 복제본입니다. 복제를 생성할 때 vCenter Server에서 해당 가상 시스템의 게스트 운영 체제를 사용자 지정할 수 있습니다. 원래 가상 시스템과 동일한 데이터 센터 내의 모든 호스트에서 새 복제를 실행할 수 있습니다.

가상 시스템을 복제하려면 가상 시스템을 선택하고 작업 > 모든 vCenter 작업 > 복제를 선택합니다. 가상 시스템 복제 마법사가 가상 시스템을 복제하는 데 필요한 단계를 안내합니다. 모든 전원 상태에서 가상 시스템을 복제할 수 있습니다.

이 프로세스 동안 다음 작업을 수행해야 합니다.

  • 가상 시스템에 데이터 센터 내의 다른 가상 시스템 및 템플릿에 대해 고유한 이름을 지정합니다.
  • 새 가상 시스템이 실행될 호스트 또는 클러스터를 선택합니다.
  • 가상 시스템 파일에 대한 데이터스토어를 할당합니다.

가상 시스템 복제 마법사를 진행하는 동안 새 가상 시스템의 게스트 운영 체제를 사용자 지정할 수 있습니다. 이 옵션을 선택하면 게스트 운영 체제 사용자 지정 마법사가 시작됩니다.
