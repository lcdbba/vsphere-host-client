데이터 센터 생성

데이터 센터를 호스트, 가상 시스템, 리소스 풀 및 클러스터에 대한 컨테이너로 생성합니다. 데이터 센터는 항상 단일 vCenter Server의 호스트 및 클러스터 인벤토리의 루트 폴더 아래, 루트 폴더 아래의 폴더 또는 Linked Mode 인벤토리의 vCenter 개체 아래에 생성됩니다.

데이터 센터를 생성하려면 데이터 센터를 추가할 vCenter Server 시스템을 선택하고 작업 > 새 데이터 센터를 선택합니다.

데이터 센터를 생성하면 데이터 센터에 호스트를 추가할 수 있습니다.
