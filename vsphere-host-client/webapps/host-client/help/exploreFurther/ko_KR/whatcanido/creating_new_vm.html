새 가상 시스템 생성

새 가상 시스템을 생성하면 프로세서 수, 메모리, 네트워크 연결 및 스토리지와 같은 옵션을 사용자 지정할 수 있습니다. 호스트 또는 클러스터에 새 가상 시스템을 생성할 수 있습니다.

가상 시스템을 생성하기 전에 새 가상 시스템이 있게 될 호스트 또는 클러스터, 새 가상 시스템에 설치할 게스트 운영 체제 유형 및 설치를 위한 CD 또는 이미지 파일의 위치를 결정합니다. 또한 가상 시스템을 생성하기 위한 적절한 vCenter Server 권한이 필요합니다.

새 가상 시스템을 생성하려면 새 가상 시스템을 실행할 인벤토리의 호스트 또는 클러스터를 선택하고 작업 > 새 가상 시스템을 선택합니다.

새 가상 시스템은 하드 디스크가 비어 있는 물리적 컴퓨터와 같습니다. 새 가상 시스템을 생성한 후 새 가상 시스템에 게스트 운영 체제를 설치해야 합니다. 또한 언제든지 가상 시스템의 설정을 변경할 수 있습니다.
