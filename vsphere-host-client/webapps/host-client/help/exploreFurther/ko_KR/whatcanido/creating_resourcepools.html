리소스 풀 생성

리소스 풀을 생성할 때 일련의 가상 시스템에 대해 CPU 및 메모리 공유와 같은 리소스를 지정할 수 있습니다. 리소스가 할당되는 방식을 결정하는 예약, 제한 및 할당량을 지정할 수 있습니다.

리소스 풀을 생성하려면 호스트를 선택하고 작업 > 모든 vCenter 작업 > 새 리소스 풀을 선택하거나 클러스터를 선택하고 작업 > 새 리소스 풀을 선택합니다. 새 리소스 풀 마법사가 생성 프로세스를 안내합니다.

리소스 풀을 생성한 후 가상 시스템을 리소스 풀 개체로 끌어 리소스 풀에 가상 시스템을 추가할 수 있습니다. 리소스 풀에 추가된 가상 시스템은 리소스 풀의 구성에 따라 CPU 및 메모리 리소스를 공유합니다.
