템플릿 생성

다음 방법으로 템플릿을 생성할 수 있습니다.

  • 가상 시스템을 템플릿으로 변환
  • 가상 시스템을 템플릿으로 복제
  • 기존 템플릿 복제

가상 시스템을 템플릿으로 복제할 때 가상 시스템과 동일한 구성의 새 템플릿이 생성되고 원래 가상 시스템은 인벤토리에서 사용 가능한 상태를 유지합니다. 마찬가지로, 템플릿을 복제할 때 템플릿의 복제본이 생성되고 원래 템플릿은 유지됩니다.

템플릿을 생성하려면 인벤토리에서 가상 시스템을 선택하고 작업 > 모든 vCenter 작업 > 템플릿 > 템플릿으로 변환 또는 작업 > 모든 vCenter 작업 > 템플릿 > 템플릿으로 복제를 선택합니다. 기존 템플릿을 복제하려면 템플릿을 선택하고 작업 > 복제를 선택합니다.

템플릿을 생성한 후 vCenter Server가 가상 시스템 및 템플릿 인벤토리 트리 또는 VM 템플릿 인벤토리 목록에 템플릿을 추가합니다.
