가상 시스템 생성

여러 방법을 사용하여 인벤토리에 추가할 가상 시스템을 생성할 수 있습니다. 비어 있는 새 가상 시스템을 생성하고 물리적 컴퓨터를 가상 시스템으로 변환하고 템플릿에서 배포하거나 기존 가상 시스템을 복제할 수 있습니다.

vSphere Web Client는 가상 시스템을 생성할 수 있는 간편하고 유연한 사용자 인터페이스를 제공합니다. 마법사가 완전하고 작동하는 가상 시스템을 생성하는 단계를 안내합니다.

가상 시스템을 생성하는 경우 가상 시스템은 선택하는 호스트에 생성됩니다. 가상 시스템은 해당 호스트에서 실행되고 해당 호스트의 리소스를 사용합니다. 동일한 데이터 센터 내의 한 호스트에서 다른 호스트로 가상 시스템을 이동할 수 있습니다.
