템플릿에서 배포

템플릿에서 가상 시스템을 배포할 때 vCenter Server에서 해당 가상 시스템의 게스트 운영 체제를 사용자 지정할 수 있습니다. 템플릿과 동일한 데이터 센터 내의 모든 호스트에서 새 가상 시스템을 실행할 수 있습니다.

템플릿 배포 마법사가 템플릿을 배포하기 위한 단계를 안내합니다. 마법사를 시작하려면 템플릿을 선택하고 작업 > 이 템플릿에서 VM 배포를 선택합니다.

이 프로세스 동안 다음 작업을 수행해야 합니다.

  • 가상 시스템에 데이터 센터 내의 다른 가상 시스템 및 템플릿에 대해 고유한 이름을 지정합니다.
  • 새 가상 시스템이 실행될 호스트 또는 클러스터를 선택합니다.
  • 가상 시스템 파일에 대한 데이터스토어를 할당합니다.

템플릿 배포 마법사를 진행하는 동안 새 가상 시스템의 게스트 운영 체제를 사용자 지정할 수 있습니다. 이 옵션을 선택하면 게스트 운영 체제 사용자 지정 마법사가 시작됩니다.
