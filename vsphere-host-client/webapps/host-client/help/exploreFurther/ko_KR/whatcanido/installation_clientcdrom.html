설치 단계: 클라이언트 CD-ROM

  1. 운영 체제 디스크를 클라이언트 CD-ROM 드라이브에 삽입합니다.
  2. 인벤토리에서 가상 시스템을 선택하고 작업 > 전원 켜기를 선택합니다.
  3. 가상 시스템의 요약 탭에서 CD/DVD-ROM 연결 아이콘을 클릭하고 로컬 디스크의 CD/DVD 이미지에 연결을 선택합니다.
  4. 콘솔 시작 링크를 클릭합니다.
  5. 가상 시스템의 전원을 켠 후 다른 컴퓨터에서 하는 것처럼 게스트 운영 체제를 설치합니다.
    Windows XP를 설치하려면 추가 드라이버가 필요합니다. 구체적인 운영 체제 설치 지침에 대해서는 게스트 운영 체제 설치 가이드를 참조하십시오.
  6. 게스트 운영 체제를 설치하여 실행한 후 VMware Tools를 설치합니다. VMware Tools는 가상 시스템의 성능을 향상시킵니다.
