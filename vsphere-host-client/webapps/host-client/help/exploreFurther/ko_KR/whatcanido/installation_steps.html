단계별 설치 가이드

게스트 운영 체제를 설치하려면:

  1. 브라우저에 VMware 클라이언트 통합 플러그인이 설치되어 있는지 확인합니다.
  2. 인벤토리에서 가상 시스템을 선택합니다.
  3. 작업 > 설정 편집을 선택해 VM 편집 대화상자를 엽니다.
  4. 가상 하드웨어 탭에서 운영 체제 CD를 삽입할 CD/DVD 드라이브를 클릭합니다.
  5. 오른쪽 창의 디바이스 유형 아래에서 운영 체제 CD/DVD-ROM 또는 ISO 이미지 파일의 위치를 선택합니다.
  6. 데이터스토어 ISO 파일을 선택한 경우에는 ISO 파일을 찾아 선택합니다. 데이터스토어 ISO 파일 및 호스트 디바이스 옵션 모두에 대해 전원을 켤 때 연결 확인란을 선택합니다.
  7. 확인을 클릭합니다.
  8. 구체적인 설치 지침을 보려면 아래에 있는 다음 링크 중 하나를 클릭합니다.
     
    • 클라이언트 CD-ROM 드라이브
    • 호스트 CD-ROM 드라이브
    • ISO 이미지 파일
