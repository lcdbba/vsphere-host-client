가상 시스템 마이그레이션

호스트 간에 가상 시스템을 이동하는 프로세스인 마이그레이션을 사용하여 가상 시스템을 이동할 수 있습니다. 마이그레이션 옵션은 다음과 같습니다.

vSphere vMotion(라이브 마이그레이션)
vSphere vMotion이 실행 중인 가상 시스템을 한 호스트에서 다른 호스트로 이동합니다. 가상 시스템의 전원이 켜져 있고 새 호스트에서 사용 가능한 스토리지 및 네트워크 옵션을 사용하는 경우 애플리케이션을 중지하거나 게스트 운영 체제를 중단하지 않고 가상 시스템을 이동할 수 있습니다.

콜드 마이그레이션
콜드 마이그레이션은 전원이 꺼진 가상 시스템을 한 호스트에서 다른 호스트로 이동합니다. 동일한 데이터 센터 내의 한 호스트에서 다른 호스트로 가상 시스템을 끌어 마이그레이션을 수행할 수 있습니다.

동일한 데이터 센터 내의 호스트 간에는 가상 시스템을 이동할 수 있지만 데이터 센터 간에는 가상 시스템을 이동할 수 없습니다. 가상 시스템을 수동으로 이동하거나 마이그레이션을 수행하도록 스케줄링된 작업을 설정할 수 있습니다.
