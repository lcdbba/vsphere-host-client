폴더 사용

다음 인벤토리 개체를 폴더로 그룹화하여 인벤토리 내의 개체 그룹화를 보다 구체화할 수 있습니다. 데이터 센터, 가상 시스템 및 템플릿, 호스트 및 클러스터, 데이터스토어 및 데이터스토어 클러스터, 네트워크 개체 등을 그룹화할 수 있습니다.

폴더 내에 그룹화된 개체들은 동일한 유형이어야 합니다.

폴더를 생성하려면

  • vCenter Server 시스템을 선택하고 작업 > 새 폴더를 선택합니다.
  • 데이터 센터를 선택하고 작업 > 모든 vCenter 작업 > 새 호스트 및 클러스터 폴더 또는 새 네트워크 폴더, 새 스토리지 폴더 또는 새 VM 및 템플릿 폴더를 선택합니다.

폴더에 개체를 추가하려면 개체를 선택하고 작업 > 다음으로 이동을 선택합니다. 개체의 대상 폴더를 선택합니다.

폴더를 삭제하려면 인벤토리에서 폴더를 선택하고 작업 > 모든 vCenter 작업 > 인벤토리에서 제거를 선택합니다. 폴더를 삭제하면 폴더 내의 모든 인벤토리 항목이 삭제됩니다. 폴더의 항목이 호스트인 경우 vCenter Server 시스템에서 연결이 끊어집니다.
