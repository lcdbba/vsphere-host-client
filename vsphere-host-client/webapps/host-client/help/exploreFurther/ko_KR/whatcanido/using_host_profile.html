호스트 프로파일 사용

호스트 프로파일 뷰는 호스트 프로파일에 대한 vSphere Web Client의 관리 영역입니다. 여기서 관리자는 호스트 프로파일을 생성, 편집 또는 삭제할 수 있습니다. vSphere Web Client 홈에서 이 뷰에 액세스하려면 규칙 및 프로파일을 클릭하고 호스트 프로파일을 클릭합니다.

프로파일 연결과 같은 기타 호스트 프로파일 관련 작업을 수행하려면 인벤토리에서 호스트나 클러스터를 마우스 오른쪽 버튼으로 클릭하고 모든 vCenter 작업 > 호스트 프로파일에서 적절한 작업을 선택합니다. 규정 준수 확인과 같은 클러스터 수준의 프로파일 작업을 수행하려면 클러스터를 선택하고 모니터 탭을 클릭한 다음 프로파일 규정 준수를 클릭합니다.

호스트 프로파일을 사용하려면 인벤토리에서 호스트에 호스트 프로파일을 연결 및 적용합니다. 호스트 프로파일을 연결할 호스트를 한 번에 여러 개 선택할 수도 있고 하나의 클러스터를 선택할 수도 있습니다.
