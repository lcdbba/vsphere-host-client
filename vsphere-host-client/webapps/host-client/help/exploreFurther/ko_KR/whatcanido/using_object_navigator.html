탐색기 사용

vSphere Web Client의 왼쪽 창이 탐색기입니다. 탐색기를 통해 모든 vCenter Server 인벤토리 및 기타 vSphere 기능에 액세스할 수 있습니다.

개체 탐색기를 사용하려면

  • vSphere Web Client의 가운데 창에서 정보를 보려는 항목을 클릭합니다.
  • > 기호가 표시된 항목을 클릭하면 해당 항목이 탐색기 창의 맨 위에 표시되고 아래쪽에 관련된 항목 범주가 표시됩니다.
  • 개수가 표시되어 있는 범주를 클릭하면 탐색 창에 해당 범주의 개체가 표시됩니다.
