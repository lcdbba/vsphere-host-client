리소스 풀 사용

리소스 풀을 생성하고 나면 vCenter Server가 공유 리소스를 관리하고 리소스 풀 내의 가상 시스템에 공유 리소스를 할당합니다. 리소스 풀을 사용하여 다음 작업을 수행할 수 있습니다.

  • 동일한 호스트 또는 클러스터에서 실행되는 가상 시스템에 프로세서 및 메모리 리소스 할당
  • CPU 및 메모리에 대한 최소, 최대 및 비례적 리소스 할당량 설정
  • 가상 시스템이 실행되고 있는 동안 할당 수정
  • 애플리케이션이 최대 성능을 위해 보다 많은 리소스를 동적으로 얻을 수 있도록 함

리소스 풀에 가상 시스템을 추가하고 리소스 사용 현황을 보고, 리소스 예약, 제한 및 할당량을 설정하려면 vSphere Web Client를 사용합니다.
