vSphere Standard Switch 아키텍처 설정

vSphere Standard Switch를 사용한 네트워킹 환경 설정 작업은 호스트의 관리 탭에서 네트워킹 탭을 선택하여 수행할 수 있습니다. 가상 시스템 네트워킹 또는 서비스 콘솔 네트워킹, vMotion 및 iSCSI 네트워킹과 같은 네트워킹 서비스에 대한 포트 그룹은 네트워킹 추가 마법사를 사용하여 구성할 수 있습니다. 포트 그룹 생성 프로세스 동안 새 Standard Switch를 생성하거나 새 포트 그룹을 이미 존재하는 Standard Switch에 연결할 수 있습니다. 가상 시스템의 NIC를 해당 네트워크의 포트 그룹에 할당하면 가상 시스템이 가상 네트워크에 연결됩니다.
