스토리지 기능 사용

vSphere Web Client의 VM 스토리지 프로파일 뷰에서 스토리지 기능에 액세스할 수 있습니다.

기존의 시스템 정의 스토리지 기능을 확인하고
사용자 정의 스토리지 기능을 추가, 편집 또는 삭제하려면
스토리지 기능 생성, 편집 또는 제거 아이콘을 클릭합니다.

새 VM 스토리지 프로파일에 스토리지 기능을 추가하려면 새 VM 스토리지 프로파일 생성 아이콘을 클릭합니다.

규정 준수를 위해 VM 스토리지 프로파일에 필요한 스토리지 기능 목록을 수정하려면 기존 VM 스토리지 프로파일을 마우스 오른쪽 버튼으로 클릭하고 VM 스토리지 프로파일 편집을 선택합니다.

데이터스토어에 특정 스토리지 기능이 있는지 확인하려면
인벤토리에서 데이터스토어를 마우스 오른쪽 버튼으로 클릭하고 스토리지 기능 할당을 선택합니다.
