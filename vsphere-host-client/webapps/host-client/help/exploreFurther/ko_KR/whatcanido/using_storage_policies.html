VM 스토리지 프로파일 사용

VM 스토리지 프로파일을 가상 시스템에 할당하려면 인벤토리에서 가상 시스템을 선택하고 관리 탭을 클릭한 다음 프로파일을 클릭합니다.

다음과 같은 마법사의 스토리지 페이지에서 VM 스토리지 프로파일을 선택하면 선택한 VM 스토리지 프로파일과 호환되거나 호환되지 않는 데이터스토어가 vSphere Web Client에 표시됩니다.

  • 새 가상 시스템
  • 가상 시스템 복제
  • 가상 시스템 마이그레이션

가상 시스템이나 가상 디스크가 VM 스토리지 프로파일과 호환되지 않는 데이터스토어에서 실행되는 경우,이를 호환되는 데이터스토어로 이동할 수 있습니다.

VM 스토리지 프로파일을 생성, 편집 또는 삭제하려면 vSphere Web Client 홈에서 규칙 및 프로파일을 클릭하고 VM 스토리지 프로파일을 클릭합니다.
