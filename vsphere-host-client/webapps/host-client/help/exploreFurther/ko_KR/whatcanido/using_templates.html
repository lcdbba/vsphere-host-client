템플릿 사용

템플릿을 생성한 후 템플릿을 사용하여 새 가상 시스템을 데이터 센터 내의 모든 호스트에 배포할 수 있습니다. 템플릿을 사용하여 다음 작업을 수행할 수 있습니다.

  • 안정성을 높이기 위해 공유 스토리지에 템플릿 저장
  • 시스템 이름과 네트워크 주소로 템플릿 사용자 지정
  • 확립된 표준으로 템플릿을 구성한 다음 가상 시스템으로 배포
  • 새 가상 시스템 생성 시 오류 및 다운타임 최소화
  • 템플릿을 사용하여 가상 시스템을 패치 및 업데이트
