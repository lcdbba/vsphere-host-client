vSphere Distributed Switch 아키텍처 설정

vSphere Distributed Switch를 사용하여 네트워킹 환경을 설정하는 첫 번째 단계는 vSphere Client의 네트워킹 인벤토리 뷰에 vSphere Distributed Switch를 생성하는 것입니다. vSphere Distributed Switch는 vSphere Client의 네트워크 뷰에서 데이터 센터 수준에 생성됩니다. vSphere Distributed Switch를 생성한 후 호스트 및 분산 포트 그룹을 이러한 스위치에 추가할 수 있습니다.

호스트 네트워킹 옵션은 개별 호스트 네트워킹 구성 페이지를 통해서나 호스트 프로파일을 사용하여 구성합니다.

마지막으로, 가상 시스템 NIC가 분산 포트 그룹에 연결됩니다. 개별 가상 시스템 NIC 구성을 통해서나 vSphere Distributed Switch 자체에서 가상 시스템 네트워킹을 마이그레이션하여 가상 시스템 NIC를 연결합니다.
