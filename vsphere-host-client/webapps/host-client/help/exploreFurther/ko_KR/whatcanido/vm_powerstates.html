가상 시스템 전원 상태

가상 시스템의 전원 상태는 가상 시스템이 활성화된 상태이고 작동 가능한지를 나타냅니다. 가상 시스템에는 여러 전원 상태가 있지만 세 가지 기본 상태는 켜짐, 꺼짐 및 일시 중단입니다. 기본 가상 시스템 전원 작업 외에 기타 기능이 있지만 이러한 기타 기능을 사용하려면 먼저 가상 시스템에 VMware Tools를 설치해야 합니다.

다음과 같은 여러 액세스 지점을 사용하여 이러한 전원 상태를 변경할 수 있습니다.

  • 전원 상태를 수동으로 변경하려면 가상 시스템을 마우스 오른쪽 버튼으로 클릭한 다음 적절한 전원 옵션 메뉴 항목을 선택합니다.
  • 자동 전원 상태 변경 작업을 스케줄링하려면 Ctrl 키를 누르고 가상 시스템을 마우스 오른쪽 버튼으로 클릭한 다음 적절한 전원 옵션 메뉴 항목을 선택합니다.

현재 전원 상태 옵션을 사용할 수 없는 경우 해당 옵션이 해제되어 있으므로 선택할 수 없습니다.
