VMware Tools

VMware Tools는 가상 시스템의 게스트 운영 체제 성능을 향상시키고 가상 시스템의 관리를 개선하는 유틸리티 모음입니다. 가상 시스템에 게스트 운영 체제를 설치한 후 VMware Tools를 설치합니다.

게스트 운영 체제에 VMware Tools를 설치하는 것은 필수적입니다. VMware Tools 없이도 게스트 운영 체제를 실행할 수 있기는 하지만 중요한 기능과 편리함을 놓치게 됩니다. VMware Tools를 설치할 때 표시, 네트워킹, SCSI, 메모리 제어, I/O 및 마우스 지원을 향상시키는 VMware Tools 서비스 및 일련의 VMware 디바이스 드라이버를 설치합니다.

VMware Tools 설치

VMware Tools를 설치하기 전에 브라우저에 VMware 클라이언트 통합 플러그인이 설치되어 있는지 확인합니다.

  1. 인벤토리에서 가상 시스템을 마우스 오른쪽 버튼으로 클릭하고 모든 vCenter 작업 > 게스트 운영 체제 > VMware Tools 설치/업그레이드를 선택합니다.
  2. 가상 시스템의 요약 탭에서 콘솔 시작을 클릭합니다.
  3. 가상 시스템 콘솔에서 설치 마법사의 지시를 따릅니다.

Windows 설치의 경우 마법사가 자동으로 나타나지 않으면 VMware Tools라는 레이블이 지정된 게스트 운영 체제의 CD-ROM 드라이브를 열고 Setup 애플리케이션을 실행합니다.

Linux 설치의 경우 게스트 운영 체제 설치 가이드의 지시를 따릅니다.
