浏览清单

清单是一种用于管理虚拟企业的对象层次结构。这些对象可以是其他对象的容器（如文件夹和数据中心），也可以是您管理的对象（如主机和虚拟机）。清单可显示所有对象和对象关系，并允许您组织这些对象以及访问用于管理对象的 vCenter 操作。

vSphere Web Client 提供了两种在清单中导航的方法。第一种是使用导航器。在导航器中，对象在 vSphere Web Client 的左侧面板中分组为多个类别。选择某一类别可查看该类别中的所有对象。选择某一对象并单击“相关对象”选项卡可查看和浏览与该对象相关联的所有其他对象。

在清单中导航的第二种方法是使用清单树。在清单树中，对象以层次结构排列在下列四个视图之一中：主机和群集、虚拟机和模板、存储和网络。
