什么是 Datastore Cluster？

Datastore Cluster 是共享资源和管理接口的数据存储的集合。将数据存储添加到 Datastore Cluster 时，群集的资源将成为 Datastore Cluster 资源的一部分。通过使用 Datastore Cluster 聚合存储资源，可以在 Datastore Cluster 级别支持资源分配策略。

创建 Datastore Cluster 后，可以使用存储 DRS 管理存储资源。仅当与 Datastore Cluster 中的数据存储连接的所有主机都正在运行 vSphere 版本 5 或更高版本时，可用于存储 DRS 的 I/O 负载平衡功能才可用。
