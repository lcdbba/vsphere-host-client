什么是解决方案？

vCenter 解决方案是 vCenter Server 的扩展，可为 vCenter Server 实例添加新功能。例如，vSphere ESX Agent Manager 是由 VMware 提供的标准 vCenter 解决方案，可用于管理为 ESX 主机添加新功能的 ESX 代理。

可以安装解决方案以将功能从第三方技术添加到 vCenter Server 的标准功能。解决方案通常以 OVF 软件包的形式提供，并可直接从 vSphere Web Client 中安装和部署。
