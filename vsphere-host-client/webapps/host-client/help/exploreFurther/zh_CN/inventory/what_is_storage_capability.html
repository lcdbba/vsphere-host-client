什么是 Storage Capability？

Storage Capability 表示数据存储提供的存储特性。这些特性可以是容量、性能、可用性等。

  • 系统定义的 Storage Capability

    启用了存储感知的存储系统将系统定义的 Storage Capability 分配给该系统上创建的数据存储。

  • 用户定义的 Storage Capability

    手动创建用户定义的 Storage Capability 并将其分配给数据存储，以标识运行在同一存储系统上或具有类似或相同性能、容量、可用性等特性的数据存储。
