什么是虚拟硬件？

正如物理机具有 CPU 和内存等硬件设备，虚拟机也具有虚拟的硬件设备。如果查看虚拟机的配置，您将找到 CPU、内存、硬盘、CD/DVD 驱动器、软盘驱动器、以太网适配器、声卡等设备。每个虚拟设备在虚拟机上执行的功能和物理机上的硬件的功能相同。

虚拟硬件使虚拟机能根据需要来分配物理硬件。例如，您可以在一个物理 CPU 上运行多个虚拟机，每个虚拟机均有其自己的虚拟 CPU。您可以将虚拟机从一个主机移到另一个主机中，此时虚拟设备和物理设备间的映射会自动改变，而虚拟设备则保持不变。通过虚拟硬件，虚拟机不受物理硬件细节的影响。

要查看虚拟设备的完整列表，请参见 vSphere Web Client 联机帮助。
