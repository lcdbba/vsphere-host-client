vSphere 的工作原理是什么？

VMware vSphere 是管理虚拟基础结构的系统。VMware vCenter Server 是管理运行多个虚拟机的多台主机服务器的工具。

借助 vCenter Server，您便能快速置备新的服务器虚拟机并创建标准化虚拟机模板库，从而使新置备的系统始终能够满足数据中心的要求。

通过 vCenter Server，您可以在主机服务器间迁移正在运行的虚拟机，从而以最少的停机时间进行硬件维护。利用 vCenter Server 其他功能可平衡主机间的虚拟机负载并管理虚拟机，以实现高可用性及灾难恢复能力。
