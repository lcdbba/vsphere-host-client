什么是清单树视图？

清单树视图以层次结构显示了由 vCenter Server 管理的所有对象（如数据中心、资源池、群集、网络、数据存储、Datastore Cluster、模板、主机和虚拟机）。

四种清单视图分别为：

  • 主机和群集
    显示除模板、网络和数据存储外的所有清单对象的清单层次结构。
  • 虚拟机和模板
    显示虚拟机和模板的列表。
  • 网络
    显示网络的列表。
  • 数据存储和 Datastore Cluster
    显示数据存储和 Datastore Cluster 的列表。

从 vSphere Web Client 主页中单击 Virtual Infrastructure 可访问清单树视图。
