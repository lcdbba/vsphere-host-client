什么是 vSphere Web Client？

vSphere Web Client 是创建、管理和监控虚拟机及其资源和主机的主界面。另外，它还提供对虚拟机的控制台访问。

vSphere Web Client 是在计算机上安装的一种 Web 应用程序，可通过网络访问 vCenter Server。您可以在 Windows 计算机上安装 vSphere Web Client，也可以将 vSphere Web Client 作为基于 Linux 的 vCenter Server Appliance 的一部分进行部署。您可以使用 Web 浏览器访问 vSphere Web Client。

尽管 vCenter Server 执行所有 vSphere 活动，但您可以使用 vSphere Web Client 来监控、管理和控制 vCenter Server。
