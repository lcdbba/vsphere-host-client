添加 vCenter Server 系统

登录到 vSphere Web Client 后，您可以访问已在安装时向客户端的 Lookup Service 中注册以及您拥有正确权限的所有 vCenter Server 5.1 系统。无需手动向 vSphere Web Client 注册 vCenter Server 5.1 系统。

如果您无法访问 vCenter Server 5.1 系统，请执行以下操作：

  • 确认 vCenter Server 系统已向 vSphere Web Client 的 Lookup Service 中注册。
  • 确认 SSO 管理员已授予您访问 vCenter Server 系统的权限。

要从 vSphere Web Client 管理 vCenter Server 5.0 系统，您必须向 vSphere Web Client 注册 vCenter Server。要向 Windows 上安装的 vSphere Web Client 实例注册 vCenter Server 5.0 系统，请使用 vSphere 管理工具。要向 vCenter Server Appliance 上安装的 vSphere Web Client 实例注册 vCenter Server 5.0 系统，请使用 admin-app 命令行脚本。
