克隆虚拟机

克隆副本是虚拟机的精确副本。创建克隆副本时，可使用 vCenter Server 自定义该虚拟机的客户机操作系统。您可以在原始虚拟机所在的同一数据中心内的任意主机上运行新的克隆副本。

要克隆虚拟机，请选择该虚拟机，然后选择操作 > 所有 vCenter 操作 > 克隆。“克隆虚拟机”向导将引导您完成克隆虚拟机所需的所有步骤。您可以在任何电源状况下克隆虚拟机。

在此过程中，您需要执行以下操作：

  • 为虚拟机提供一个名称，此名称需有别于数据中心内的其他虚拟机和模板。
  • 选择将运行新虚拟机的主机或群集。
  • 为虚拟机文件分配数据存储。

“克隆虚拟机”向导运行期间，您可以自定义新虚拟机的客户机操作系统。选中此选项可启动“客户机操作系统自定义”向导。
