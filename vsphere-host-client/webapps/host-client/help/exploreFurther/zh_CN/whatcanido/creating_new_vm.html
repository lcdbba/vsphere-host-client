创建新虚拟机

创建新虚拟机时，您可以自定义处理器数量、内存、网络连接和存储器等选项。您可以在主机或群集上创建新虚拟机。

创建虚拟机之前，请先决定要将新虚拟机驻留在哪个主机或群集上、将在新虚拟机上安装哪种类型的客户机操作系统以及安装光盘或安装映像文件所在的位置。此外，您还需要适当的 vCenter Server 权限以创建虚拟机。

要创建新虚拟机，请从清单中选择您希望运行新虚拟机的主机或群集，然后选择操作 > 新建虚拟机。

新虚拟机就像一台装有空白硬盘的物理机。新虚拟机创建完成后，您需要在上面安装客户机操作系统。另外，虚拟机的设置可以随时更改。
