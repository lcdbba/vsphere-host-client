创建虚拟机

您可以使用多种方法创建虚拟机，然后将其添加到清单中。例如，您可以创建新的空白虚拟机、将物理机转换成虚拟机、从模板部署或克隆现有虚拟机。

vSphere Web Client 提供了一个简单灵活的用户界面，可用于创建虚拟机。向导会引导您逐步生成一个完整且可以正常运行的虚拟机。

创建虚拟机时，虚拟机将创建在您选择的主机上。虚拟机将在该主机上运行并使用该主机的资源。您可以将虚拟机从一台主机移到同一数据中心的另一台主机中。
