自定义客户机操作系统

从模板部署新虚拟机或克隆现有虚拟机时，可以自定义新的客户机操作系统。您可以自定义虚拟机客户机操作系统的标识和网络设置，以便该系统可在目标环境中立即投入工作。然后，您可以将该虚拟机设置保存为规范，以便日后恢复和重新使用。

在完成“从模板部署”或“克隆虚拟机”向导之前，您可以选择对客户机操作系统进行自定义。如果选择此选项，自定义向导将引导您完成配置选项的设置。

要访问已保存的自定义规范，请在 vSphere Web Client 主页中选择规则和配置文件，然后选择虚拟机自定义规范。您可以在此编辑现有规范，或者创建新规范并存储以供日后使用。
