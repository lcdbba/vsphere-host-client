从模板部署

从模板部署虚拟机后，vCenter Server 允许您自定义此虚拟机的客户机操作系统。您可以在模板所在的同一数据中心内的任意主机上运行新虚拟机。

“部署模板”向导将引导您完成部署模板的所有步骤。要启动该向导，请选择模板并选择操作 > 从该模板部署虚拟机。

在此过程中，您需要执行以下操作：

  • 为虚拟机提供一个名称，此名称需有别于数据中心内的其他虚拟机和模板。
  • 选择将运行新虚拟机的主机或群集。
  • 为虚拟机文件分配数据存储。

“部署模板”向导运行期间，您可以自定义新虚拟机的客户机操作系统。选中此选项可启动“客户机操作系统自定义”向导。
