导入虚拟设备

虚拟设备是安装了操作系统和其他软件的预建虚拟机。导入虚拟设备后，您将拥有一台全功能的虚拟机。

您可从多种来源获取虚拟设备。VMware Virtual Appliance Marketplace 拥有大量可下载的虚拟设备。VMware 或其他供应商也可能会在 CD 或 DVD 光盘上分装虚拟机。

如果已获得虚拟设备，则可以从本地硬盘进行安装，或者也可以通过输入虚拟设备所在的 Internet 位置的 URL 进行安装。

要启动向导，请选择某一主机或群集并选择操作 > 部署 OVF 模板。
