安装步骤：主机 CD-ROM

  1. 将操作系统光盘插入到主机 CD-ROM 驱动器中。
  2. 请在清单中选择虚拟机，然后选择操作 > 打开电源。
  3. 在虚拟机的“摘要”选项卡上，单击“连接 CD/DVD-ROM”图标，然后选择连接到主机 CD 设备。
  4. 单击启动控制台链接。
  5. 打开虚拟机电源后，可像在任何其他计算机上一样安装客户机操作系统。
    安装 Windows XP 需要其他驱动程序。有关特定操作系统的安装说明，请参见客户机操作系统安装指南。
  6. 客户机操作系统安装运行后，请安装 VMware Tools。这可以提高虚拟机的性能。
