分步安装指南

要安装客户机操作系统，请执行以下步骤：

  1. 确保您的浏览器中已安装 VMware 客户端集成插件。
  2. 在清单中选择虚拟机。
  3. 选择操作 > 编辑设置以打开“编辑虚拟机”对话框。
  4. 在虚拟硬件选项卡上，单击要将操作系统 CD 插入到的 CD/DVD 驱动器。
  5. 在右侧面板中的“设备类型”下选择操作系统 CD/DVD-ROM 或 ISO 映像文件的位置。
  6. 如果选择了数据存储 ISO 文件，请浏览到该 ISO 文件并选择该文件。对于“数据存储 ISO 文件”和“主机设备”选项，请选中打开电源时连接复选框。
  7. 单击确定。
  8. 有关特定安装说明，请单击以下链接之一：
     
    • 客户端 CD-ROM 驱动器
    • 主机 CD-ROM 驱动器
    • ISO 映像文件
