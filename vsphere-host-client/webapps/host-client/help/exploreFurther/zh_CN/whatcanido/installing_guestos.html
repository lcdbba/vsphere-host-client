安装客户机操作系统

新虚拟机就像一台装有空白硬盘的物理机。在使用之前，需要先安装操作系统。虚拟机上的操作系统称为客户机操作系统。

在虚拟机上安装客户机操作系统与在物理机上安装操作系统的方法基本相同。您必须拥有操作系统供应商提供的包含安装文件的 CD-ROM 或 ISO 映像。与物理机一样，每次安装都需要单独的操作系统许可证。

安装客户机操作系统后，您就可以通过 vCenter Server 使用虚拟机执行诸如安装应用程序或管理电源操作等任务，就像使用物理机时一样。

可以从多种不同位置安装客户机操作系统：
客户端 CD-ROM 驱动器
主机 CD-ROM 驱动器
ISO 映像文件
