使用群集

使用群集可最大限度地利用主机，并为虚拟机提供高可用性。群集在主机间分发资源并管理负载平衡。使用群集，可以启用以下 VMware 解决方案：

  • vSphere Distributed Resource Scheduler (DRS)
    vSphere DRS 可持续保持 ESX/ESXi 主机间虚拟机工作负载的平衡。当运行虚拟机的主机所提供的 CPU 或内存资源无法满足虚拟机活动所需时，vCenter Server 系统会触发 vMotion 迁移来平衡工作负载。
  • vSphere High Availability (HA)
    vSphere HA 可使运行在 ESX/ESXi 主机上的虚拟机从主机故障中自动恢复过来。一台主机出现故障时，VMware HA 会立即重新启动群集中其他主机上的所有关联的虚拟机。
  • VMware Enhanced vMotion Compatibility (EVC)
    EVC 可确保群集内的所有主机向虚拟机提供相同的 CPU 功能集，从而帮助确保群集中各主机之间的 vMotion 兼容性。

此外，您还可以通过创建资源池将群集中的可用 CPU 和内存资源以层次结构形式进行划分，从而进一步优化群集的使用情况。
