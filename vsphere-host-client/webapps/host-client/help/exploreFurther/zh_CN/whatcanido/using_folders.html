使用文件夹

可以使用文件夹对以下清单对象进行分组，以进一步细化清单内的对象分组：数据中心、虚拟机和模板、主机和群集、数据存储和 Datastore Cluster 以及网络对象。

划分到同一个文件夹中的对象必须为同一种类型。

创建文件夹：

  • 选择 vCenter Server 系统，然后选择操作 > 新建文件夹。
  • 选择数据中心，然后选择操作 > 所有 vCenter 操作 > 新建主机和群集文件夹、新建网络文件夹、新建存储文件夹或新建虚拟机和模板文件夹。

要将某一对象添加到某一文件夹中，请选择该对象，然后选择操作 > 移至。选择该文件夹作为该对象的目标。

要删除某一文件夹，请在清单中选择该文件夹，然后选择操作 > 所有 vCenter 操作 > 从清单中移除。删除文件夹时会删除其中的所有清单项目。如果文件夹中的项目为主机，则会断开其与 vCenter Server 系统的连接。
