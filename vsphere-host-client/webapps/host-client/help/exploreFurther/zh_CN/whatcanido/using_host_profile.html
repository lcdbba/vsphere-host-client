使用主机配置文件

“主机配置文件”视图是 vSphere Web Client 中的主机配置文件管理区域，管理员可在其中创建、编辑或删除主机配置文件。要从 vSphere Web Client 主页访问该视图，请单击规则和配置文件，然后单击主机配置文件。

要执行其他主机配置文件操作（如附加配置文件），请在清单中右键单击主机或群集，然后从所有 vCenter 操作 > 主机配置文件中选择相应的操作。要在群集级别执行配置文件操作（如检查合规性），请选择群集并单击监控选项卡，然后单击配置文件合规性。

要使用主机配置文件，请将其附加并应用到清单中的某个主机上即可。您可以一次选择多个主机或者选择一个群集以附加主机配置文件。
