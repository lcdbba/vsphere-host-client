使用资源池

创建资源池后，vCenter Server 可管理共享资源并将它们分配到资源池内的虚拟机。使用资源池，您可以：

  • 将处理器和内存资源分配给运行在同一个主机或群集上的多台虚拟机。
  • 设置 CPU 和内存的最小、最大以及按比例分配的资源份额。
  • 于虚拟机正在运行之时修改分配。
  • 使得应用程序可动态获取更多资源，从而实现最佳性能。

使用 vSphere Web Client 可将虚拟机添加到资源池、查看资源使用情况以及设置资源预留、限制和份额。
