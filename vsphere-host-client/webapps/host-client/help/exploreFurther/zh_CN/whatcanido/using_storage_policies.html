使用存储服务类

要向虚拟机分配存储服务类，请在清单中选择虚拟机，单击管理选项卡，然后单击服务类。

在以下向导的“存储”页面中选择存储服务类时，vSphere Web Client 将显示与该存储服务类兼容或不兼容的数据存储：

  • 新建虚拟机
  • 克隆虚拟机
  • 迁移虚拟机

如果虚拟机或虚拟磁盘在与存储服务类不兼容的数据存储上运行，您可以将它们移动到兼容的数据存储。

要创建、编辑或删除存储服务类，请从 vSphere Web Client 主页中单击规则和配置文件，然后单击存储服务类。
