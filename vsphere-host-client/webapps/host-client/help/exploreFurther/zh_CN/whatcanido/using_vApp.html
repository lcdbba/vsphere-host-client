使用 vApp

vApp 与虚拟机的基本操作相同，但可以含有多个虚拟机或多个服务。

与独立的虚拟机一样，vApp 是可以使用 vSphere 管理的清单对象。在 vSphere Web Client 中，vApp 显示在 vApp 清单列表以及主机、群集、虚拟机和模板清单树中。在这些视图中，您可以创建 vApp、配置 vApp、打开或关闭 vApp 电源以及克隆 vApp，就如同对独立的虚拟机所进行的操作一样。

要指定包含在 vApp 中的虚拟机启动顺序，请编辑 vApp 属性。

可以导入和导出 vApp。vApp 的分发格式是 OVF。
