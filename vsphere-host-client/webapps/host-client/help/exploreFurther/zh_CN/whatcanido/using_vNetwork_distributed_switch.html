设置 vSphere Distributed Switch 架构

使用 vSphere Distributed Switch 设置网络环境的第一步是，在 vSphere Client 的网络清单视图中创建 vSphere Distributed Switch。vSphere Distributed Switch 将在 vSphere Client 的网络视图中的数据中心级别创建。vSphere Distributed Switch 创建后，主机和分布式端口组即被添加到其中。

通过独立的主机网络配置页面或使用主机配置文件可配置主机网络选项。

最后，将虚拟机的网卡连接到分布式端口组。通过独立的虚拟机网卡配置或通过从 vSphere Distributed Switch 本身迁移虚拟机均可连接虚拟机网卡。
