虚拟机电源状态

虚拟机的电源状况指示虚拟机是处于活动还是正常运行状态。虚拟机有多种电源状况，但三种基本状况分别为：启动、关闭和挂起。除基本的虚拟机电源操作外，虚拟机还具备一些其他功能，但必须先安装 VMware Tools 才能使用这些功能。

可使用多个访问点更改这些电源状况：

  • 要手动更改电源状况，请右键单击虚拟机，然后选择适当的电源选项菜单项。
  • 要调度自动更改电源状况的任务，请按 Ctrl 键并右键单击虚拟机，然后选择适当的电源选项菜单项。

如果某项电源状况选项当前不可用，该选项将被禁用并且无法选中。
