VMware Tools

VMware Tools 是一套实用程序，能够提高虚拟机的客户机操作系统的性能，并增强虚拟机的管理。请先在虚拟机上安装客户机操作系统，而后再安装 VMware Tools。

在客户机操作系统中安装 VMware Tools 非常重要。尽管客户机操作系统在不安装 VMware Tools 的情况下也可以运行，但这将失去重要的功能性和便利性。安装 VMware Tools 时会安装 VMware Tools 服务及一组 VMware 设备驱动程序，它们可增强显示、网络、SCSI、内存控制、I/O 和鼠标支持等功能。

安装 VMware Tools

安装 VMware Tools 前，请确保您的浏览器中已安装 VMware 客户端集成插件。

  1. 右键单击清单中的虚拟机，然后选择所有 vCenter 操作 > 客户机操作系统 > 安装/升级 VMware Tools。
  2. 单击虚拟机摘要选项卡中的启动控制台。
  3. 在虚拟机控制台中，请按照安装向导中的说明进行操作。

对于 Windows 安装，如果向导没有自动出现，请打开客户机操作系统中标记为 VMware Tools 的 CD-ROM 驱动器，然后运行安装应用程序。

对于 Linux 安装，请按照客户机操作系统安装指南中的说明进行安装。
