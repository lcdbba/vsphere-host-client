瀏覽詳細目錄

詳細目錄是可讓您管理虛擬企業的物件階層。這些物件可以是其他物件 (如資料夾和資料中心) 的容器，也可以是您所管理的物件 (如主機和虛擬機器)。詳細目錄顯示所有物件和物件關聯性，並可讓您組合管理這些物件以及存取用於管理物件的 vCenter 作業。

vSphere Web Client 提供兩種方法來導覽詳細目錄。第一種是導覽器。在導覽器中，物件在 vSphere Web Client 的左側面板中分組為多個類別。選取某個類別即可檢視該類別中的所有物件。選取個別物件並按一下「相關物件」索引標籤即可檢視和瀏覽與該物件相關聯的所有其他物件。

導覽詳細目錄的第二種方法是詳細目錄樹狀結構。在這些樹狀結構中，物件以階層方式排列在下列四個視圖中:「主機和叢集」、「虛擬機器和範本」、「儲存空間」和「網路」。
