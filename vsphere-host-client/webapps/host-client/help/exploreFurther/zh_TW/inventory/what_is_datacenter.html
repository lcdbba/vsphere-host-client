什麼是資料中心?

資料中心是主機和虛擬機器等詳細目錄物件的主要容器。從資料中心可新增和組合管理詳細目錄物件。一般而言，您會將主機、資料夾和叢集新增至資料中心。

vCenter Server 可包含多個資料中心。大型公司可能會使用多個資料中心來代表企業內的組織單位。

詳細目錄物件可以在資料中心內互動，但資料中心之間的互動是受到限制的。例如，您可以採用 vMotion 技術在一個資料中心內的主機之間移動虛擬機器，但不能將虛擬機器移至另一個資料中心的主機上。
