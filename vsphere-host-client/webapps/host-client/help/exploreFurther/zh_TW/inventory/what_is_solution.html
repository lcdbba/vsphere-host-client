什麼是解決方案?

vCenter 解決方案是 vCenter Server 的擴充功能，可向 vCenter Server 執行個體新增功能。例如，vSphere ESX Agent Manager 是由 VMware 提供的標準 vCenter 解決方案，可讓您管理用於向 ESX 主機新增功能的 ESX 代理程式。

您可以安裝解決方案，以將第三方技術的功能新增至 vCenter Server 的標準功能。解決方案通常以 OVF 套件的形式提供，並可直接從 vSphere Web Client 進行安裝和部署。
