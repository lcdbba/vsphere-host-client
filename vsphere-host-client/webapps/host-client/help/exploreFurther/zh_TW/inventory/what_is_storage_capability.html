什麼是儲存空間功能?

儲存空間功能指示資料存放區提供的儲存空間特性。這些特性可以是容量、效能、可用性等等。

  • 系統定義的儲存空間功能

    已啟用儲存空間感知功能的儲存空間系統會將系統定義的儲存空間功能指派給在該系統上建立的資料存放區。

  • 使用者定義的儲存空間功能

    您可以手動建立使用者定義的儲存空間功能並將其指派給資料存放區，以識別在同一個儲存空間系統上執行的資料存放區，或具有類似或相同效能、容量、可用性等特性的資料存放區。
