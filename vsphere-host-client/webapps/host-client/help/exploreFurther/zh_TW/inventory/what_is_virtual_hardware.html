什麼是虛擬硬體?

正如實體電腦配有 CPU 和記憶體等硬體裝置一樣，虛擬機器也配有虛擬硬體裝置。查看下虛擬機器的組態，您會找到 CPU、記憶體、硬碟、CD/DVD 光碟機、軟碟機、乙太網路卡、音效卡等裝置。每台虛擬裝置在虛擬機器上執行的功能如同硬體在實體電腦上執行的功能。

虛擬硬體讓虛擬機器能夠根據需要來分配實體硬體。例如，您可以在單一實體 CPU 上執行多台虛擬機器，每台虛擬機器可能有自己的虛擬 CPU。您可以將虛擬機器從一台主機移到另一台主機中，此時虛擬裝置和實體裝置之間的對應會自動變更，而虛擬裝置則保持不變。透過虛擬硬體，虛擬機器便不會受到實體硬體細節的影響。

如需虛擬裝置的完整清單，請參閱 vSphere Web Client 線上說明。
