vSphere 如何運作?

VMware vSphere 是用於管理虛擬基礎結構的系統。VMware vCenter Server 是一種工具，可用來管理執行多台虛擬機器的多台主機伺服器。

透過 vCenter Server，您可以快速佈建新的伺服器虛擬機器並建立標準化虛擬機器範本程式庫，從而使新佈建的系統始終滿足資料中心的需求。

vCenter Server 可讓您在主機伺服器之間移轉執行中的虛擬機器，從而以最少的停機時間執行硬體維護工作。vCenter Server 的其他功能可讓您平衡主機之間的虛擬機器工作負載並管理虛擬機器，以實現高可用性及災害復原能力。
