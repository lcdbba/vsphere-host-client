什麼是詳細目錄樹狀結構視圖?

詳細目錄樹狀結構視圖可供您檢視由 vCenter Server 管理的所有物件，例如，資料中心、資源集區、叢集、網路、資料存放區、資料存放區叢集、範本、主機和虛擬機器。

四個詳細目錄視圖分別為:

  • 主機和叢集
    顯示除範本、網路和資料存放區以外的所有詳細目錄物件的詳細目錄階層。
  • 虛擬機器和範本
    顯示虛擬機器和範本清單。
  • 網路
    顯示網路清單。
  • 資料存放區和資料存放區叢集
    顯示資料存放區和資料存放區叢集清單。

按一下 vSphere Web Client 首頁中的「虛擬基礎結構」即可存取詳細目錄樹狀結構視圖。
