什麼是 vCenter Server?

VMware vCenter Server 向資料中心提供便利的單一控制點。它在 Windows 伺服器的頂層執行，以集中管理 VMware ESX/ESXi 主機，並提供基本的資料中心服務，如存取控制、效能監控和組態。

vCenter Server 可整合個別主機的資源，以供整個資料中心的所有虛擬機器共用。實現這一目的的方式是，根據系統管理員設定的原則，管理主機的虛擬機器指派以及指定主機內虛擬機器的資源指派。

vCenter Server 可讓您使用 vSphere DRS、vSphere HA 和 vMotion 等 vSphere 進階功能。
