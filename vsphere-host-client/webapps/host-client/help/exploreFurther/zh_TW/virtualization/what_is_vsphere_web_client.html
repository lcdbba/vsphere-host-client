什麼是 vSphere Web Client?

vSphere Web Client 是用於建立、管理和監控虛擬機器及其資源和主機的主要界面。它還提供對虛擬機器的主控台存取。

vSphere Web Client 是一種 Web 應用程式，安裝在可透過網路存取 vCenter Server 的機器上。您可以將 vSphere Web Client 安裝在 Windows 機器上，也可以將其部署為基於 Linux 之 vCenter Server Appliance 的一部分。您可以使用網頁瀏覽器存取 vSphere Web Client。

雖然 vCenter Server 負責執行所有 vSphere 活動，但您可使用 vSphere Web Client 來監控、管理和控制 vCenter Server。
