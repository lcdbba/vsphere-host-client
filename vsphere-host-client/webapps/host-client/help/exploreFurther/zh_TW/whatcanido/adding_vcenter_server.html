新增 vCenter Server 系統

登入 vSphere Web Client 後，您可以存取已於安裝期間向用戶端 Lookup Service 登錄且您對其擁有正確權限的所有 vCenter Server 5.1 系統。您不必手動向 vSphere Web Client 登錄 vCenter Server 5.1 系統。

如果您對 vCenter Server 5.1 系統不具有存取權限，請執行下列動作:

  • 確認已向 vSphere Web Client 的 Lookup Service 登錄 vCenter Server 系統。
  • 確認 SSO 系統管理員已向您授予 vCenter Server 系統的存取權限。

若要從 vSphere Web Client 管理 vCenter Server 5.0 系統，您必須向 vSphere Web Client 登錄 vCenter Server。若要向 Windows 上所安裝的 vSphere Web Client 執行個體登錄 vCenter Server 5.0 系統，請使用 vSphere 系統管理工具。若要向 vCenter Server Appliance 上所安裝的 vSphere Web Client 執行個體登錄 vCenter Server 5.0 系統，請使用 admin-app 命令行指令碼。
