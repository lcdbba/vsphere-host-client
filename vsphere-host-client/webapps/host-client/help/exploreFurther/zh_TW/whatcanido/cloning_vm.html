複製虛擬機器

複製品是虛擬機器的完整複本。建立複製品時，vCenter Server 可讓您自訂該虛擬機器的客體作業系統。您可以在原始虛擬機器所在之相同資料中心內的任一主機上執行新的複製品。

若要複製虛擬機器，請選取虛擬機器，然後選取「動作」>「所有 vCenter 動作」>「複製」。「複製虛擬機器」精靈將引導您完成複製虛擬機器所需的步驟。您可以在任何電源狀態下複製虛擬機器。

在此程序中，您需要:

  • 為虛擬機器提供一個名稱，此名稱需有別於資料中心內的其他虛擬機器和範本。
  • 選擇新虛擬機器執行所在的主機或叢集。
  • 為虛擬機器檔案指派資料存放區。

在「複製虛擬機器」精靈執行期間，您可以自訂新虛擬機器的客體作業系統。選擇此選項可啟動「客體作業系統自訂」精靈。
