建立新的虛擬機器

建立新的虛擬機器可讓您自訂處理器數目、記憶體、網路連線和儲存空間等選項。您可以在主機或叢集上建立新的虛擬機器。

建立虛擬機器之前，請決定用於存放新虛擬機器的主機或叢集、在新虛擬機器上安裝的客體作業系統類型，以及安裝 CD 或映像檔的位置。您還需要適當的 vCenter Server 權限才能建立虛擬機器。

若要建立新的虛擬機器，請在詳細目錄中選取希望在其上執行新虛擬機器的主機或叢集，然後選取「動作」>「新增虛擬機器」。

新的虛擬機器就像一台裝有空白硬碟的實體電腦。建立新的虛擬機器後，您需要在該虛擬機器上安裝客體作業系統。您也可以隨時變更虛擬機器的設定。
