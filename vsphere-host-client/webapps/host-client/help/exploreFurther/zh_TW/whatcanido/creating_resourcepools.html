建立資源集區

建立資源集區時，您可以為一組虛擬機器指定 CPU 和記憶體共用率等資源。您還可以指定保留區、限制和共用率，以決定資源配置方式。

若要建立資源集區，請選取主機並選取「動作」>「所有 vCenter 動作」>「新增資源集區」，或選取叢集並選取「動作」>「新增資源集區」。「新增資源集區」精靈將引導您完成建立程序。

建立資源集區後，將虛擬機器拖至資源集區物件上即可將虛擬機器新增至資源集區。新增至資源集區的虛擬機器會根據資源集區的組態共用 CPU 和記憶體資源。
