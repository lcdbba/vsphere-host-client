建立範本

您可以使用下列方式建立範本:

  • 將虛擬機器轉換成範本
  • 將虛擬機器複製到範本
  • 複製現有範本

將虛擬機器複製到範本時，將建立一個與該虛擬機器具有相同組態的新範本，而詳細目錄中的原始虛擬機器仍然可用。同樣地，複製範本時，也將建立一個範本複本並保留原始範本。

若要建立範本，請從詳細目錄中選取虛擬機器，然後選取「動作」>「所有 vCenter 動作」>「範本」>「轉換成範本」或「動作」>「所有 vCenter 動作」>「範本」>「複製到範本」。若要複製現有範本，請選取該範本，然後選取「動作」>「複製」。

建立範本後，vCenter Server 將在「虛擬機器和範本」詳細目錄樹狀結構或「虛擬機器範本」詳細目錄清單中新增此範本。
