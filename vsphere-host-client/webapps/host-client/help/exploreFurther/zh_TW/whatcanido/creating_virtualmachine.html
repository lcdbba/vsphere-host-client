建立虛擬機器

您可以使用多種方式建立虛擬機器，然後將其新增至詳細目錄。例如，您可以建立新的空白虛擬機器、將實體電腦轉換成虛擬機器、從範本部署或複製現有的虛擬機器。

vSphere Web Client 提供簡單且具有彈性的使用者介面，可讓您在其中建立虛擬機器。精靈會引導您逐步產生一個完整且運作正常的虛擬機器。

建立虛擬機器後，它將建立在您選取的主機上。虛擬機器在該主機上執行並使用該主機的資源。您可以在同一個資料中心內的主機之間移動虛擬機器。
