自訂客體作業系統

從範本部署新的虛擬機器或複製現有的虛擬機器時，可以自訂新的客體作業系統。您可以自訂虛擬機器客體作業系統的身分識別和網路設定，以便該系統可在目標環境中立即開始運作。然後，您可以將虛擬機器設定儲存在規格中，以供日後重新叫用和重新使用。

在完成「從範本部署」或「複製虛擬機器」精靈之前，您可以選擇自訂客體作業系統。如果選擇此選項，自訂精靈將引導您完成組態選項。

若要存取已儲存的自訂規格，請從 vSphere Web Client 首頁選取「規則和設定檔」，然後選取「虛擬機器自訂規格」。您可以在此編輯現有規格，或建立新規格並將其儲存以供日後使用。
