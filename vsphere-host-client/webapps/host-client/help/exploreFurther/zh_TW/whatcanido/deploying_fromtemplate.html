從範本部署

從範本部署虛擬機器時，vCenter Server 可讓您自訂該虛擬機器的客體作業系統。您可以在範本所在之相同資料中心內的任一主機上執行新的虛擬機器。

「部署範本」精靈將引導您完成範本部署步驟。若要啟動精靈，請選取範本，然後選取「動作」>「從這個範本部署虛擬機器」。

在此程序中，您需要:

  • 為虛擬機器提供一個名稱，此名稱需有別於資料中心內的其他虛擬機器和範本。
  • 選擇新虛擬機器執行所在的主機或叢集。
  • 為虛擬機器檔案指派資料存放區。

在「部署範本」精靈執行期間，您可以自訂新虛擬機器的客體作業系統。選擇此選項可啟動「客體作業系統自訂」精靈。
