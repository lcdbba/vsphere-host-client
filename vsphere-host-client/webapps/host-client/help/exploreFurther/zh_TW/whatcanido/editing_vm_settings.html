編輯虛擬機器設定

「編輯虛擬機器」對話方塊是一個編輯器，可用來變更虛擬機器的硬體組態、虛擬機器選項、SDRS 規則及 vApp 選項。

在大多數情況下，您必須關閉虛擬機器電源才能修改其設定。

若要存取虛擬機器設定，請選取要編輯的虛擬機器，然後選取「動作」>「編輯設定」。

「虛擬機器內容」對話方塊包含四個索引標籤: 「虛擬硬體」、「虛擬機器選項」、「SDRS 規則」和「vApp 選項」。「虛擬硬體」索引標籤可讓您新增、移除和重新設定虛擬硬體裝置。「虛擬機器選項」索引標籤可讓您修改虛擬機器的組態設定。「SDRS 規則」索引標籤可讓您修改虛擬機器的 SDRS 規則。「vApp 選項」索引標籤可讓您設定 OVF 內容，並指定此虛擬機器的 IP 配置和產品資訊。
