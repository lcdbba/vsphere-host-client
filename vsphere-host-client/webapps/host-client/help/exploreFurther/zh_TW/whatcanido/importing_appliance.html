匯入虛擬應用裝置

虛擬應用裝置是已安裝作業系統和其他軟體之預先建立的虛擬機器。匯入虛擬應用裝置後，虛擬機器即可完全正常運作。

您可從各種來源取得虛擬應用裝置。VMware Virtual Appliance Marketplace 提供大量可供下載的虛擬應用裝置。VMware 或其他廠商可能也會在 CD 或 DVD 光碟上散發虛擬機器。

如果您已取得虛擬應用裝置，您可以從本機硬碟進行安裝，或者可以輸入虛擬應用裝置所在之網際網路位置的 URL。

若要啟動精靈，請選取主機或叢集，然後選取「動作」>「部署 OVF 範本」。
