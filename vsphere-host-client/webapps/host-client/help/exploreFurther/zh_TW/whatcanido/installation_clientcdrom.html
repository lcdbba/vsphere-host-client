安裝步驟:用戶端 CD-ROM

  1. 將作業系統光碟插入用戶端 CD-ROM 光碟機中。
  2. 在詳細目錄中選取虛擬機器，然後選取「動作」>「開啟電源」。
  3. 在虛擬機器的「摘要」索引標籤上，按一下「連線 CD/DVD-ROM」圖示，然後選取「連線到本機磁碟上的 CD/DVD 映像」。
  4. 按一下「啟動主控台」連結。
  5. 開啟虛擬機器電源後，依照在任何其他電腦上的相同方式安裝客體作業系統。
    安裝 Windows XP 需要其他驅動程式。如需特定作業系統安裝指示，請參閱《客體作業系統安裝指南》。
  6. 客體作業系統安裝完成並執行後，請安裝 VMware Tools。這可以提高虛擬機器的效能。
