逐步安裝指南

安裝客體作業系統:

  1. 確認您的瀏覽器中已安裝 VMware 用戶端整合外掛程式。
  2. 在詳細目錄中選取虛擬機器。
  3. 選取「動作」>「編輯設定」以開啟「編輯虛擬機器」對話方塊。
  4. 在「虛擬硬體」索引標籤上，按一下要將作業系統 CD 插入其中的 CD/DVD 光碟機。
  5. 在右側面板的「裝置類型」下選取作業系統 CD/DVD-ROM 或 ISO 映像檔的位置。
  6. 如果已選取「資料存放區 ISO 檔案」，請瀏覽至該 ISO 檔案並選取該檔案。對於「資料存放區 ISO 檔案」和「主機裝置」選項，請勾選「開啟電源時連線」核取方塊。
  7. 按一下「確定」。
  8. 如需特定安裝指示，請按下方其中一個連結:
     
    • 用戶端 CD-ROM 光碟機
    • 主機 CD-ROM 光碟機
    • ISO 映像檔
