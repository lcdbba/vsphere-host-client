安裝客體作業系統

新的虛擬機器就像一台裝有空白硬碟的實體電腦。您必須安裝作業系統才可使用該虛擬機器。虛擬機器上的作業系統稱為客體作業系統。

在虛擬機器上安裝客體作業系統與在實體電腦上安裝作業系統的方法基本相同。您必須擁有作業系統廠商提供的 CD-ROM 或 ISO 映像 (含安裝檔案)。與實體電腦一樣，每次安裝都需要單獨的作業系統授權。

安裝客體作業系統後，vCenter Server 可讓您使用虛擬機器來執行各項工作，例如安裝應用程式或管理電源作業，就像使用實體電腦時一樣。

您可以從多個不同位置安裝客體作業系統:
用戶端 CD-ROM 光碟機
主機 CD-ROM 光碟機
ISO 映像檔
