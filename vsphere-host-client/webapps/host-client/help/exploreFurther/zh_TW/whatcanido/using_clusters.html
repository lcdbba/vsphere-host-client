使用叢集

使用叢集可盡可能提高主機使用率，並為虛擬機器提供高可用性。叢集會在主機之間分配資源並管理負載平衡。透過使用叢集，您可以啟用下列 VMware 解決方案:

  • vSphere Distributed Resource Scheduler (DRS)
    vSphere DRS 可持續平衡 ESX/ESXi 主機之間的虛擬機器工作負載。當虛擬機器活動所需的 CPU 或記憶體資源超過其執行所在主機能夠提供的資源時，vCenter Server 系統便會觸發 vMotion 移轉功能以平衡工作負載。
  • vSphere High Availability (HA)
    vSphere HA 可讓 ESX/ESXi 主機上執行的虛擬機器自動從主機故障中復原。當某台主機出現故障時，叢集中其他主機上所有相關聯的虛擬機器都會立即重新啟動。
  • VMware Enhanced vMotion Compatibility (EVC)
    EVC 可確保叢集中的所有主機均向虛擬機器提供相同的 CPU 功能集，從而協助確保叢集中主機之間的 vMotion 相容性。

您還可以建立資源集區，以階層方式分割叢集中可用的 CPU 和記憶體資源，從而進一步優化叢集使用情況。
