使用資料夾

您可以將下列詳細目錄物件分組到資料夾中，以進一步細分詳細目錄內的物件群組:資料中心、虛擬機器和範本、主機和叢集、資料存放區和資料存放區叢集以及網路物件。

分到同一個資料夾中的物件必須屬於相同類型。

建立資料夾:

  • 選取 vCenter Server 系統，然後選取「動作」>「新增資料夾」。
  • 選取資料中心，然後選取「動作」>「所有 vCenter 動作」>「新增主機和叢集資料夾」、「新增網路資料夾」、「新增儲存空間資料夾」或「新增虛擬機器和範本資料夾」。

若要將物件新增至資料夾，請選取物件，然後選取「動作」>「移至」。選取該資料夾作為該物件的目的地。

若要刪除資料夾，請在詳細目錄中選取資料夾，然後選取「動作」>「所有 vCenter 動作」>「從詳細目錄中移除」。刪除資料夾時也會刪除其中的所有詳細目錄項目。如果資料夾中的項目為主機，則會中斷這些主機與 vCenter Server 系統的連線。
