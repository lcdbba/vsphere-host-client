使用主機設定檔

「主機設定檔」視圖是 vSphere Web Client 中的主機設定檔管理區域，系統管理員可在此建立、編輯或刪除主機設定檔。若要從 vSphere Web Client 首頁存取此視圖，請按一下「規則和設定檔」，然後按一下「主機設定檔」。

若要執行其他的主機設定檔動作 (如附加設定檔)，請在詳細目錄中的主機或叢集上按一下滑鼠右鍵，然後從「所有 vCenter 動作」>「主機設定檔」選取適當的動作。若要執行叢集層級設定檔作業 (如檢查符合性)，請選取叢集，然後按一下「監控」索引標籤，再按一下「設定檔符合性」。

若要使用主機設定檔，請將其附加並套用至詳細目錄中的主機。您可以一次選取多台主機或選取一個叢集以用於附加主機設定檔。
