使用資源集區

建立資源集區後，vCenter Server 可管理共用資源並將其配置給資源集區內的虛擬機器。透過使用資源集區，您可以:

  • 將處理器和記憶體資源配置給在相同主機或叢集上執行的虛擬機器。
  • 建立 CPU 和記憶體共用的資源下限、上限和比例。
  • 於虛擬機器正在執行時修改配置。
  • 讓應用程式動態取得更多資源，以實現最佳效能。

使用 vSphere Web Client 可將虛擬機器新增至資源集區、檢視資源使用情況，以及設定資源保留區、限制和共用率。
