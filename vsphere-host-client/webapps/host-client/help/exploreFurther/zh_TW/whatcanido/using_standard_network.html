設定 vSphere Standard Switch 架構

從主機「管理」索引標籤的「網路」索引標籤可設定使用 vSphere Standard Switch 的網路環境。使用「新增網路」精靈可設定虛擬機器網路或網路服務 (例如，服務主控台網路、vMotion 和 iSCSI 網路) 的連接埠群組。在連接埠群組建立程序中可建立新的 Standard Switch，或者您可以將新的連接埠群組連線到現有的 Standard Switch。透過將虛擬機器的 NIC 指派給虛擬網路的連接埠群組，可將此虛擬機器連線到該網路。
