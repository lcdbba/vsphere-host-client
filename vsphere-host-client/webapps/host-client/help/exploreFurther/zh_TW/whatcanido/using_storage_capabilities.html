使用儲存空間功能

您可以從 vSphere Web Client 的「虛擬機器儲存空間原則」視圖存取儲存空間功能。

按一下「建立、編輯或移除儲存空間功能」圖示，以檢視
系統定義的現有儲存空間功能，以及新增、編輯或刪除
使用者定義的儲存空間功能。

按一下「建立新的虛擬機器儲存空間原則」圖示，以將儲存空間功能新增至新的虛擬機器儲存空間原則。

在現有的虛擬機器儲存空間原則上按一下滑鼠右鍵並選取「編輯虛擬機器儲存空間原則」，以修改虛擬機器儲存空間原則要符合規定所需的儲存空間功能清單。

在詳細目錄中的資料存放區上按一下滑鼠右鍵並選取
「指派儲存空間功能」，以確定該資料存放區具有特定的儲存空間功能。
