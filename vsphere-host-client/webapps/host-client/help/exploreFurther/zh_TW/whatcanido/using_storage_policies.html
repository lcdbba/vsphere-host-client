使用虛擬機器儲存空間原則

若要將虛擬機器儲存空間原則指派給虛擬機器，請在詳細目錄中選取虛擬機器，然後依序按一下「管理」索引標籤和「虛擬機器儲存空間原則」。

在下列精靈的「儲存空間」頁面中選取虛擬機器儲存空間原則時，vSphere Web Client 會顯示與該虛擬機器儲存空間原則相容或不相容的資料存放區:

  • 新增虛擬機器
  • 複製虛擬機器
  • 移轉虛擬機器

如果虛擬機器或虛擬磁碟在與虛擬機器儲存空間原則不相容的資料存放區上執行，您可以將其移至相容的資料存放區。

若要建立、編輯或刪除虛擬機器儲存空間原則，請從 vSphere Web Client 首頁按一下「規則和設定檔」，然後按一下「虛擬機器儲存空間原則」。
