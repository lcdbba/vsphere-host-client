使用 vApp

vApp 與虛擬機器的基本作業相同，但可以包含多台虛擬機器或多個 vApp。

與個別虛擬機器一樣，vApp 是可以使用 vSphere 進行管理的詳細目錄物件。在 vSphere Web Client 中，vApp 顯示在 vApp 詳細目錄清單及「主機和叢集」和「虛擬機器和範本」詳細目錄樹狀結構中。從這些視圖中，您可以建立、設定和複製 vApp 以及開啟和關閉其電源，如同操作個別虛擬機器時一樣。

若要指定 vApp 中所含虛擬機器的啟動順序，請編輯 vApp 內容。

您可以匯入和匯出 vApps。vApp 的散發格式是 OVF。
