設定 vSphere Distributed Switch 架構

設定使用 vSphere Distributed Switch 的網路環境時，第一步是要在 vSphere Client 的「網路」詳細目錄視圖中建立 vSphere Distributed Switch。vSphere Distributed Switch 建立在 vSphere Client 的「網路」視圖中的資料中心層級。建立 vSphere Distributed Switch 後，可向其新增主機和分散式連接埠群組。

透過個別主機網路組態頁面或使用主機設定檔可設定主機網路選項。

最後，虛擬機器 NIC 會連線到分散式連接埠群組。透過個別虛擬機器 NIC 組態或透過從 vSphere Distributed Switch 本身移轉虛擬機器網路，來連線虛擬機器 NIC。
