虛擬機器電源狀態

虛擬機器的電源狀態指示虛擬機器是否處於作用中且正常運作。虛擬機器有多種電源狀態，但三種基本狀態為:「開啟」、「關閉「和「暫停」。除了基本的虛擬機器電源作業外，虛擬機器還具備一些額外功能，但必須先在其中安裝 VMware Tools 才能使用這些功能。

您可以使用多個存取點來變更這些電源狀態:

  • 若要手動變更電源狀態，請在虛擬機器上按一下滑鼠右鍵，然後選取適當的電源選項功能表項目。
  • 若要排程電源狀態自動變更，請按 Ctrl 鍵並在虛擬機器上按一下滑鼠右鍵，然後選取適當的電源選項功能表項目。

如果某個電源狀態選項目前無法使用，則表示該選項已停用且無法選取。
