VMware Tools

VMware Tools 是一套公用程式，可提高虛擬機器客體作業系統的效能並改進虛擬機器管理。請先在虛擬機器上安裝客體作業系統，然後再安裝 VMware Tools。

在客體作業系統中安裝 VMware Tools 非常重要。雖然客體作業系統在不安裝 VMware Tools 的情況下也可以執行，但這樣會失去重要功能和便利性。安裝 VMware Tools 時還會安裝 VMware Tools 服務及一組 VMware 裝置驅動程式，它們可增強顯示、網路、SCSI、記憶體控制、I/O 和滑鼠支援。

安裝 VMware Tools

安裝 VMware Tools 前，請確認您的瀏覽器中已安裝 VMware 用戶端整合外掛程式。

  1. 在詳細目錄中的虛擬機器上按一下滑鼠右鍵並選取「所有 vCenter 動作」>「客體作業系統」>「安裝/升級 VMware Tools」。
  2. 在虛擬機器的「摘要」索引標籤上按一下「啟動主控台」。
  3. 在虛擬機器主控台中，依照安裝精靈中的指示進行。

對於 Windows 安裝，如果精靈沒有自動出現，請開啟客體作業系統中標記為 VMware Tools 的 CD-ROM 光碟機，然後執行安裝應用程式。

對於 Linux 安裝，請依照《客體作業系統安裝指南》中的說明進行。
